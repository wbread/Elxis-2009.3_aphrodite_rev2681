Διαχειριστής Γλωσσών

You can use Language Manager to control the available languages at the Front end and Back end. Under Language Manager you will find two submenus named Site Languages and Administrator Languages.

This page shows a list of the Languages for the core text on the Frontend of your site.

  • Γλώσσες Δημόσιου Τμήματος

  • Γλώσσες Διαχείρισης

 

Γλώσσες Δημόσιου Τμήματος

Γραμμή εργαλείων

Δημοσίευση: To Publish a front end language, select the radio button next to the name of the language and click the "Publish" icon.

Αποδημοσίευση: To Unpublish a front end language, select the radio button next to the name of the language and click the "Unpublish" icon.

Προκαθορισμένο: Is the language that is used by default on your site. To set an installed language as the default, select the radio button next to the name of the language and click the "Default" icon.

Νέο: Click the "New" icon to upload/install a new language to your site. You will be then taken to the Install/Uninstall screen.

Επεξεργασία: Select the radio button next to the name of the language and click the "Edit" icon, or simply click the name of the language to edit the contents. Here is where you edit the Frontend language file.

Διαγραφή: To delete a language, select the radio button next to the name and click the "Delete" icon. You will be asked to confirm the delete operation. Select the "Yes" icon to delete the client or the "No" icon to cancel. The language file will be deleted from the server.

Βοήθεια:  Click the "Help" icon to view the Help Screen to get explanations of this screen.

 

Στήλες

Γλώσσα: This column shows the name of the languages installed.

Σημαία: Shows the flag of the country language.

Δημοσιευμένο: This column shows the active languages that are published and available for use.

Default: This column shows the language that is set as default at Front-end.

Έκδοση: This column shows the version of the language file.

Ημερομηνία: This column shows the date of the language file.

Συγγραφέας: This column shows the author of the language file.

Author Email: This column shows author's email address, if available.

 

Γλώσσες Διαχείρισης
