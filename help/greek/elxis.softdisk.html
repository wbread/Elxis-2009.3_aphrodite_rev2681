SoftDisk

Το SoftDisk είναι μία εικονική περιοχή αποθήκευσης μεταβλητών και παραμέτρων κάθε είδους. You can add new or edit/delete existing SoftDisk's entries except the ones labeled as owned by the system. Also entries marked as "Undeletable" can only be edited. For naming SoftDisk's sections and value names you are only allowed to use capital latin letters, digits and underscore (_).

 

Γραμμή εργαλείων

Νέο: Create a new SoftDisk variable.

Επεξεργασία: Edit selected SoftDisk variable.

Διαγραφή: Delete selected SoftDisk variable. You can not delete variables marked as 'Undeletable'.

Βοήθεια: This Help Screen.

Ενημέρωση Συστήματος: Update system specific values stored in SoftDisk.

 

Section Filter

You can use the Section filter to display only SoftDisk variables that belong to the selected section.

 

Στήλες

Checkboxes column: A padlock icon, informs that this variable cannot be deleted.

Type column: Shows the type of SoftDisk variable. Available types are:

  • Δεκαδικός (i.e. - 10.45)
  • Email (i.e. - user@domain.com)
  • Εικόνα (i.e. - http://www.domain.com/image01.png)
  • Ακέραιος (i.e. - 10)
  • Χορδή (i.e. - Hello World)
  • Κείμενο (i.e. - Full Text)
  • Ώρα
  • URL (i.e. - http://www.domain.com)
  • ΝΑΙΟΧΙ

Ενότητα: The section a SoftDisk variable belongs. SoftDisk section have nothing to do with Content Sections.

Όνομα: Softisk variable name. Click to edit. Note: Hover the mouse over help icons to get info about the use of SoftDisk variables.

Τιμή: Variable's value.

Σύστημα: Informs if variable is of a system type or not.

Μη-Διαγριάψιμο: You cannot delete variables marked as undeletable.

Last modified: Date and time, value was modified for the last time.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of SoftDisk variables is larger than the number of variables displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed items per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
