Διαχειριστής Module: Νέο

The User module is a custom module created when clicking the "New" icon in the Module Manager. You may use this module to display:

  • Content that you create using the WYSIWYG editor (this content is not related to Sections, Categories or Autonomous Pages)
  • News from an RSS Feed

 

Module Details

Title: Module's title. It will be used as module's header.

Show title: In Front-End you can display or not, module's title.

Position: Select module's position.

Module Order: Define the order modules will appear in a module position.

Language: Select the language(s) under which this module will be available. If you select "All", it will be available and viewable, under all languages.

Access Level: Select the user group a user must belong, in order to view this module.

Published: Select "Yes" to make it available to this site. Select "No" to hide it.

Description: A short description about module.

 

Parameters

Module Class Suffix: (Common setting). By providing a module class suffix, you can customize the way this module renders in Front-End. You have to provide in template's CSS file, the modified classes.

These parameters affect custom module's behavior when it is used to display news through an RSS Feed.

RSS URL: Type the URL to the RSS feed.

Feed Description: Select if you want description provided by the RSS feed to be visible or not.

Feed Image: Select if you want image provided by the RSS feed to be visible or not.

Items: Type the number of news items you want to display from the RSS feed.

Item Description: Select if you want to show news items descriptions. If you select 'No', only titles will be shown.

Word Count: Type the number of words you want to be displayed from each news item. Set to 0 to display all available text.

Enable Cache: Enables or disables cache.

 

Custom Output

Use the WYSIWYG editor to add your content. Content will be displayed in the form of a module.

 

Pages / Items

The list shows all the available menus and their menu items. Based on them, you can select when the module will be visible.

  • You can select more than one menu items, by holding the CTRL key pressed.
  • If you want the module to be visible in every page, select "All".
  • If you want not to be visible, select "None".
