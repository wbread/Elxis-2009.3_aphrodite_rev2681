Διαχειριστής Πολυμέσων

Elxis Media Manager is an advanced system that let you manage all the multimedia files used in this site.

Supports the following media file formats:

Images: gif, jpg, png, bmp, jpeg
Video: mpg, avi, wmv, mov, flv, mpeg, asf, rm, nsv, xvid
Audio: mp3, wav, mid, ram, wma, ogg, aac
Other: doc, xls, csv, ppt, swf, pdf, txt, zip, tar, rar, gz, tgz

As 'Home' is considered the '/images' directory. You can see all directories (folders) and files that exist in it. Media Manager allows you to handle files and directories.

Available operations for Files: Open, Preview, Rename, Delete, Copy to, Resize, Watermark.

Available operations for Directories (Folders): Open, Rename, Delete, Copy to.

Media Manager is organized in three areas. At top there is the Toolbar, in the middle is the area where files and folders are displayed, and at the bottom there are the Upload controls.

 

Toolbar

From left to right there are the following controls:

Home: Click to return to '/images' directory.

Up: Click to move on level up in directories hierarchy.

Copy: Select a file and click this icon to copy it. You will be prompted for the destination folder. You cannot copy directories.

Delete: Select a file or directory and click this icon to delete it. All files and folders inside a directory will also be deleted.

Rename: Select a file or directory and click this icon to rename it.

New Folder: Click this icon to create a new folder (directory).

Permissions: Select a file or a directory and use this combo box to select a new permissions value for it. Click the 'Change Permissions' icon next to it, to apply changes.

Icons/Thumbnails View: Toggle switch that changes the way files are displayed.

Extract Zip: Select a zip file and click this icon to extract its contents.

Relative Path: It has two uses. It is used to display the current path or it can be used to type a new relative path and by clicking the 'arrow' icon, move to it.

 

Files and Folders

In this area you can see the files and folders that exist in the current directory.

Hover the mouse over an object to see its properties.

Right click an object and a context fly-out menu will appear with all available actions.

Double click a folder to open it.

Click a file's name to download it.

 

Resize Images

Right click an image file and select 'Resize'. A new window will open, from where you can select the new image dimensions. Click 'Go' to resize image.

 

Watermark Images

Using this feature, you can add a text or image watermark to an existing image.

Right click an image file and select 'Watermark'. A new window will open. You can select the properties, opacity and position of the watermark.

You have the option to save the image with the same name, overriding the original, or save it under a new name, keeping the original.

 

Elxis Media Manager is a flexible and powerful system for managing this web site's multimedia content. Play around and discover its potentials.
