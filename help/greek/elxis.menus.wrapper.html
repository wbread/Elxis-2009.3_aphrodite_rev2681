Περιτύλιγμα

This screen, adds a new menu item of type "Wrapper". Uses an IFrame to show output from another web site, output from an external application, or from an external web page. Together with RSS Feeds, this is a very handy method if you want to include in this site, information from other external sources.

 

Toolbar Icons

Back: Return to previous screen and select a new menu item type to create.

Save: Save changes and return to previous screen.

Apply: Save changes and remain to this screen.

Cancel: Aborts the creation of the new menu item. Changes are discarted if you are editing a menu item.

Help: This Help Screen.

 

Details

Name: This is the name of the menu item, as it will be shown in this site.

SEO title: Type here a Search Engines and Human-Friendly page title. Use only latin characters. Instead of spaces use '-' or '_' characters. It will be used only if SEO Pro is enabled, but it is a good habit to always fill it.

  • Suggested SEO title - Click it in order to let Elxis suggest a SEO title. Suggested SEO title will be based on 'Name' field.
  • Validate SEO title - After typing a SEO title, you can click this link in order to check if it is valid. If it is not please fix it!

Wrapper Link: Type here the URL address to the page or site you want to display inside the wrapper.

URL: Shows the URL to this menu item. This readonly field is shown only after saving the menu item.

Parent Item: You can select an existing menu item as the parent of this new menu item. That way you create hierarchical structures.

Ordering: You select the order you want this menu item to appear. Available only after saving the new menu item. New menu items default to the last place.

Language: Select the language under which this menu item will be available. If you select "All", it will be available and viewable, under all languages.

Access Level: Select the user group a user must belong, in order to view this menu item.

Published: Select "Yes" to make the menu item available to this site. Select "No" to hide this menu item.

 

Parameters

Menu Image: If you want you can show a small image next to the menu item. Select this image from the dropdown list. Note: For this image to be visible, the "Show Menu Icons" parameter of the menu module, must also be set to "Yes".

Use only the Menu Image: If you set this parameter to 'Yes', then instead of the menu name the menu item will be drawn only using the menu image.

Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page, and will have to be defined in your template's CSS stylesheet for usage. i.e. if you enter "-test" (without the quotes), you will have to define two extra classes: "contentpaneopen-test" and "contentheading-test".

Back Button: Controls whether the "Back" text link is shown or not at the bottom of the page. Clicking the "Back" link users are returned to the previous page. The dropdown list has 3 options:

  • "Use Global" - Shows or hides the Back link based on the setting in Global Configuration.
  • "Hide" - Hide the Back link, overriding the Global setting.
  • "Show" - Show the Back link, overriding the Global setting.

Show Page Title: Decide if you want to show a page title or not.

Page Title: If you wish to use a special text as title, please type it here.

Scroll Bars: Decide how you want scroll bars to behave in the wrapper iFrame. Available options are:

  • "No" - Do not use scroll bars. If the width or height of wrapper contents are larger than its dimensions, crop them.
  • "Yes" - Always show scroll bars, even if they are not needed.
  • "Auto" - Show scroll bars only if they are needed.

Width: Specify the width of the wrapper's iFrame. You can type an absolute value in pixels (e.g. 400), or use a relative value (e.g. 80%).

Height: Specify the height of the wrapper's iFrame. Only in pixels.

Auto Height: Set it to "Yes", if you want the wrapper's iFrame height to be calculated automatically in order to fit the external page. Works as expected only with pages from your own domain.

Auto Add: By default, if "http://", or "https://" are missing from URL address, they will be added automatically. You can disable this functionality, in order to have full control of the URL address.
