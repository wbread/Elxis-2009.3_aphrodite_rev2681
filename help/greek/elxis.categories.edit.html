Διαχειριστής Κατηγοριών:
Νέο/Επεξεργασία

Αυτή η σελίδα εμφανίζεται όταν επεξεργάζεστε ή δημιουργείτε μία Κατηγορία.

Η σελίδα αυτή καλείται από:

  • Αντικείμενα Περιεχομένου
  • Επαφές
  • Δελτία Τύπου (News Feeds)
  • Συνδέσμους (Web Links)

 

Γραμμή Εργαλείων

Αποστολή: Ανοίγει ένα νέο παράθυρο από το οποίο μπορείτε να στείλετε (ανεβάσετε) ένα αρχείο ή εικόνα από τον υπολογιστή σας στον κατάλογο /media.

Αποθήκευση: Αποθηκεύει την Κατηγορία και επιστρέφει στην προηγούμενη οθόνη.

Εφαρμογή: Αποθηκεύει τις αλλαγές και παραμένει στην ίδια οθόνη.

Άκυρο: Ακυρώνει τις αλλαγές και επιστρέφει στην προηγούμενη οθόνη.

Βοήθεια: Αυτή η Οθόνη Βοήθειας

 

Λεπτομέρειες Κατηγορίας

Ενότητα: Καθώς προσθέτετε μία νέα Κατηγορία, μπορείτε να επιλέξετε την Ενότητα στην οποία θα ανήκει. Ενώ επεξεργάζεστε μία Κατηγορία, το πεδίο δεν είναι επεξεργάσιμο και δείχνει την Ενότητα στην οποία ανήκει η Κατηγορία.

Τίτλος Κατηγορίας: Πληκτρολογήστε εδώ τον τίτλο της Κατηγορίας, όπως θέλετε να εμφανίζεται σε αυτό τον ιστότοπο.

Όνομα Κατηγορίας: Πληκτρολογήστε εδώ το όνομα της Κατηγορίας όπως θέλετε να εμφανίζεται σαν επικεφαλίδα.

Τίτλος SEO: Πληκτρολογήστε ένα Φιλικό προς τις Μηχανές Αναζήτησης και τους Ανθρώπους τίτλο για τη σελίδα. Χρησιμοποιήστε ΜΟΝΟ λατινικούς χαρακτήρες. Αντί για κενά, χρησιμοποιήστε τους χαρακτήρες '-' ή '_'. Θα χρησιμοποιηθεί μόνο αν είναι ενεργοποιημένο το SEO Pro, αλλά είναι καλή συνήθεια να το συμπληρώνετε πάντα.

  • Προτεινόμενος τίτλος SEO - Κάντε κλικ, προκειμένου να ζητήσετε από το Elxis να σας υποδείξει έναν τίτλο SEO. Ο SEO τίτλος που θα υποδειχθεί θα βασίζεται στο πεδίο 'Όνομα Κατηγορίας'.
  • Επικύρωση τίτλου SEO - Αφού πληκτρολογήσετε έναν τίτλο SEO title, μπορείτε να κάνετε κλικ ώστε να ελέγξετε την εγκυρότητά του. Εάν δεν είναι, παρακαλούμε διορθώστε τον!

Εικόνα: Εάν θέλετε, μπορείτε να επιλέξετε μία εικόνα η οποία θα χρησιμοποιηθεί δίπλα στο κείμενο της περιγραφής της Κατηγορίας. Η λίστα παρουσιάζει τις εικόνες από τον κατάλογο images/stories.

Θέση Εικόνας: Επιλέξτε τη θέση της εικόνας. Διαθέσιμες επιλογές είναι Δεξιά ή Αριστερά.

Κατάταξη: Επιλέξτε τη σειρά κατάταξης της Κατηγορίας. Επηρεάζει επίσης τη σειρά ταξινόμησης των Κατηγοριών στο Δημόσιο τμήμα. Οι νέες κατηγορίες τοποθετούνται εξ ορισμού στην τελευταία θέση. Η κατάταξη μπορεί να τροποποιηθεί αφού εφαρμόσετε τις αλλαγές.

Επίπεδο Πρόσβασης: Επιλέξτε την ομάδα χρηστών στην οποία πρέπει να ανήκει ένας χρήστης ώστε να μπορεί να δει αυτή την κατηγορία και τα περιεχόμενά της.

Δημοσιευμένο: Επιλέξτε "Ναι" για να την κάνετε διαθέσιμη στον ιστότοπο. Επιλέξτε "Όχι" για να την κρύψετε. Επίσης κρύβονται όλα τα αντικείμενα περιεχομένου που ανήκουν σε αυτή την Κατηγορία.

Γλώσσα: Επιλέξτε τη γλώσσα(ες) για τις οποίες θέλετε να είναι διαθέσιμη αυτή η Κατηγορία. Εάν επιλέξετε "Όλες οι Γλώσσες", θα είναι διαθέσιμη και ορατή, σε όλες τις γλώσσες.

Περιγραφή: Μπορείτε να πληκτρολογήσετε ένα περιγραφικό κείμενο για αυτή την Κατηγορία.

 

Κάτω από το πεδίο της Περιγραφής, υπάρχει ένας αριθμός από ειδικά κουμπιά. Εξ ορισμού, αυτά είναι 'Image', 'Page Break' και 'Flash Video'. Αυτά λειτουργούν σαν συντομεύσεις για την εισαγωγή κώδικα των Bots. Για παράδειγμα, κάνοντας κλικ στο κουμπί 'Page Break', θα εισαχθεί ο κώδικας '{mospagebreak}'. Εναλλακτικά, μπορείτε να εισάγετε με το χέρι τον κώδικα των Bots. Το αποτέλεσμα θα είναι το ίδιο.
Μπορείτε να εγκαταστήσετε και να χρησιμοποιήσετε περισσότερα Bots.

 

Οι ακόλουθες παράμετροι θα εμφανιστούν μόνο αφότου αποθηκεύσετε την κατηγορία για πρώτη φορά ή εφαρμόσετε τις αλλαγές.

 

Σύνδεση με Μενού

Παρέχει έναν εναλλακτικό τρόπο για να δημιουργήσετε ένα νέο αντικείμενο μενού ενός από τους ακόλουθους τύπους: "Πίνακας Κατηγορίας Περιεχομένου", "Blog Κατηγορίας Περιεχομένου", "Blog Αρχείου Κατηγορίας Περιεχομένου" που συνδέεται στην Κατηγορία.

Επιλέξτε ένα Μενού: Επιλέξτε το μενού στο οποίο θέλετε να δημιουργήστε το αντικείμενο μενού.

Αντικείμενο Μενού: Επιλέξτε τον τύπο του αντικειμένου μενού.

Όνομα Αντικειμένου Μενού: Πληκτρολογήστε το όνομα του αντικειμένου μενού.

Σύνδεση με Μενού: Κάντε κλικ σε αυτό το κουμπί για να δημιουργήσετε το νέο αντικείμενο μενού.

Υπάρχοντες Δεσμοί Μενού: Εμφανίζει μία λίστα με όλα τα αντικείμενα μενού που αναφέρονται σε αυτή την Κατηγορία.
