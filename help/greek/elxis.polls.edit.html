Νέο/Επεξεργασία

This page is shown when you add or edit a poll.

Toolbar Icons

Preview: Click to preview this poll in a pop-up window.

Save: Save poll and return to previous screen.

Close: Cancel changes and return to previous screen

Help: This Help Screen.

 

Details

Title: Type poll's title. It will also be displayed as the question asked.

SEO title: Type here a Search Engines and Human-Friendly page title. Use only latin characters. Instead of spaces use '-' or '_' characters. It will be used only if SEO Pro is enabled, but it is a good habit to always fill it.

  • Suggested SEO title - Click it in order to let Elxis suggest a SEO title. Suggested SEO title will be based on 'Name' field.
  • Validate SEO title - After typing a SEO title, you can click this link in order to check if it is valid. If it is not please fix it!

Published: Select "Yes" to make it available to this site. Select "No" to hide it.

Locked: Select if the poll will be locked or not.

Access: Select the user group a user must belong, in order to view this poll.

Lag: Type the time lag (in seconds) between votes.

  • For 24 hours use: 86400.
  • For 12 hours use: 43200.
  • For 6 hours use: 21600.
  • For 1 hour use: 3600.

Language: Select the language(s) under which this poll will be available. If you select "All", it will be available and viewable, under all languages.

Options: Type the possible answers (maximum 12). Empty items will not be displayed. Note: You may add additional options to an already published poll. Deleting options from a published poll will corrupt results accuracy.

 

Show on menu items

List shows all available menu items. You can select for which menu items this poll will be available. As the navigation in an Elxis site is performed using menus, indirectly you control the context areas where you want this poll to appear.

  • You can select more than one menu items, by holding the CTRL key pressed.
  • If you want the module to be visible in every page, select "All".
  • If you want not to be visible, select "None".
