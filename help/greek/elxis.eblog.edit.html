Ιστολόγιο Elxis (Eblog): Νέο/Επεξεργασία

Αυτή η σελίδα εμφανίζεται όταν δημιουργείτε ή επεξεργάζεστε ένα ιστολόγιο του Elxis.

 

Γραμμή εργαλείων

Αποθήκευση: Αποθηκεύει το ιστολόγιο και ανακατευθείνει στην προηγούμενη σελίδα.

Άκυρο: Ακυρώνει τις αλλαγές και ανακατευθείνει στην προηγούμενη σελίδα.

Βοήθεια: Εμφανίζει αυτή την οθόνη βοήθειας

 

Λεπτομέρειες ιστολογίου

Τίτλος: Γράψτε τον τίτλο του ιστολογίου όπως θέλετε να εμφανίζεται.

Τίτλος SEO: Γράψτε εδώ έναν φιλικό προς τις μηχανές αναζήτησης τίτλο σελίδας. Χρησιμοποιήστε μόνο μικρούς λατινικούς χαρακτήρες. Αντί για κενά χρησιμοποιήστε '-' ή '_'. Θα χρησιμοποιηθεί μόνο αν το SEO PRO είναι ενεργοποιημένο, αλλά απαιτείται να το συμπληρώσετε.

  • Προτεινόμενος τίτλος SEO - Πατήστε το προκειμένου το Elxis να σας προτείνει έναν τίτλο SEO. Ο προτινόμενος τίτλος SEO θα βασιστεί στον πεδίο Τίτλου του Blog.
  • Επικύρωση τίτλου SEO - Αφότου πληκτρολογήσετε έναν τίτλο SEO μπορείτε να πατήσετε αυτό το σύνδεσμο προκειμένου να ελέγξετεαν είναι έγκυρος. Αν δεν είναι παρακαλώ διοθρώστε τον!

Σλόγκαν: Αν θέλετε, μπορείτε να εισάγετε ένα σλόγκαν που θα εμφανίζεται στην κορυφή του ιστολογίου. Σημειώστε πως μπορείτε να γράψετε HTML σε αυτό το πεδίο. Γράφοντας HTML μπορείτε να πρωσοποποιήσετε την εμφάνιση του πάνω τμήματος του ιστολογίου.

Ιδιοκτήτης ιστολογίου: Επιλέξτε τον ιδιοκτήτη του ιστολογίου. Αυτός ο χρήστης θα μπορεί να διαχειρίζετε το ιστολόγιο και να γράφει άρθρα από το δημόσιο τμήμα του ιστότοπου.

Αρχεία CSS/RTL CSS: Τα αρχεία CSS χρησιμοποιούνται στο να παράσχουν πληροφορίες μορφοποίησης. Το αρχείο RTL CSS είναι για γλώσσες με κατεύθυνση γραφής από δεξιά προς τα αριστερά. Κάθε ιστολόγιο μπορεί να έχει το δικό του στυλ και χαρακτήρα. Επιλέξτε εδώ το αρχείο μορφοποίησης που θέκλετε να χρησιμοποιήσετε στο επιλεγμένο ιστολόγιο. Μπορείτε να ανεβάσετε περισσότερα αρχεία μορφοποίησης (αρχεία .css) στον κατάλογο components/com_eblog/css/.

Αποστολή: Επιλέξτε αν επιθυμείτε ο ιδιοκτήτης του ιστολογίου να ανεβάζει αρχεία πολυμέσων όπως εικόνες. Αν ενεργοποιηθεί τότε ένας ειδικός κατάλογος θα δημιουργηθεί αυτομάτως για αυτόν τον χρήστη για την αποθήκευση των αρχείων που ανεβάζει. Ο χρήστης θα μπορεί να έχει πρόσβαση μόνο στα αρχεία εντός του προσωπικού του καταλόγου. Οι κατάλογοι των χρηστών δημιουργούνται εντός του καταλόγου images/userfiles/, οπότε βεβαιωθείτε ότι αυτός ο κατάλογος είναι εγγράψιμος (δικαιώματα 777). Οι κατάλογοι των χρηστών ονοματίζονται με βάση τον κωδικό χρήστη (user id) προκειμένου να γνωρίζεται εύκολα ποιος κατάλογος, και κατ' επέκταση αρχεία, ανοίκει σε ποιο χρήστη.

Όλες οι άλλες επιλογές έχουν ενσωματωμένη βοήθεια. Αιωρήστε τον κέρσορά σας πάνω από το εικονίδιο βοηθείας για να λάβετε πληροφορίες για κάθε επιλογή.
