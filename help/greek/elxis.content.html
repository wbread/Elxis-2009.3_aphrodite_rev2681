Διαχειριστής Αντικειμένων Περιεχομένου

Content Items Manager is one of the most important features of Elxis CMS. It is used when you add, edit, delete content items (articles) for this site.

It can be accessed from two administration menus. Based on how it is called, shows a different set of content items. The following list explains that:

  • Called from Content -> Content by Section -> [Section] -> [Section] items
    Shows the content items that belong to this section.
  • Called from Content -> All Content Items
    Shows all content items.

 

Toolbar

New: Creates a new content item.

Edit: Edits the selected content item.

Publish and Unpublish: Publishes or unpublishes selected content item(s). Only published items are visible in this web site.

Move: Moves selected content items to another Section/Category.

Copy: Copies selected content items to another Section/Category.

Translate: Select a content item and click this icon to translate it. After successful translation, a new content item will be created holding the translated text.

Archive: Archive selected content items. Archived content items are moved to a special place called 'Archive'. It can be seen only through special menu items that present archived content.

Trash: Removes selected content items by moving them to the Trash Bin. Trashed items are not visible in this site. Trashed items are not actually deleted. They can be restored using 'Trash Manager'.

Help: This Help Screen.

 

Filter

You can use the Filter field to limit listed content items or search for specific content item. You can type a phrase or part of a word and press <Enter>. The table will be updated showning only the content items that have these characters in their title. If you want to show again all content items, leave empty and press <Enter>.

 

Simple/Extended View

Below the Filter field, you will find a switch that let you change the number of colums displayed. 'Simple' shows only the most important columns accelarating editing on web sites with large number of entries. 'Extended', shows all columns. You can try it and use it if it helps you.

 

Column Filters

You will notice that several columns have an arrow next to there headers. When you click them a filter window popups. You can use this filters to select the content items that you want to be displayed. This feature is addressed to web sites that have a large number of content items. It helps administrators to focus only to content items that each time interest them.

 

Columns

Title: This is the title of the Content Item. Click to edit Item. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the item.

Published: Shows if the item is published or not. Hover over this icon to see a summary of publishing information. Click to change state. Acts as a toggle switch.

Frontpage: If it is checked, then this item is also visible in the front page.

Reorder: You can change the order of Content Items, by using the arrows. You can also change the order by using the 'Order' boxes, described next.

Order: Shows the current order of Content Items. You can change it, by editing the order numbers and then clicking the "Save Order" icon next to column header.

Access: This is the user group that is allowed to view this content item. Click on it to change it. A popup window will appear.

ID: Shows content item's ID. Non editable.

Section: (Displayed only when you view 'All Content Items'). This is the Section the item belongs. Section name acts as a link to section parameters.

Category: This is the Category the item belongs. Category name acts as a link to category parameters.

Author: Shows Content Item's author name. Author names act as links to author details.

Language: Shows for which site languages this Content Item is available. It will be shown only if users are viewing this web site in one of the listed languages. If it is set to 'All', then it will be visible for all languages.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of content items is larger than the number of items displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed items per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.

Legend - Displays and explains the various publishing states of a content item. These icons are displayed in the 'Published' column.
