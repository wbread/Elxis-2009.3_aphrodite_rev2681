Διαχειριστής Κατηγοριών

Αυτή η σελίδα εμφανίζει μία λίστα με τις Κατηγορίες που έχουν οριστεί σε αυτό τον ιστότοπο. Εμφανίζονται ταξινομημένες βάσει της Ενότητας στην οποία ανήκουν.

Μπορείτε να χρησιμοποιήσετε αυτή την οθόνη για να προσθέσετε, επεξεργαστείτε, διαγράψετε, μετακινήσετε, ή αντιγράψετε Κατηγορίες.

Αυτή η σελίδα καλείται από:

  • Αντικείμενα Περιεχομένου
  • Επαφές
  • Δελτία Τύπου
  • Συνδέσμους

 

Γραμμή Εργαλείων

Δημοσίευση και Παύση Δημοσίευσης: Δημοισεύει ή αποδημοσιεύει τις επιλεγμένες κατηγορίες. Μόνο τα αντικείμενα στις δημοσιευμένες κατηγορίες είναι ορατά σε αυτό τον ιστότοπο.

Νέο: Δημιουργεί μία νέα Κατηγορία.

Μετακίνηση: Μετακινεί τις επιλεγμένες Κατηγορίες (συμπεριλαμβανομένων των αντικειμένων περιεχομένου), σε μία άλλη Ενότητα.

Αντιγραφή: Αντιγράφει τις επιλεγμένες Κατηγορίες (συμπεριλαμβανομένων των αντικειμένων περιεχομένου), σε μία άλλη ενότητα.

Επεξεργασία: Ανοίγει την επιλεγμένη Κατηγορία για επεξεργασία.

Διαγραφή: Διαγράφει τις επιλεγμένες κατηγορίες. This action cannot be performed if the Category contains content items.

Βοήθεια: Αυτή η Οθόνη Βοήθειας

 

Στήλες

Όνομα Κατηγορίας: Αυτό είναι το όνομα της Κατηγορίας. Κάντε κλικ στο όνομα για να επεξεργαστείτε την Κατηγορία. Αν υπάρχει ένα λουκέτο δίπλα σε αυτό, σημαίνει ότι είναι δεσμευμένη. Για να την αποδεσμεύσετε, κάντε κλικ στο όνομα για να την επεξεργαστείτε. Μετά θα πρέπει να την αποθηκεύσετε ή απλά να την κλείσετε.

Δημοσιευμένο: Δείχνει αν η Κατηγορία είναι δημοσιευμένη ή όχι. Κάντε κλικ για να αλλάξετε την κατάστασή της. Λειτουργεί σα διακόπτης.

Κατάταξη: Εμφανίζει την τρέχουσα κατάταξη των Κατηγοριών. Μπορείτε να την αλλάξετε, επεξεργαζόμενοι τα νούμερα κατάταξης και στη συνέχεια κάνοντας κλικ στο εικονίδιο "Αποθήκευση Κατάστασης" δίπλα στην επικεφαλίδα.

Πρόσβαση: Αυτή είναι η ομάδα χρηστών στην οποία επιτρέπεται να δει την Κατηγορία. Κάντε κλικ επάνω της για να την αλλάξετε. Θα εμφανιστεί ένα παράθυρο pop-up.

Ενότητα: Εμφανίζει την Εντότητα στην οποία ανήκει η Κατηγορία. Δεν αλλάζει από εδώ. Κάντε κλικ για να επεξεργαστείτε την Ενότητα.

Κωδ. Κατηγορίας: Εμφανίζει τον κωδικό της Κατηγορίας. Μη επεξεργάσιμο.

# Ενεργά: Εμφανίζει το πλήθος των δημοσιευμένων Αντικειμένων Περιεχομένου που έχει αυτή η Κατηγορία.

# Απορρίμματα: Εμφανίζει το πλήθος των Αντικειμένων Περιεχομένου που ανήκουν σε αυτή την Κατηγορία και βρίσκονται στον Κάδο Απορριμμάτων.

Γλώσσες: Εμφανίζει τις γλώσσες για τις οποίες είναι διαθέσιμη αυτή η Κατηγορία. Η κατηγορία θα εμφανιστεί στο Δημόσιο τμήμα, μόνο αν ο χρήστης έχει επιλέξει να δει τον ιστότοπο σε μία από τις αναφερόμενες γλώσσες. Εάν έχει οριστεί στο 'Όλες', θα είναι διαθέσιμη για όλες τις γλώσσες.

 

Υποσέλιδο

Στο κάτω μέρος της σελίδας υπάρχει το υποσέλιδο.

Πλοήγηση - Εάν το πλήθος των Κατηγοριών είναι μεγαλύτερο από το πλήθος των αντικειμένων που εμφανίζονται ανά σελίδα, τότε ενεργοποιούνται τα πλήκτρα πλοήγησης. Μπορείτε να τα χρησιμοποιήσετε για να περιηγηθείτε στις σελίδες.

Εμφάνιση # - Χρησιμοποιήστε αυτό το στοιχείο ελέγχου για να αλλάξετε το πλήθος των ανά σελίδα εμφανιζόμενων αντικειμένων. Η εξ ορισμού τιμή ελέγχεται από την παράμετρο 'Μήκος Λίστας' που ορίζεται στις Γενικές Ρυθμίσεις.
