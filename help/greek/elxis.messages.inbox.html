Εισερχόμενα

You can use this page to view private messages send to you. Private messages are send from Elxis or from other administrators. For example, when a user subitted a new article, Elxis is sending you a message.

 

Toolbar Icons

New: Create a new message.

Delete: Delete selected messages.

Help: This Help Screen.

 

Columns

Subject: Message subject. Click to read message.

From: Who send the message.

Date: When it was send.

Read: 'Read' if you have read the message. 'Unread' if you haven't yet read the message.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of messages is larger than the number of messages displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed messages per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
