Πρόσθετα Πεδία

Using this screen, you can add, edit or delete extra user fields for user profiles. Based on this feature you can create customized user profiles that suite the mission of your web site.

This help screen covers the List Table and the Edit/New Extra Field screens.

List Table

 

Toolbar Icons

New: Create a new extra field

Edit: Edit selected extra field. Alternatively you can click field's name to edit it.

Delete: Delete selected extra field(s).

Help: This Help Screen.

 

Columns

Name: The name of the field as it will appear on user's profile.

Type: Shows the type of the field.

Published: Shows if the extra field is published or not. Click to toggle.

Reorder: Use the up and down arrows to change the order of extra fields.

Required: Shows if the extra field is required or not.

Profile: Shows if the extra field is visible in user's profile.

Registration: Shows if the extra field is shown during registration.

Read Only: Shows if this is a Read Only field.

 

New/Edit Extra Field

This screen is shown when adding a new field or editing an existing one.

Toolbar Icons

Save: Save changes and return to previous screen.

Apply: Save changes and remain on this screen for further editing.

Cancel: Cancel changes and return to the previous screen.

Help: This Help Screen.

 

Details

Name: Type here the name of the extra field as it will appear in forms.

Type: Select the type of the new field. Options area (right side) is updated to reflect the available options for this type of field. Possible types are:

  • text - Text Field. You can specify the default value and the max length.
  • select - Selection list/ You can specify up to 20 values and select which one will be preselected.
  • radio - Radio button. You can specify up to 20 values and select which one will be preselected.
  • checkbox - Check box. You can specify up to 20 values and select which one will be preselected.
  • hidden - Hidden field. You can specify the default value.

Published: Select if the field will be published or not.

Required: Select if this will be a required field.

Show in Registration: If you want this field to be available in registration form, you have to set this oprion to Yes.

Show in Profile: If you want this field to be visible in user's profile, you have to set this option to Yes.

Read Only: If you want to make this a read only field, you have to set the option to Yes.

Options Alignment: Select the alignment direction for the options. Allowed values are: Horizontal or Vertical.

Preview: This is not an editable option. After you Save or Apply the changes, you can preview its appearance.
