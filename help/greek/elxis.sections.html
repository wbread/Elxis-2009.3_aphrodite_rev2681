Διαχειριστής Ενοτήτων

Sections manager is used to create, edit and delete sections used in Elxis.

 

Γραμμή εργαλείων

Δημοσίευση και Αποδημοσίευση: Publishes or unpublishes the selected Sections. Only items contained in published Sections are visible in this web site.

Νέο: Creates a new Section.

Αντιγραφή: Copies selected Sections (including Categories and Content Items), to a new Section.

Επεξεργασία: Edits the selected Sections.

Διαγραφή: Deletes selected Sections. This action cannot be performed if the Section contains Categories or Content Items.

Βοήθεια: This Help Screen.

 

Στήλες

Όνομα Ενότητας: This is Section's name. Click it to edit. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the Section.

Published: Shows if the Section is published or not. Click to change state. Acts as a toggle switch.

Reorder: You can change the order of Sections, by using the arrows. You can also change the order by using the 'Order' boxes, described next.

Order: Shows the current order of Sections. You can change it, by editing the order numbers and then clicking the "Save Order" icon next to column header.

Πρόσβαση: This is the user group that is allowed to view this Section. Click on it to change it. A popup window will appear.

Section ID: Shows Section's ID. Non editable.

# Categories: Shows how many Categories this Section has.

# Active: Shows the number of published Content Items this Section has.

#Trash: Shows the number of Content Items belonging to this Section and are currently in the Trash Bin.

Γλώσσα: Shows for which site languages this Section is available. It will be shown only if users are viewing this web site in one of the listed languages. If it is set to 'All', then it will be visible for all languages.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of Sections is larger than the number of items displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed items per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
