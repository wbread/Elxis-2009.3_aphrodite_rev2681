Διαχειριστής Banner

Χρησιμοποιώντας το Διαχειριστή Banners μπορείτε να διαχειριστείτε τα banners που εμφανίζονται σε αυτό τον ιστότοπο.

Γραμμή Εργαλείων

Δημοσίευση και Τέλος Δημοσίευσης: Δημοσιεύει ή αποδημοσιεύει τα επιλεγμένα banners. Μόνο τα δημοσιευμένα banners είναι ορατά σε αυτό τον ιστότοπο.

Αποστολή: Ανοίγει ένα νέο παράθυρο από το οποίο μπορείτε να στείλετε (ανεβάσετε) ένα αρχείο ή εικόνα από τον υπολογιστή σας στον κατάλογο /images/banners.

Νέο: Δημιουργία νέου banner.

Επεξεργασία: Ανοίγει το επιλεγμένο banner για επεξεργασία.

Διαγραφή: Διαγράφει τα επιλεγμένα banners.

Βοήθεια: Αυτή η Οθόνη Βοήθειας.

 

Φίλτρα Στήλης

Θα παρατηρήσετε ότι διάφορες στήλες έχουν ένα βελάκι δίπλα στις επικεφαλίδες τους. Όταν κάνετε κλικ σε αυτό, εμφανίζεται ένα παράθυρο φίλτρου. Μπορείτε να χρησιμοποιήσετε αυτά τα φίλτρα για να επιλέξετε τα banners που θέλετε να εμφανίζονται.

 

Στήλες

Όνομα Banner: Αυτό είναι το όνομα του banner. Κάντε κλικ για να το επεξεργαστείτε.

Όνομα Πελάτη: Κάθε banner σχετίζεται με έναν πελάτη. Αυτό είναι το όνομα του πελάτη.

Δημοσίευση: Εμφανίζει αν το banner είναι δημοσιευμένο ή όχι. Κάντε κλικ για να αλλάξετε την κατάστασή του. Λειτουργεί σαν διακόπτης.

Εμφανίσεις που έγιναν: Σας πληροφορεί για το πόσες φορές έχει εμφανιστεί ένα banner σε αυτό τον ιστότοπο.

Εμφανίσεις που απομένουν: Σας πληροφορεί για το πλήθος των εμφανίσεων που απομένουν. 'Απεριόριστες' σημαίνει ότι δεν υπάρχει περιορισμός.

Κλικς: Σας πληροφορεί για το πόσες φορές έχει γίνει κλικ σε ένα banner.

% Κλικς: Μετρά την αποτελεσματικότητα ενός banner.
