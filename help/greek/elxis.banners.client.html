Διαχειριστής Πελατών Banner

Κάθε banner σχετίζεται με έναν πελάτη. Χρησιμοποιήστε αυτή τη φόρμα για να προσθέσετε ή να επεξεργαστείτε πελάτες για τα banners.

Γραμμή Εργαλείων

Νέο: Δημιουργεί ένα νέο πελάτη.

Επεξεργασία: Άνοιγμα του επιλεγμένου πελάτη για επεξεργασία.

Διαγραφή: Διαγρφή του επιλεγμένου πελάτη.

Βοήθεια: Αυτή η Οθόνη Βοήθειας.

 

Στήλες

Όνομα Πελάτη: Το όνομα του πελάτη. Κλικ για επεξεργασία.

IDs των Πελατών: Το ID του πελάτη. Μπορεί να χρησιμοποιηθεί σαν παράμετρος στο module Banners.

Επαφή: Η επαφή γι' αυτό τον πελάτη.

Πλήθος Ενεργών Banner: Ο αριθμός των banners που ανήκουν σε αυτό τον πελάτη και είναι δημοσιευμένα.
