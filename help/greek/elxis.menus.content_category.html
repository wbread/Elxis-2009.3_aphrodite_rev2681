Αντικείμενο Μενού Τύπου:
Εμφάνιση Πίνακα - Κατηγορία Περιεχομένου

This screen, adds a new menu item of type "Table - Content Category". A menu item of this type, presents the content items that belong to a category in the form of table rows. From each article is presented only the Title. You can click on a title to read the full article.

We can define a number of parameters for this menu item type, that control the number and the way the tables and rows are presented.

 

Toolbar Icons

Back: Return to previous screen and select a new menu item type to create.

Save: Save changes and return to previous screen.

Apply: Save changes and remain to this screen.

Cancel: Aborts the creation of the new menu item. Changes are discarted if you are editing a menu item.

Help: This Help Screen.

 

Details

Name: This is the name of the menu item, as it will be shown in this site.

Category: Select the content Category from which you want to show articles. You can select only one.

URL: Shows the URL to this menu item. This readonly field is shown only after saving the menu item.

Parent Item: You can select an existing menu item as the parent of this new menu item. That way you create hierarchical structures.

Ordering: You select the order you want this menu item to appear. Available only after saving the new menu item. New menu items default to the last place.

Language: Select the language under which this menu item will be available. If you select "All", it will be available and viewable, under all languages.

Access Level: Select the user group a user must belong, in order to view this menu item.

Published: Select "Yes" to make the menu item avaialble to this site. Select "No" to hide this menu item.

 

Parameters

Menu Image: If you want you can show a small image next to the menu item. Select this image from the dropdown list. Note: For this image to be visible, the "Show Menu Icons" parameter of the menu module, must also be set to "Yes".

Use only the Menu Image: If you set this parameter to 'Yes', then instead of the menu name the menu item will be drawn only using the menu image.

Show Page Title: If you wish the page title to be visible select "Yes".

Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page. You have to define them in your template's CSS stylesheet. For example, if you enter "-test" (without the quotes), you will have to define extra classes like: "contentpaneopen-test" and "contentheading-test".

Back Button: Controls whether the "Back" text link is shown or not at the bottom of the page. Clicking the "Back" link users are returned to the previous page. The dropdown list has 3 options:

  • "Use Global" - Shows or hides the Back link based on the setting in Global Configuration.
  • "Hide" - Hide the Back link, overriding the Global setting.
  • "Show" - Show the Back link, overriding the Global setting.

Order by: Select the order you want the items to be shown.

Date Format: Select the format you want dates to be shown. If it is left blank it will load the format from the language file. The format is based on PHP's strftime function parameters. For more info on the available parameters, visit: function strftime. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Effects date format.
  • Search Engine Friendly URLs = 'SEO Pro' - Has no effect.

Date Column: Shows or hides the date info. You may use the global setting or bypass it locally.

Author Column: Shows or hides the author info. You may use the global setting or bypass it locally.

Hits Column: Shows or hides the hits info. You may use the global setting or bypass it locally.

Table Headings: Deprecated. Has no effect.

Navigation Bar: Normally, there is a navigation bar bellow the table. You may select if you want to be visible or not. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Always visible (5 items per page).

Order Select: If you select this option you can give visitors the option to order the table rows according to their preferences. A dropdown list appears, above the table wilth all the available options. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Always visible.

Display Select: You can let user change the number of items per page. A dropdown list is shown with available options. You can set the initial value from the next parameter. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Not visible.

Display Number: Select the initial items per page value for this menu item.

Filter: You have the option to show a filter text box above the table. Users can type a word to filter the displayed content items. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Always visible.

Other Categories: You may select to show, all the other categories that belong to the Section that this category belongs to. Other Categories are presented as links bellow the table. The next three parameters control the behavior and data of the links. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Not visible.

Empty Categories: Deprecated. Has no effect. Empty categories never shown.

# Category Items: Select if you want to show the number of content items each category has. Works only if 'Search Engine Friendly URLs' global parameter is set to 'No' or 'SEO Basic'.

Category Description: Select if you want to display also the description text for each category link. Its operation depends on the 'Search Engine Friendly URLs' global setting.

  • Search Engine Friendly URLs = 'No' or 'SEO Basic' - Works as expected.
  • Search Engine Friendly URLs = 'SEO Pro' - Always Visible.
