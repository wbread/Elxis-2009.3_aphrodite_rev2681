Θέσεις Module

Modules in Elxis are displayed inside placeholders called 'module positions'. These placeholders are defined in templates.

Placeholders are identified by names. By default Elxis comes with a number of predefined names that you can use when developing templates.

You can use this screen to view existing module position names or define new ones.

 

Toolbar Icons

Save: Save changes and return to previous screen.

Cancel: Cancel changes and return to previous screen

Help: This Help Screen.

 

Columns

Position: Module position name. Please do not change existing names, unless you know what you are doing.

Description: You can enter comments about a position. Only internal use.
