This is a helper function. Using this screen, you can check which sections, categories and content items will be visible by a user, based on the selected site language.

Tree does not display empty sections and categories.

Items marked with grey color and italic are unpublished.

Help: This Help Screen

 

Language Selector

You can use the language selection drop down list, to change simulated language. The language map will be updated to reflect visible content.

 

Site Map

Shows the visible content based on selected language.

Content is organized in Sections and Categories.

Next to each Section, Category, Content Item you can see inside parenthesis, the languages for which it is visible. '(All)', means that is visible for all languages.

Click yellow arrows to collapse or expand sections.

Click 'Edit Icons' and you will be transfered to the edit screen of Sections, Categories and Content Items.

Press and hold a title to rename it. The title will switch to Edit Mode. To save changes or simply cancel, click anywhere outside the edit box. A 'message from server' will appear saying that the item is renamed.
