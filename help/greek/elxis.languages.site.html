Use Languages Manager (Site) to view, edit, publish/unpublish and install languages used at the Front-End.

This page shows the list of available site languages. Site languages are used in the Front-End of this site.

 

Γραμμή εργαλείων

Δημοσίευση: To Publish a Site language, check it and click the "Publish" icon. Only published languages are available to web site users.

Αποδημοσίευση: To Unpublish a Site language, select it and click the "Unpublish" icon.

Προκαθορισμένο: Default is the language that is used by default on this site. To set an installed language as the site's default, select it and click the "Default" icon.

Νέο: To install a new site language, click the "New" icon. You will be transferred to the install/uninstall screen.

Επεξεργασία: To edit a site language and modify the language strings to fit your needs, select it and click the "Edit" icon, or simply click the name of the language.

Διαγραφή: To delete a site language, select it and click the "Delete" icon. You will be asked to confirm the deletion. Select the "Yes" icon to proceed, or the "No" icon to cancel. The site language files will be deleted from the server.

Βοήθεια: Click the "Help" icon to view this Help Screen.

 

Στήλες

Γλώσσα: This column shows the names of the site languages installed. You can click a name to edit the site language file.

Σημαία: Shows the flag icon for this language.

Δημοσιευμένο: Shows if a site language is published or not. Only published site languages are available to web site users.

Προκαθορισμένο: Shows the site language that is set as default. Only one site language can be the default.

Έκδοση: Shows the version of the language file.

Ημερομηνία: Shows the date the language file was created.

Συγγραφέας: Shows the author of the language file.

E-mail Συγγραφέα: Shows author's email address, if available.
