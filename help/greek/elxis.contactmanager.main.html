With Contact Manager you can manage a list of contacts for your site. Visitors can find contact information for different people or departments.

 

Toolbar Icons

Publish and Unpublish: Publishes or unpublishes the selected contact(s). Only published contacts are visible in this web site.

New: Creates a new contact.

Edit: Edits the selected contact.

Delete: Deletes selected contacts.

Help: This Help Screen.

 

Filter

You can use the Filter field to limit listed contacts or search for specific contacts. You can type a phrase or part of a word and press <Enter>. The table will be updated showning only the contacts that have these characters in their name. If you want to show again all contacts, leave empty and press <Enter>.

 

Column Filter

You will notice that Category column has an arrow next to its header. When you click it a filter window popups. You can use this filter to display only the contacts that belong to the selected category.

 

Columns

Name: Contact's name.

Published: Shows if the contact is published or not. Click to change state. Acts as a toggle switch.

Reorder: You can change the order of contacts, by using the arrows.

Category: This is the category the contact belengs. Category name acts as a link to contact category parameters.

Linked to User: Contact items can be linked (associated) with users. If such associations exist, they are displayed here.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of contacts is larger than the number of items displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed contacts per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
