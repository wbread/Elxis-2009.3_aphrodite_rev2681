Αντιγραφή

This page lets you copy a Menu and all its Menu Items. A list displays the name of the Menu and Menu Items copied.

 

Toolbar

Copy: A new "Menu Name" module will be created in the Menu Manager list. A new mod_mainmenu module, with the "Module Name" you have entered below, will also be created. Parameters for the created module can be edited through the Modules -> Site Modules.

Cancel: Click to cancel and return.

Help: This Help Screen.

 

Fields

New Menu Name: This is the identification name used by Elxis to identify this menu within the code so it must be unique. It is recommended to use no spaces within this name. For example, in the core Elxis the User Menu's Menu Name is "usermenu" and the Main Menu's Menu Name is "mainmenu".

New Module Name: Type here a name for the Module that handles your new menu. This is the name given to the new mod_mainmenu module which is automatically created when you save this menu. It is also the Name of the menu as it will appear on the page if you choose to show it (this may be changed from Modules Manager). In the example described above these are "User Menu" and "Main Menu".
