Διαχειριστής Μενού:
Νέο/Επεξεργασία

Using this screen you can add new Menus to Elxis.

 

Toolbar Icons

Save: Save the new Menu.

Cancel: Cancel changes and return.

Help: This Help Screen.

 

Fields

Menu Name: This is the name used internally by Elxis to identify the Menu. This is the name shown on Menu Manager list.

Title Module: As each Menu is associated with a Module, this is the name of the Module that is associated with the Menu. You can change this name later, from Modules Manager.
