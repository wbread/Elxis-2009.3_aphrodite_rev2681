Πίνακας Διαχείρισης

The administration panel is the first page you see when you log into the Administration area (Back-End). You can return to it by clicking the Home menu item in the top menu bar. In administration panel there are 15 shortcut icons to easy access the most frequently used functions. You can add your own Shortcuts, using the Shortcuts Tool.

Front Page Manager - Click this shortcut icon to access the Front Page Manager. You will be able to edit which Content items appear on your Frontpage.

Sections Manager - Click the shortcut icon to access the Sections Manager. You will be able to edit your content Sections.

Categories Manager - Click the shortcut icon to access the Categories Manager. You will be able to edit your content Categories.

All Content Items - Click the shortcut icon to access the All Content Items. All Content Items lists all of your Content Items. You will be able to edit any of your Content Items.

Autonomous Pages Manager - Click the shortcut icon to access the Autonomous Pages Manager. You will be able to edit Autonomous Pages.

Media Manager - Click the shortcut icon to access the Media Manager. You will be able to browse, add, delete media files used in your site.

Menus Manager - Click this shortcut icon to access the Menus Manager. You will be able to edit any of your menus.

Users Manager - Click this shortcut icon to access the Users Manager. You will be able to edit, add, delete registered users to your site.

Global Configuration - Click this shortcut icon to access Global configuration. You will be able to edit your website configuration settings.

Access - Click this shortcut icon to open Access Manager. You can use Access Manager to add, delete or edit User Groups and their access rights.

Database Manager - Click this shortcut icon to access the Database Manager. You will be able to inspect or perform maintenance operations on your database.

Languages Manager - Click this shortcut icon to access the Site Languages Manager. You will be able to add, delete, edit site languages. For languages used in the Administration area, please select Administrator Languages from the Home menu.

Trash Manager - Click this shortcut icon to access the Trash manager. You will be able to delete permanently or restore any items placed in the Trash Manager.

Tools - Click this shortcut icon to access the Tools page. You will be able to select and use any of the availble Elxis tools.

Help - Click this shortcut icon to view an Index of the available Help Screens.

Softdisk - Click this shortcut icon to access the Softdisk system.

 

On the right side of the screen there are five tabs:

Logged - Displays a list of users currently logged on (both in Administration Panel and the Front-End). Click on a user's name to go to the User Edit page for that user.

Components - Shows a list with the installed components and lets you interact with them.

Popular - Shows the most popular content items. Click a Content Item to edit it.

Latest Items - Click on this shortcut tab to access the screen which lists the most recently added Content items. Also shows the creation date and the author. Click a Content item to edit it.

Menu Stats - Shows the defined menus and the number of items they contain.
