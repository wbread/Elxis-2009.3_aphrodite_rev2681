Elxis Blog (Eblog): New/Edit

This page is shown when you are creating or editing an Elxis Blog.

 

Toolbar Icons

Save: Saves the blog and returns to previous screen.

Cancel: Cancels changes and returns to previous screen.

Help: Shows this Help Screen

 

Blog details

Title: Type here the title of the Blog as you want to be displayed.

SEO title: Type here a Search Engines and Human-Friendly page title. Use only latin characters. Instead of spaces use '-' or '_' characters. It will be used only if SEO Pro is enabled, but it is required to fill it in.

  • Suggested SEO title - Click it in order to let Elxis suggest a SEO title. Suggested SEO title will be based on 'Blog's Title field.
  • Validate SEO title - After typing a SEO title, you can click this link in order to check if it is valid. If it is not please fix it!

Slogan: If you want, you can enter a slogan to be displayed in the blog header. Notice that you can write HTML in this field. By writing HTML you can customize your blog header appearance.

Blog owner: Select the owner of the blog. That user will be able to manage this blog and post articles from the site front-end.

CSS/RTL CSS files: CSS files are used to provide style information. The RTL CSS file is for languages with text direction from right to left. Each blog can have its own style and character. Select here the stylesheet you want to use for the selected blog. You can upload more stylesheets (.css files) inside folder components/com_eblog/css/.

Upload: Select of you wish the blog owner to be able to upload media files such as images. If enabled then a special folder will be automatically created for that user to save the uploaded files. User will be able to access only the files inside his personal folder. User folders are created inside the images/userfiles/ directory, so make sure this directory has write permissions (777). User's folders are named as their user id in order to be able to easily know which folder belongs to which user.

All other options have help integrated. Hover your mouse over the help icon to see information for each option.
