メニューアイテムタイプ:
テーブル - ウェブリンクカテゴリ

この画面は「テーブル - ウェブリンクカテゴリ」というタイプの新しいメニューアイテムを追加します。このタイプのメニューは特定のウェブリンクカテゴリに属するウェブリンクを提供します。

アイテムが提供される方法と数をコントロールするこのメニューアイテムのためのパラメータを定義できます。

 

ツールバーアイコン

戻る: 前の画面へ戻って作成する新しいメニューアイテムのタイプを選択します。

保存: 変更を保存して前の画面へ戻ります。

適用: 変更を保存してこの画面を維持します。

キャンセル: 新しいメニューアイテムの作成を中止します。メニューアイテムを編集中の場合変更は破棄されます。

ヘルプ: このヘルプ画面です。

 

詳細

名前: これはこのサイトに表示されるメニューアイテムの名前です。

カテゴリ: ウェブリンクを表示したいウェブリンクカテゴリを選択します。

URL: このメニューアイテムへのURLを表示します。この読込のみのフィールドはメニューアイテムの保存後にのみ表示されます。

親アイテム: この新しいメニューアイテムの親として既存のメニューアイテムを選択できます。その方法は階層構造を作成します。

並び順: このメニューアイテムを表示したい順番を選択します。新しいメニューアイテムを保存後にのみ利用可能になります。新しいメニューアイテムはデフォルトで最後の位置になります。

言語: このメニューアイテムが利用可能な言語を選択します。「全て」を選択すると全ての言語で閲覧可能になります。

アクセスレベル: このメニューアイテムを閲覧するためにユーザが属すべきユーザグループを選択します。

公開済み: このサイトでメニューアイテムを利用可能にするには「はい」を選択します。隠すには「いいえ」を選択します。

 

パラメータ

メニュー画像: このページを表示するのに使用されるメニューアイテムの隣に小さな画像を表示することができます。ドロップダウンリストからこの画像を選択します。注: この画像を閲覧可能にするためにメニューモジュールの「メニューアイコンの表示」パラメータを「はい」に設定しなければなりません。

メニュー画像のみ使用: このパラメータを「はい」に設定すると、メニュー名の代わりにメニューアイテムはメニュー画像のみ使用します。

ページクラスの接尾語(Page Class Suffix): このページにカスタマイズされたスタイルを設定することができます。ここで入力する接尾語はこのページのCSSクラスとして自動的に追加されるので、テンプレートのCSSのスタイルシートに定義をしておかなければなりません。例として、「-test」と入力したなら(「」を除いて)、二つの追加クラスに次のように定義をしなければなりません: 「contentpaneopen-test」、「contentheading-test」

戻るボタン: ページ下部に表示される「戻る」のテキストリンクを表示するかどうかをコントロールします。

ページタイトルの表示: ページタイトルを表示するかどうかを選択します。

ページタイトル: このページのタイトルとして特別なテキストを表示したいならここに入力します。

テーブルヘッダ: 効果はありません。

ヒット数: 効果はありません。

リンク説明: ウェブリンクの説明文を表示するかどうかです。

他のカテゴリ: ページの下部に利用可能なウェブリンクカテゴリの一覧を表示するかどうかです。
