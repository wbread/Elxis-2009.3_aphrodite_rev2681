ユーザ管理: ユーザの追加又は編集

この画面は新しいユーザの追加又はユーザのプロフィールを編集するときに表示されます。それは2つの主なエリアに分かれています。左はユーザの詳細で右はユーザの外部フィールドと連絡先情報です。

 

ツールバーアイコン

保存: Save changes and return to Users Manager.

適用: 変更を保存してこの画面を維持します。

閉じる: 変更をキャンセルしてユーザ管理へ戻ります。

ヘルプ: このヘルプ画面です。

 

ユーザ詳細

名前: ユーザの実名です。例：John Doe

ユーザ名: ユーザのアカウントユーザ名です。ログインするためにパスワードと共に用いられます。ユーザ管理ではテーブルはユーザIDとして表示されます。

メールアドレス: ユーザのメールアドレスをここに入力します。

新しいパスワード: ユーザのパスワードを入力します。ユーザを編集中にパスワードを変更したい場合に使用されます。

新しいパスワードの確認: パスワードを再入力します。

グループ: ユーザが属するユーザグループを選択します。

選択言語: ユーザが通知を受け取る言語です。

アバター: このユーザのアバターを選択します。空にすることができます。アバターとして使用される新しい画像をアップロードすることもできます。

ユーザのブロック: 「いいえ」を設定するとユーザはログインできます。「はい」に設定するとログインをブロックします。デフォルト値は「いいえ」です。

送信メールの受信: (最高管理者グループに属するユーザアカウントのみ見ることができます。) この最高管理者が新しい送信について通知を受け取りたいなら「はい」を選択します。

登録日: ユーザが登録された日付を表示します。

最終訪問日: ユーザが最後にこのサイトを訪れた日付を表示します。

 

外部フィールド

Elxisの外部プロフィールフィールドは2つのグループに分かれます。デフォルトと外部ユーザフィールドです。
デフォルトは削除又は変更ができない事前に定義されたフィールドです。
外部ユーザフィールドは定義して既存のものに追加されたカスタムユーザフィールドです。 ユーザ自身がプロフィールの設定を定義できる、プロフェッショナルなウェブサイトには非常に重要なことです。

外部フィールドはフィールドの両方のタイプを一覧にします。どんなものも定義されているなら最初にデフォルトを見てそれらの下に拡張フィールドを見ます。

ソフトディスクを使用してデフォルトフィールドを有効または無効にできます。

 

連絡先情報

Elxisではオプションで連絡先アイテムをユーザのプロフィールにつけることができます。連絡先アイテムは連絡先フォームの表示と連絡先情報の定義に使用されます。コンポーネント -> 連絡先 -> 連絡先管理から連絡先アイテムをユーザにリンクすることができます。リンクを実行した場合、情報はここに表示され、下部のリンクは連絡先ページの編集への直接アクセスを提供します。
