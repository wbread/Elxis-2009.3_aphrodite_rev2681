Gestionnaire de Modules: Editer

Cette page affiche les Modules Elxis que vous pouvez éditer pour modifier certains paramètres.

 

Icône de la Barre d'Outils

Aperçu: Ouvre une nouvelle fenêtre qui visionne le module de l'Utilisateur comme il vous sera présenté sur le site.

Sauver: Sauvegardez les changements et retournez à l'écran précédent.

Appliquer: Sauvegardez les changements et restez sur cet écran.

Annuler: Annule la création de menu. Aucun changements ne sera fait. Les changements ne sont pas pris en compte si vous éditez un menu.

Aide: Cet écran d'aide.

 

Détails des Modules

Titre: Titre du Module. Il sera utilisé dans l'entête du module.

Afficher le titre: Sur la partie-Public vous pouvez afficher ou pas, le titre du module.

Position: Sélectionnez la position du module.

Ordre du Module: Défini l'ordre dans lequel apparaitra les positions des modules.

Langue: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Niveau d'Accès: Sélectionnez le groupe d'utilisateurs auquel l'utilisateur doit appartenir pour afficher cet article du menu.

Publié: Sélectionnez "Oui" pour rendre disponible le menu sur votre site. Sélectionnez "Non" pour le cacher.

Description: Une courte description sur le module.

 

Paramètres

Les paramètres varient suivant la fonction du module. En modifiant les paramètres vous contrôlez le comportement du module.

 

Pages / Eléments

La liste montre tous les menus disponibles ainsi que leurs articles. A partir d'eux, vous pouvez sélectionner quand le module sera visible.

  • Vous pouvez sélectionner plus d'un articles du menu, en tenant la touche CTRL enfoncé.
  • Si vous voulez que le module soit visible dans chaque page, sélectionnez "Tous."
  • Si vous voulez qu'il ne soit pas visible, sélectionnez "Aucun".
