Gestionnaire de Sauvegarde

Sur cette page vous pouvez afficher les sauvegardes de la base de données que vous avez enregistré ou créé sur le serveur.

 

Fichiers de Sauvegarde

Les fichiers sont sauvegardés dans le répertoire administrator/backups/. Leur nom de fichier est formé d'une manière structurée. Le format est expliqué ci-dessous:

yyyymmddhhmmss_elxis[_full].aaaaaaaa.sql/xml

Chaques début du nom du fichier commence par la date de la sauvegarde (yyyymmdd), suivi de l'heure (hhmmss). Suit ensuite la chaîne "_elxis". Si c'est une sauvegarde complète (structure et données) il y a le mot "_full". Au Final il y a 8 positions de chaîne alphanumérique aléatoire, suivie par l'extension du type de sauvegarde (xml ou sql).

Vous pouvez télécharger le fichier sur votre ordinateur, pour cela faite un clic de bouton droit sur le fichier sélectionné puis faire "Enregistrer-Sous".

Tous les fichiers sont codés en UTF-8.

 

Icone de la Barre d'Outils

Supprimer les Sauvegardes Sélectionnées: Supprimera la sauvegarde sélectionnée en cliquant sur l'icône.

Nouvelle Sauvegarde complète en XML: En cliquant sur cet icone vous lancerez une sauvegarde complète (structure et données) au format XML.

Nouveau Suavegarde complète en SQL: En cliquant sur cet icone vous lancerez une sauvegarde complète (structure et données) au format SQL.

Aide: Cet écran d'aide.

 

Colonnes

Fichier: Le nom du Fichier de la sauvegarde. Le format est expliqué ci-dessous.

Type: Description du type de sauvegarde.

Taille: Taille du fichier en KB.

Date: Date a laquelle la sauvegarde a été faite
