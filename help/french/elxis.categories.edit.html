Gestionnaire de Catégories : Ajouter/Editer

Cette page vous permet de créer ou d'éditer une catégorie.

Cette page est appelée à partir de:

  • Eléments de Contenus
  • Contacts
  • Flux de nouvelles
  • Liens Web

 

Icônes de la Barre d'Outils

Envoyer:Ouvre une nouvelle fenêtre vous permettant d'envoyer un fichier ou une image depuis votre ordinateur vers le répertoire /medias

Sauver: Sauvegarde la catégorie et revient au Gestionnaire.

Appliquer: Sauvegarde la catégorie mais reste sur la page en cours d'édition.

Annuler: Annule l'opération. Toute modification sera perdue. Si vous étiez en train d'éditer, la catégorie retournera en mode vérification.

Aide: Cet écrand d'aide

 

Détails des Catégories

Section: Sélectionnez ici la section à laquelle vous voulez faire appartenir votre catégorie. Ce choix ne sera plus disponible une fois que la catégorie aura été "appliquée" ou "sauvegardée". A la place, sera affichée la section à laquelle la catégorie est affectée.

Titre de la Catégorie: Saisissez ici le nom de la catégorie tel que vous désirez le voir affiché sur votre site.

Nom de la Catégorie: Saisissez ici le nom de la catégorie tel que vous désirez le voir apparaître dans la base de données et dans le partie administrative.

Titre SEO: Saisissez ici les termes pour les moteurs de recherche pour le titre de la page. Utilisez uniquement des caractères latin. De même pour les espaces utilisez les caractères '-' ou '_'. Cela ne sera valable que si le système SEO Pro est actif, mais c'est une bonne habitude de toujours remplir ce champ.

  • Titre SEO suggéré - Cliquez ici pour laisser le SEO de Elxis vous suggérer un titre. Le titre SEO suggéré sera basé sur les champs 'Categories's Nom'.
  • Titre SEO valide - En tapant un titre SEO, vous pouvez cliquer sur ce lien afin de vérifier si il est valide. Si il ne l'est pas veuillez le corriger!

Images: Sélectionnez ici une image à afficher à côté du texte descriptif de la catégorie. La liste déroulante offre le choix d'une image tirée du répertoire images/stories.

Position de l'image: Sélectionnez ici la position de l'image par rapport au texte descriptif. Gauche ou Droite.

Ordre: Sélectionnez ici la place à laquelle est affichée la catégorie par rapport aux autres catégories de la même section. Cela va se répercuter dans le Gestionnaire de Catégories.

Niveau d'accès: Ceci est le niveau d'accès utilisateur pour cette catégorie. Public, Enregistré ou Spécial.

Publié: Cochez la case pour laisser l'élément s'afficher sur votre site.

Langues: Sélectionnez le ou les langue(s) pour que la catégorie soit visible. Si vous sélectionnez "Tous", celle-ci sera visible sous toutes les langues.

Description: Saisissez ici une description pour la catégorie. Ce texte va s'afficher si vous l'avez décidé quand la catégorie est affichée comme une table, un blog ou d'autres catégories.

 

Description, il ya un certain nombre de boutons. Par défaut sont actifs 'Image', 'Page Break' et 'Flash Video'. Ce sont des raccourcis pour insérer des codes Bots. Par exemple, clicquer sur le bouton 'Page Break', ajoutera '{mospagebreak}'. Alternativement, vous pouvez insérer plusieurs codes. L'effet sera le même.
Vous pouvez installer et utiliser d'autres Bots.

 

Les paramètres suivants ne seront affichés qu'une fois la catégorie "appliquée" ou "sauvegardée".

 

Lien vers Menu

Cela va créer un nouvel élément de menu dans le menu que vous avez sélectionné.

Sélectionner un Menu: Mettez ici en surbrillance un des menus définis pour votre site.

Sélectionner un Type de Menu: Sélectionnez ici les types d'éléments de Menus.

Nom Elément de Menu: Le nom du menu tel qu'il apparaîtra sur votre site.

Lien vers Menu: Cliquez sur ce bouton pour créer l'élément de menu.

Liens vers un Menu Existant: Affiche une liste de tous les articles des Menus référencés dans la catégorie.
