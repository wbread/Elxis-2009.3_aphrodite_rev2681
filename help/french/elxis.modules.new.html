Gestionnaire de Modules: Nouveau

Le module Utilisateur est un module personnalisé créé quand vous cliquez sur l'icone "Nouveau" dans le Gestionnaire du Module. Vous pouvez utiliser ce module pour afficher le Contenu:

  • Le contenu créé utilisera l'éditeur WYSIWYG (ce contenu n'est pas mis en rapport avec les Sections, Categories ou les page de Contenu Statiques)
  • Nouvelles du Flux RSS

 

Détails du Module

Titre: Le titre de module. Il sera utilisé comme en-tête du module.

Afficher le titre: Sur la partie-Public vous pouvez afficher ou pas, le titre du module.

Position: Sélectionnez la position du module.

Ordre du Module: Défini l'ordre dans lequel apparaitra les positions des modules.

Langues: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Niveau d'Accès: Sélectionnez le groupe d'utilisateurs auquel l'utilisateur doit appartenir pour afficher cet article du menu.

Publié: Sélectionnez "Oui" pour rendre disponible le menu sur votre site. Sélectionnez "Non" pour le cacher.

Description: Une courte description sur le module.

 

Paramètres

Classe Suffixe de la Page: vous permet de personnaliser le style de cette page. Le suffixe que vous saisissez ici est automatiquement ajouté à la classe CSS pour cette page.

Ces paramètres affectent le comportement du module personnalisé quand il est utilisé pour afficher des nouvelles à travers une flux RSS.

URL RSS: Entrez ici l'URL du flux RSS que vous voulez afficher sur votre site.

Description: Choisissez de montrer ou pas le texte de la description du flux.

Images: Choisissez de montrer ou pas l'image de la description du flux.

Articles: Entrez ici le nombre d'articles RSS à afficher. Si ce nombre est plus grand que le nombre fixé par le site concerné, c'est ce dernier qui sera utilisé.

Description: Sélectionnez si vous voulez afficher la description des articles. Si vous sélectionnez 'Non', seul le titre apparaitra.

Nombre de Mots: Choisissez le nombre maximal de mots visibles disponibles pour la description du Nouvel Article. 0 affichera le texte en entier.

Activer le Cache: Sélectionnez la mise en cache ou pas du contenu de ce module.

 

Personnalisation

Utilisez l'éditeur WYSIWYG pour ajouter votre contenu. Le contenu sera affiché avec une mise en forme du module.

 

Pages / Eléments

La liste montre tous les menus disponibles ainsi que leurs articles. A partir d'eux, vous pouvez sélectionner quand le module sera visible.

  • Vous pouvez sélectionner plus d'un articles du menu, en tenant la touche CTRL enfoncé.
  • Si vous voulez que le module soit visible dans chaque page, sélectionnez "Tous."
  • Si vous voulez qu'il ne soit pas visible, sélectionnez "Aucun".
