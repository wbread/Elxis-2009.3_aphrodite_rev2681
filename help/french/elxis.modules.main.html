Gestionnaire de Modules

Les modules sont utilisés pour afficher des articles d'information ou des caractéristiques interactives partout dans le site Elxis. Ils sont positionnés dans des emplacements définies dans les Templates, cela est appelé positions du module. Ces positions agissent comme des emplacements pour un ou plusieurs modules.

Les exemples de modules sont: Menus, Connexion et formulaires de Recherche. Les Modules HTML simples peuvent être créés directement du Gestionnaire des Modules. Dans les cas plus compliqués, ils devront être développés et installés.

Le Gestionnaire des modules liste et gére les modules installés.

 

Filtre

Vous pouvez utiliser le Champ Filtre pour limiter une recherche spécifique des modules. Vous pouvez saisir une phrase ou une partie et appuyer sur la touche <Entrée>. Le tableau sera mis àjour et n'affichera que les modules avec les caractères compris dans le titre. Si vous voulez voir à nouveau tous les modules, laissez le champ vide et appuyez sur la touche <Entrée>.

 

Colonne Filtres

Vous remarquerez que plusieurs colonnes ont une fléche à côté de l'en-tête. Quand vous cliquez sur le menus contextuel filtre une fenêtre popup s'ouvre. Vous pouvez utiliser ce filtre pour sélectionner les modules que vous voulez afficher. Cette caractéristique est adressée aux sites web qui ont un grand nombre de modules. Il aide les administrateurs à ce concentrer uniquement aux modules qui les intéressent.

 

Icônes de la Barre d'Outils

Publié et Dépublié: Publie ou dépublie le ou les modules sélectionnés. Seuls les modules publiés sont visibles sur le site web.

Copier: Copier les modules sélectionnés, dupliqué. Copiez le nom des modules avec 'Copié de'. Bien sûr vous pouvez changer leur nom.

Nouveau: Créez un nouveau module de type mod_user.

Editer: Editer le module sélectionné.

Supprimer: Supprimer le(s) module(s) sélectionné(s).

Aide: Cet écran d'aide.

 

Colonnes

Nom du Module: C'est le nom du Module utilisé par le titre du module.

Publié: C'est le nom du module. Il peut être éditer et mis sur la page d'accueil de votre site.

Réordonner: Cette colonne vous permet de changer d'ordre d'affichage des modules publiés sur la partie public de votre site. Cliquez sur les flèches "haut" ou "bas" pour modifier l'ordre d'affichage et leurs positions.

Ordre: Ceci montre l'ordre des modules affichés en page d'accueil. Pour modifier cet ordre, saisissez le nombre que vous voullez voir présenter. Puis, cliquez sur l'icône "Ordre" à côté de ordre d'en-tête des colonnes.

Accès: Ceci montre qui peut voir les modules sur la page d'accueil d'une page Web. Cliquant sur le lien pour changer le niveau.

Position: Ceci montre dans quelle position du template le module sera affiché.

Pages: Cela montre dans quelles pages le module s'affichera. Les options incluses sont 'Variables' ou 'Aucun' si vous ne voulez par le voir.

ID: Numéro ID du Module.

Type: Affiche le type de Module, Par exemple 'mod_mainmenu' est un "type" de module utilisé par le Menu Utilisateur.

Langues: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

 

Pied de Page

En bas de page il y a le pied de page.

Navigation - Si le nombre de messages est plus grand que le nombre de messages affichés par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.
