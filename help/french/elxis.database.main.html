Gestionnaire de la Base de Données

Le gestionnaire de base de données, est un composant très utile pour les administrateurs, il les aide à gérer leur base de données Elxis, sans avoir besoin d'une application supplémentaire, comme phpMyAdmin.

Les administrateurs peuvent:

  • Voir la structure de la base.
  • Voir la structure des tables.
  • Prendre des informations de la base.
  • Videz la structure d'une base de données et les données dans le format SQL.
  • Videz la structure d'une base de données et les données dans le format XML.
  • Téléchargement des Dumps de la base ou la laisser sur le serveur.
  • Moniteur des Etats
  • Moniteur des Tables
  • Optimiser la base de donnéees
  • Réparer la base de données

Tous les fichiers sont encodés en UTF-8.

 

Icone de la Barre d'Outils

Gestion des Sauvegardes: Ouvre les gestionnaire de sauvegarde de la base de données: Page des Backups, la description fait partie d'une autre aide, pour pouvoir créer une nouvelle sauvegarde ou supprimer une ancienne.

Aide: Cet écran d'Aide.

 

La page est divisée en deux zones. La gauche où vous pouvez voir les tables existantes et la droite où vous pouvez effectuer des opérations sur la base de données.

 

Zone de Gauche

La zone de gauche montre toutes les tables de la base de données. Chaque ligne a deux icônes. "Parcourir" et "Structure".

Cliquer sur l'icone "Parcourir" affichera les données de la table. En-dessous de la table de données, il y a deux liens. Cliquez sur la "Structure des Tables" vous affichera les détails de la structure. Cliquez sur "Retour à la Base de Donénes" vous fera revenir à l'écran d'accueil.

Cliquer sur l'icone "Structure" vous fera afficher les détails de la structure. En dessous de la structure des tables, il y a deux liens. Cliquez sur "Parcourir les tables" vous fera parcourir les données des tables. Cliquez sur "Retour à la Base de Donénes" vous fera revenir à l'écran d'accueil.

 

Zone de Droite

La zone de gauche montre deux onglets: "Information" et "Sauver Base de données".

 

Onglet Information

L'onglet information afiiche les informations sur votre base de données.

Onglet Sauver Base

Cet onglet sert à sauvegarder des données de la base dans une structurer en XML ou au format SQL.

Comment sauver en XML
  • Choisissez le type de sauvegarde en XML.
  • Sélectionnez le type de sauvegarde.
    Choisissez "Complet/Full" pour faire une sauvegarde de la structure et des données. Choisissez "Structure Seule" pour faire une sauvegarde de la structure de votre base de données.
  • Sélectionnez la destination
    Choisissez "Sauver sur le Serveur" pour faire une sauvegarde locale que vous pourrez télécharger plus tard. Vous pouvez gérer vos sauvegardes via le gestionnaire: Backups.
    Choisissez "Télécharger" pour télécharger votre sauvegarde sur votre PC.
  • Cliquez sur le bouton "Backup XML" pour lancer la sauvegarde.

 

Comment sauver en SQL:
  • Choisissez le type de sauvegarde en SQL. Vous verrez alors d'autres options.
  • Sélectionnez le type de sauvegarde
    Choisissez "Structure & Données " pour faire une sauvegarde de la structure et des données. Choisissez "Structure Uniquement" pour faire une sauvegarde de la structure de la base de données. Choisissez "Données Uniquement" pour faire une sauvegarde des données.
  • Options:
    Verrouillez les Tables: Sélectionnez "Oui" pour verrouiller les tables durant la sauvegarde. Utile si la base de données contient des tables pleines de données.
    Syntaxe Complète: Sélectionnez "Oui" pour faire la sauvegarde en mode INSERT SQL complet. Recommandé.
    Passer les Tables: Sélectionnez "Oui" pour ajouter les commandes de fichier à la sauvegarde, et laisser tomber les tables existantes pendant la restauration.
    Tables: Sélectionnez les tables à sauvegarder. Choisissez "Toutes les Tables" pour sauvegarder toutes les tables. Choisissez "Sélection des tables", apparaitra une liste dans un menu déroulant des tables disponibles. Par défaut on sélectionne toutes les tables. Vous pouvez sélectionner ou non les tables que vous voulez, presser sur la touche CTRL et cliquez pour supprimer ou ajouter une table.
  • Sélectionner la destination
    Choisissez "Sauver sur le Serveur" pour faire une sauvegarde locale que vous pourrez télécharger plus tard. Vous pouvez gérer vos sauvegardes via le gestionnaire: Backups.
    Choisissez "Télécharger" pour télécharger votre sauvegarde sur votre PC.

 

Onglet Maintenance

Cet écran montre des informations au sujet de l'état et des performance de la base de données. Les administrateurs peuvent utiliser ces données pour optimiser leurs bases Elxis.

Optimiser: Optimise la base de données et améliore les performances.

Réparer: Répare toutes les erreurs existantes dans les tables de la base de données.
