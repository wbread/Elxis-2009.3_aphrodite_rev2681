Gestionnaire de Bannières

Cette page montre la liste des bannières que vous pouvez afficher sur votre site.

Icônes de la Barre d'Outils

Publier et Dépublier: Cliquez sur les icônes "Publier" ou "Dépublier" pour activer ou non n'importe quelle bannière sélectionnée.

Transférer: Ouvre une nouvelle fenêtre vous permettant de transférer un fichier ou une image depuis votre ordinateur vers le répertoire images/banners.

Nouveau: Cliquez sur cet icône pour ajouter une nouvelle bannière.

Editer: Cliquez sur cet icône pour modifier les détails de la bannière.

Supprimer: Cliquez sur cet icône pour supprimer une ou plusieurs bannière.

Aide: Cet écran d'aide.

 

Colonne Filtres

Vous remarquerez que plusieurs colonnes ont une flèche à côté de l'en-tête. Quand vous cliquez sur filtrer une fenêtre popups s'ouvre. Vous pouvez utiliser ces filtres pour sélectionner les bannières que vous voulez afficher.

 

Colonnes

Nom Bannière: Le nom donné à la bannière. Si vous cliquez sur ce nom, vous pouvez en éditer les paramètres qui y sont rattachés.

Nom du Client: Chaque bannière est associée à un client. C'est le nom de votre client.

Publié: Cela indique si la bannière est actuellement Publiée ou non vers l'affichage frontal du site. Cliquez sur l'icône de colonne pour en permuter l'état.

Impressions Effectuées: Ceci est le nombre de fois que votre bannière a été affichée sur votre site.

Impressions Restantes: Ceci est le nombre d'impressions restant à afficher si une limite a été inscrite lors de la création ou l'édition de la bannière.

Clics: Ceci est le nombre de clics effectués par les visiteurs du site sur cette bannière particulière.

% Clics: Ceci est le ratio en pourcentage du nombre de clics réalisés par rapport au nombre d'impressions (affichages). 1% voudrait dire que 1 personne sur 100 a cliqué sur la bannière.
