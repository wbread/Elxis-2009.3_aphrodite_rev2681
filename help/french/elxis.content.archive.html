Gestionnaire des Archives

Cette page affiche la liste de tous les éléments archivés. Vous pouvez filtrer l'affichage par Section, Catégorie et/ou Auteur.

Vous "sélectionnez" les éléments en cochant la case qui se trouve au début de chaque ligne.

 

Icones de la Barre d'Outils

Désarchiver: Replace l'élément à son emplacement d'origine.

Corbeille: Cliquez sur l'icône "Corbeille" pour envoyer les éléments à la poubelle. Ils ne sont pas complètement supprimés mais ne peuvent plus être affichés sur votre site. Vous pouvez restaurer ces éléments de contenus via le Gestionnaire de Corbeille.

 

Colonnes

Titre: C'est le titre de l'élément de contenu.

Ordre: Affiche l'ordre dans lequel sont présentés les éléments de contenu (le paramètre "Par Défaut" "Ordre Principal" est éditable via l'élément spécifique du Menu Archive ). Cliquez sur les icônes figurant des flèches pour déplacer les éléments vers le haut ou vers le bas dans la liste.

Catégorie: Ceci identifie le numéro de Catégorie à laquelle l'élément est attaché. Cliquez pour éditer la Catégorie.

Auteur: C'est le nom de l'Auteur de ce contenu. Cliquez pour éditer l'Auteur.

Date: C'est la date a laquelle le contenu a été créé.

Langues: C'est la(les) langue(s) des aticles ci-dessous.
