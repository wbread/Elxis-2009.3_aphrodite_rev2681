Gestionnaire de Liens Web

Le Gestionnaire de liens est la place d'où vous pouvez ajouter ou administrer les liensweb présentés sur ce site. Vous pouvez créer de nouveaux liens web, publier, dépublier ou les supprimer. Cette page montre une liste des liens web.

 

Barre d'Outils

Transfert: Ouvre un fenêtre popup pour vous permettre de transférer un fichier ou une image de votre PC dans le dossier /media.

Publié et Dépublié: Publier ou dépublier un lien sélectionné. Seuls les liens web publiés seront visibles sur le site.

Nouveau: Créer un nouveau lien web.

Supprimer: Supprime le lien web sélectionné.

Aide: Cet écran d'aide.

 

Filtre

Vous pouvez utiliser le champ Filtre pour limiter la liste des liens web ou pour une recherche spécifique. Vous pouvez saisir une phrase ou une partie d'un mot et appuyer sur la touche <Entrée>. Le tableau sera mis à jour et affichera uniquement les liens web avec les caractères souhaités. Si vous voulez réafficher tout l'ensemble des liens, laissez le champ vide et appuyez sur la touche <Entrée>.

 

Colonnes

Title: C'est le nom du lien. Cliquez dessus pour éditer le lien. Une icône cadenas à côté de lui montre qu'il est contrôlé. À l'enregistrement, cliquez sur le nom pour le modifier. Vous devez, alors, l' Enregistrer ou le Fermer.

Publié: Shows if the web link is published or not. Click to change state. Acts as a toggle switch.

Approuvé: Montre si le lien à été approuvé et s'il est en ligne (les utilisateurs peuvent vous soumettrent des liens, dans ce cas ils ne sont pas automatiquement approuvés)

Réordonner: Cliquez sur les icônes pour déplacer l'élement dans la liste vers le haut ou le bas.

Catégorie: C'est la catégorie auquelle appartient le lien.

Clics: C'est le nombre de fois ou le lien a été visité.

Langues: Affiche les langues du site disponibles. Si vous mettez à 'Tous', les liens seront visibles dans toutes les langues.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.

 

Titre: C'est le nom du lien. Cliquez dessus pour éditer le lien.

Catégorie: C'est la catégorie auquelle appartient le lien.

Clics: C'est le nombre de fois ou le lien a été visité.

Vérifié: TMontre le nom d'un utilisateur si le lien est verrouillé. Vous ne serez pas capable de modifier une entrée qui est verrouillé par un autre utilisateur. Elle peut être déverrouillée par un administrateur par le biais de la commande Sytème de la barre de menu, puis Global Chekin. L'entrée peut également être déverouillée par l'utilisateur qui l'a verrouillé en cliquant sur l'icône "Editer" du lien et en cliquant sur l'icône "Sauvegarder" ou "Annuler".

Approuvé: Montre si le lien à été approuvé pour la publication. les utilisateurs peuvent vous soumettrent des liens, dans ce cas ils ne sont pas automatiquement approuvés.

Réordonner: Cliquez sur les icônes pour déplacer l'élement dans la liste vers le haut ou le bas.

Pour Publier ou Dépublier un ou plusieurs lien(s), sélectionnez les cases à côté de leur(s) nom(s) et cliquez sur l'icône "Publier" ou "Dépublier"

Cliquez sur l'icône Nouveau pour ajouter un nouveau lien web.

Cliquez sur le nom d'un lien web pour éditer ces informations, ou sélectionnez la case près de son nom and cliquez sur l'icône "Editer".

Pour effacer un lien, sélectionnez la case à côté de son nom et cliquez sur l'icône "Supprimer". Un message vous demandera de confirmer la suppression. Sélectionnez "Oui" pour effacer le lien, ou "Non" pour annuler.
