Gestionnaire des Sondages

Cette page montre la liste des Sondages que vous pouvez afficher sur votre site. Vous pouvez éditer ou ajouter un nouveau sondage.

 

Icônes de la Barre d'Outils

Publié et Dépublié: Publie ou dépublie le ou les sondages sélectionnés. Seuls les sondages publiés sont visibles sur le site web.

Nouveau: Créer un nouveau sondage.

Editer: Editer un sondage sélectionné.

Supprimer: Supprimer le ou les sondage sélectionné(s).

Aide: Cet écran d'aide.

 

Colonnes

Titre du Sondage: C'est le nom du sondage. Si vous cliquez sur ce nom, vous pourrez éditer les paramètres qui y sont relatifs.Une icône verrou à coté permet de savoir si le sondage est en train d'être édité. Pour déverouiller, cliquez sur le nom pour l'éditer et sauvegardez.

Publié: Indique quand le sondage est publié. Cliquez sur l'icone pour définir l'état du sondage.

Verrouillé: Affiche si le sondage est verrouillé pour le public (les utilisateurs peuvent voir les résultats mais ne peuvent plus voter). Cliquez sur l'icone pour définir l'état du sondage.

Options: Pas utilisé.

Lag: Durée en secondes entre les votes d'un même utilisateur pour un sondage. Par défaut le temps est de 86.400 secondes (24 heures).
