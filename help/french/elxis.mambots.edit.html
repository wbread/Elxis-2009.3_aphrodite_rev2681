Bots: Nouveau/Editer

La page Bots: Nouveau/Editer vous permet d'acceder à l'édition des Bots du site

 

Icônes de la Barre d'Outils

Sauver: Cliquez ici pour sauvegarder les nouveaux paramètres et retournez dans le Manager des Bots.

Appliquer: Cliquez ici pour sauvegarder les nouveaux paramètres et maintenir la même fenêtre ouverte pour l'édition.

Annuler: Cliquez ici pour annuler toutes les modifications apportées et fermer la fenêtre. Toutes les modifications des Bots seront annulées.

 

Détails du Bot

Nom: Nom du Bot qui apparaît dans la liste de Bots installés. Ceci n'a aucun impact sur votre site.

Répertoire: Ceci montre dans quel réprtoire est placé le Bot.

Fichier Bot: Ceci est le nom de fichier qui apparaît dans le répertoire Bots.

Ordre du Bot: Choisissez l'ordre d'affichage de Bots.

Niveau d'Accès: Choisissez qui pourra voir ce contenu. Public, Enregistré ou Spécial.

Publier: Sélectionnez "Oui" pour l'activer sur le site. Sélectionnez "Non" pour le cacher.

Description: C'est une description courte sur la fonction du Bot.

 

Paramètres

Si disponibles, ils seront affichés ici.
