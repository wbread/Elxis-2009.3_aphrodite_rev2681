Le panneau de contrôle

Le panneau de contrôle est la première page visible lorsqu'on accède à la zone d'administration d'un site. A tout moment, vous pouvez revenir au panneau de contrôle en cliquant sur "Accueil" de la barre de menus. A partir du panneau de contrôle, vous pouvez facilement accéder aux fonctions d'administration les plus utilisées grâce aux 15 raccourcis sous forme d'icônes.

Gestion de la page d'accueil - Cliquez sur cette icône pour ouvrir le gestionnaire de la page d'accueil. Cela vous permettra de choisir quels articles seront affichés sur la page d'accueil.

Gestion des sections - Cliquez sur cette icône pour ouvrir le gestionnaire de sections. Cela vous permettra de modifier les sections.

Gestion des Categories - Cliquez sur cette icône pour accéder au gestionnaire de catégories. Cela vous permettra de modifier les catégories.

Tous les Articles - Cliquez sur l'icône de raccourci pour accéder à Tous les Articles du Contenu. La liste de Tous les Articles de vos Contenus. Vous pouvez modifier chacun de vos Articles.

Gestion des contenus statiques - Cliquez sur cette icône pour ouvrir le gestionnaire de contenus statiques. Cela vous permettra de modifier ces types d'articles.

Gestion des Médias - Cliquez sur cette icône pour ouvrir le gestionnaire de Médias. Vous serez en mesure de visualiser, d'ajouter, de supprimer les fichiers multimédias utilisés sur votre site.

Gestion des menus - Cliquez sur cette icône pour ouvrir le gestionnaire de menus. Vous pourrez modifier les menus.

Gestion des Utilisateurs - Cliquez sur cette icône pour accéder au gestionnaire des Utilisateurs. ous serez en mesure de modifier, ajouter, supprimer des utilisateurs enregistrés sur votre site.

Configuration générale - Cliquez sur cette icône pour modifier les configurations générales du site.

Accès - Cliquez sur cette icône pour ouvrir le gestionnaire d'accès. Vous pouvez utiliser le Gestionnaire d'accès pour ajouter, supprimer ou modifier des groupes d'utilisateurs et leurs droits d'accès.

Gestion de la base de données - Cliquez sur cette icône pour accéder au gestionnaire de base de données. Vous serez en mesure de contrôler ou d'effectuer des opérations de maintenance sur votre base de données.

Gestion des Langues - Cliquez sur cette icône pour accéder au gestionnaire de Langues du site. Vous aurez la possibilité d' ajouter, supprimer, éditer les langues. Pour les langues utilisées dans l'administration, veuillez sélectionner les langues de l'Administrateur au menu d'accueil.

Gestion de corbeille - Cliquez sur cette icône pour accéder au gestionnaire de supression. Cela vous permettra de suprimer définitivement ou restaurer les éléments placées dans le corbeille.

Outils - Cliquez sur cette icône pour accèder à la page des Outils. Vous serez en mesure de choisir et d'utiliser l'un des outils disponibles sur Elxis.

Aide - Cliquez sur cette icône pour afficher l'index des aides disponibles.

Softdisk - Cliquez sur cette icône pour accèder au système Softdisk.

 

On the right side of the screen there are five tabs:

Connectés - Affiche la liste de tous les utilisateurs connectés (aussi bien au Frontend que dans la zone d'administration). Cliquez sur le nom de l'utilisateur pour ouvrir la page de modification de son profil.

Composants - Pour séléctionner un composant à modifier.

Populaires - Pour afficher, par ordre croissant, la liste des articles les plus visités de votre site. Sont affichés les dates de créations des articles et leur nombre de visites. Cliquez sur un article pour le modifier.

Articles récents - Pour afficher la liste des derniers articles publiés. Sont également affichés les dates de rédaction des articles et leurs auteurs respectifs. Cliquez sur un article pour le modifier.

Statistiques des menus - Pour afficher la liste de tous les menus avec, respectivement, le nombre d'éléments qui y sont liés. Ce nombre correspond au total des menus créés et non pas uniquement celui des menus publiés. Cliquez sur un menu pour afficher la liste des éléments correspondant.
