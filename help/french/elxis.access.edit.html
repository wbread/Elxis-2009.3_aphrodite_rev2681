Gestion des Accès - Ajouter/Editer un Groupe d'utilisateurs

Cet écran est utilisé pour changer le nom du groupe des utilisateurs ainsi que le groupe parent auquel il appartient.

Note Importante: Veuillez utiliser cet écran avec prudence. Une mauvaise utilisation, peut endommager la configuration de votre site web.

 

Icônes de la barre d'outils

Sauver: Sauvegarde les changements.

Annuler: Annule les changements.

Aide: Cet écran d'aide.

 

Détails du Groupe

Nom du Groupe: Nom utilisé pour le groupe.

Groupe Parent: Sélectionnez par le menu déroulant le groupe parent auquel vous voulez qu'il soit attribué.

Groupe d'utilisateurs existant: Affiche la hiérarchie des groupes existant.
