Gestionnaire des Utilisateurs - Champs

Utiliser cet écran, vous pouvez ajouter des champs personnalisés au profil d'un utilisateur. Basé sur cette caractéristique vous pouvez créer des profils d'utilisateur personnalisés pour les besoins de votre site.

Cet écran d'aide liste les Tableaux et Edite/Modifie lesChamps Supplémentaires.

Liste des Tableaux

 

Icones de la barre d'outils

Nouveau: Créer un champ supplémentaire

Editer: Modifiez un champ supplémentaire. Vous devez en premier, activer la case à cocher à côté du nom de champ. Ou bien vous pouvez cliquer sur le nom de champ pour le modifier.

Supprimer: Supprime un champ supplémentaire.

Aide: Cet écran d'aide.

 

Colonnes

Nom: Le nom du champ comme il paraîtra dans le profil utilisateur.

Type: Montre le type du champ.

Publié: Montre si le champ supplémentaire est publié ou pas. Cliquez pour basculer.

Réordonner: Utilisez les flèches pour monter ou descendre la commande de champs supplémentaires.

Requis: Montre si le champ supplémentaire est exigé ou pas.

Profil: Montre si le champ supplémentaire est visible dans le profil de l'utilisateur.

Enregistrement: Montre si le champ supplémentaire est affiché pendant l'enregistrement.

Lecture Seule: Montre si le champs est en lecture seule

 

Editer/Nouveau Champs Personnalisés

Cet écran est utilisé pour modifier un champs existant ou pour en créer un nouveau.

Icones de la barre outils

Sauver: Sauvegarde les changements et retourne à l'écran précedent.

Appliquer: Sauvegarde les changements et retourne au même écran pour conserver l'édition.

Annuler: Annule les actions de cet écran. Vous retournerez à l'écran précedent.

Aide: Cet écran d'aide.

 

Détails

Nome: Saisissez ici le nom du champ supplémentaire comme il paraîtra dans les formulaires.

Type: Sélectionnez le type du nouveau champ. La zone des options est actualisée pour refléter les options disponibles pour ce type de champ. Vous pouvez choisir un des types suivants:

  • text - Vous pouvez spécifier la valeur par défaut et la longueur maximum.
  • select - Vous pouvez spécifier jusqu'à 20 valeurs et choisir laquelles sera présélectionnées.
  • radio - Vous pouvez spécifier jusqu'à 20 valeurs et choisir laquelles sera présélectionnées.
  • checkbox - Vous pouvez spécifier jusqu'à 20 valeurs et choisir laquelles sera présélectionnées.
  • hidden - Vous pouvez spécifier la valeur par défaut.

Publié: Choisissez si ce champ sera publié ou pas.

Requis: Choisissez si ce champ est requis.

Afficher dans l'enregistrement: Si vous voulez que ce champ soit disponible dans le formulaire d'inscription, vous devez mettre cet option à Oui.

Afficher dans le Profil: Si vous voulez que ce champ soit visible dans le profil de l'utilisateur, vous devez mettre cette option à Oui.

Lecture Seule: Si vous voulez faire que ce champs soit en lecture seule, vous devez mettre l'option à Oui.

Options d'Alignement: Sélectionnez l'alignement pour les options. Les valeurs possibles sont: Horizontal, vertical.

Aperçu:Ce n'est pas une option editable. Après avoir Enregistré ou Appliquez les changements à un champ, vous pouvez afficher un aperçu.
