Gestionnaire de Ponts(Bridge)

Les ponts ou Bridges est un système unique intégré au CMS Elxis et vous servira pour intégrer une application tièrces dans Elxis. Cet écran affiche les Ponts installés sur votre site.

Les ponts sont installés à part.

 

Icones de la Barre d'Outils

Publier/Dépublier: Cliquez sur l'icone 'Publier' ou 'Dépublier' pour publier ou dépublier un pont.

Editer: Cliquer sur l'icone du pont à éditer pour accèder à la configuration des paramètres. Vous pouvez cliquer également directement sur le nom du pont pour l'éditer.

Désinstaller:Sélectionner un pont et cliquez sur désinstaller pour supprimer complétement le Pont.

Activer/Désactiver un système de Ponts: Cliquez sur l'icone d'état du Pont système pour l'activer. Quand un système de ponts est désactivé, toutes les fonctions des ponts comme les modules les api sont stoppés. Utilisez cette option si vous n'utilisez pas de pont sur votre site. Vous pouvez le ré-activer chaque fois que vous le souhaitez.

Aide: Cet écran d'aide.

 

Colonnes

Ponts: Nom du pont.

Publié: Affiche les ponts publiés ou pas. Cliquez pour changer son état. Agir comme un commutateur.

Description: Une description courte des fonctionnalités du pont.

Auteur: Nom de son auteur.

Version: Version du Pont.

Date: Info sur la date.

 

Enregistrement

Registry keeps a list of all available Bridges for Elxis and a unique identification number for each of them. Do not modify Registry by hand! Keep your registry file up to date. If you experience problems saving new registry make sure that /administrator/components/com_bridge/registry/registry.ini file is writable.

 

Edition des Ponts

L'enregistrment garde une liste des Ponts disponibles pour Elxis et un numéro d'identification unique pour chacun d'eux . Ne modifiez pas les enregistrements manuellement! Gardez votre fichier à jour. Si vous rencontrez des problèmes et vous avez de l'expérience, vous pouvez enregistrez mais soyez sur que le fichier registry.ini est modifiable.
