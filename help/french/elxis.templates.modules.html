Position des Modules

La page des Positions du Module de Elxis vous permet d'entrer les détails des positions du module que vous utilisez dans votre modèle.

Si vous devriez utiliser des positions qui n'existent pas dans Elxis par défaut vous pouvez ajouter simplement un nom pour une position du module et une description. La description est optionnelle.

Pour créer une nouvelle position du module dans votre modèle, vous devez insérer ce code base dans votre modèle pour pouvoir assigner des positions particulières aux modules.

 

Icônes de la Barre d'Outils

Sauver: Sauvegarde les changements et retourne à l'écran précédent.

Annuler: Annule les changements et retourne à l'écran précédent.

Aide: Cet écran d'aide.

 

Colonnes

Position: Nom de la position du Module. Veuillez ne pas changer les noms existants, à moins que vous sachiez ce que vous faites.

Description: Vous pouvez entrer des commentaires au sujet d'une position. Uniquement à usage interne.
