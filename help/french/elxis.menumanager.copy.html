Gestion de Menu: Copier un Menu

Cette page vous permet de copier un Menu et tous ces Éléments de Menu. Une liste affiche le nom du Menu et des Éléments de Menu copiés.

 

Barre d'Outils

Copier: Un nouveau "Nom de Menu du" module sera créé dans la liste du Gestionnaire de Menu Manager. Un nouveau module mod_mainmenu, avec comme "Nom de Module" sera alors créé. Les Paramètres de création du module peuvent être édité au travers de Modules -> Modules du Site.

Annuler: Cliquez sur cet icone si vous voulez annuler l'action en cours.

Aide: Cet écran d'aide.

 

Champs

Nouveau nom de Menu: C'est le nom d'identification utilisée par Elxis pour identifier ce menu dans le code ; il doit être unique. Il est recommandé de ne pas utiliser l'espace dans ce nom. Par exemple dans le noyau Elxis le nom du Menu Utilisateur est usermenu et le Menu principal est nommé mainmenu.

Nouveau nom de Module: Ceci est le nom donné au nouveau module mod_mainmenu qui est créé automatiquement lorsque vous sauvegarder ce menu. C'est aussi le Nom du Menu tel qu'il apparaîtra sur la page si vous choisissez de l'afficher (ceci peut être changé en éditant le Module). Dans l'exemple décrit au-dessus il y a User Menu et Main Menu.
