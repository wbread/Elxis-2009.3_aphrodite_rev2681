Gestionnaire des Templates
[Site, Administrateur, Ecran de Connexion]

Elxis utilisent trois types de templates:

  • Templates du site, c'est l'aspect graphique de la partie public.
  • Templates d'Administration, c'est l'aspect graphique de la partie administrative du site.
  • Ecran de Connexion, c'est l'aspect graphique de l'écran d'accueil de l'administration.

Tous les templates utilisent le même style de Gestionnaire de Templates. Donc cette description, s'applique à tous

Utilisez le Gestionnaire de Template pour afficher tous les modèles installés. Vous pouvez également éditer les Templates. Les Templates affectent la direction votre site, serveur et apparence des écrans de connexion. Vous devez en mettre un par défaut. Vous pouvez assigner d'autres modèles aux Menux/Articles/pages que vous aurez choisi. Maintenant, vous pouvez changer si vous le souhaitez, le look de votre site, suivant le contexte.

 

Templates du Site

Ce sont les templates utilisées sur la partie-public. Il y a différent contrôle pour mettre en valeur votre site web.

 

Barre d'outils

Défaut: Assigne ce template par défaut à touts les éléments de menu/pages.

Assign: Ouvre une page sur laquelle vous pouvez assigner ce template à des éléments de menu/pages spécifiques. Ce choix annulera le template par défaut pour les pages choisies. Si vous choisissez 'Aucun', l'effet sera identique au paramètre 'Défaut'. Cette fonction n'est pas disponible pour les templates de l'administration.
Note: Affecté au modèle par défaut il sera supplanté par le module du Sélecteur de Template sur la partie-public. Tout modèle affiché dans cette liste sera répertorié dans le module Sélecteur de Template.Affecté au modèle par défaut il sera supplanté par le module du Sélecteur de Template sur la partie-public. Tout modèle affiché dans cette liste sera répertorié dans le module Sélecteur de Template.

Nouveau: Permet de transférer/installer un nouveau template sur votre site.

Editeur HTML: Sélectionnez un modèle et cliquez sur cette icône pour modifier son fichier index.php.

Editeur CSS: Permet d'éditer le fichier templates/votre-template/css/template_css.css qui contient les définitions des classes CSS pour ce template particulier.

Supprimer: Vous permet de supprimer un template. Il vous sera demandé de confirmer ce choix. Sélectionner 'Oui' pour supprimer ou 'Non' pour abandonner. Le template sera définitivement supprimé de votre base de données.

Aide: Cet écran d'aide.

Aperçu du Template: Si cette option est sélectionnée lorsque vous survollez avec la souris sur un template, vous aurez alors un petit aperçu du template. Cette image est prise du fichier template_thumbnail.png qui est inclus avec chaque template.

Colonnes

Nom: Nom du template.

Défaut: C'est le template qui est publié actuellement comme template du site (seul un peut être publié à la fois).

Assigné: Cela dit si le template est assigné dynamiquement à une des pages sur le site.

Auteur: Nom de l'auteur du Template.

Version: Donne la version du template.

Date: Donne la date de création du template.

Email Auteur: Email de l'auteur.

Templates d'Administration

Ce sont les templates utilisées sur l'administration. Il y a différent contrôle pour mettre en valeur votre interface d'administration.

Barre d'outils

Défaut: Sélectionnez ce template et cliquez sur l'icône pour mettre le template par défaut.

Nouveau: Cliquez sur l'icône pour transférer/installer une nouvelle template d'administration pour le site. Ouvre une nouvelle page.

Editeur HTML: Sélectionnez le template et cliquez sur l'icône pour éditer le fichier index.php.

Editeur CSS: Sélectionnez le template et cliquez sur l'icône pour éditer le fichier template_css.css qui contient les classes CSS classes de la template.

Défaut: C'est le template qui est publié actuellement comme template du site (seul un peut être publié à la fois).

Auteur: Nom de l'auteur du Template.
