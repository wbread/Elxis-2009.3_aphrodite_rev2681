Gestionnaire de Menu : Menu Principal

Cette page montre une liste des Éléments de Menu associés au menu que vous venez de choisir.

Niveaux max: Sélectionne le nombre maximum de Niveau que l'arborescence de Menu affichera sur cette page d'administration.

Filtre: Saisissez dans la zone de texte pour chercher du texte dans les titres des les Éléments de Menu.

 

Icône de la Barre d'Outils

Nouveau: Cliquez sur l'icône pour ajouter un Élément de Menu.

Editer: Cliquez sur l'icône pour éditer un Élément de Menu.

Publier et Dépublier: Ces icônes ont la même fonction que la colonne Publier. Cliquez l'un ou l'autre de ces icônes pour publier ou dépublier n'importe quel élément.Vous pouvez utilisez ces icônes avec plusieurs éléments sélectionnés.

Déplacer: Cliquez sur cette icône pour déplacer un élement de menu sélectionné dans un autre menu.

Copier: Cliquez sur cette icône pour copier un élement de menu sélectionné dans un autre menu.

Corbeille: Cliquez sur cette icône pour envoyer certains éléments du menu à la corbeille. Les éléments du menu peuvent être récupérées par l'intermédiaire du menu de Admin: Site->Corbeille.

Aide: Cet écran d'aide.

 

Colonnes

Élément de menu: Ceci est le nom du lien tel qu'il apparaît sur votre site . Cliquez sur ce nom de deux façons à éditer si les paramètres relatifs à cet Élément. Un icône cadenas proche montre qui il a été bloqué. Pour le débloquer, cliquez-le pour ouvrir l'écran propriétés et cliquez sur l'icône 'Sauver' ou 'Fermer' .

Publié: Ceci affiche si un Élément de Menu est publié ou non publié. Cliquez sur l'icône pour changer son état.

Réordonné: Ceci affiche l'ordre des Éléments de Menu dans le Menu. Cliquez sur les icônes flèche pour monter ou descendre l'Élément dans la liste.

Ordre: Ceci affiche l'ordre des Éléments de Menu dans le Menu. Pour changer l'ordre saisissez le numéro d'ordre que vous souhaitez avoir. Puis cliquez sur l'icône "Sauver l'Ordre" dans la colonne de tête.

Accès: Cliquez sur le Niveau d' Accès pour définir un autre Niveau d' Accès pour cet Élément de Menu. Une fenêtre pop-up apparaîtra à tous les groupes d'utilisateurs. Choisissez celle que vous souhaitez. Seuls les membres qui appartiennent à des groupes seront en mesure de le voir.

ID Elément: Ceci montre l' ID pour cet Élément de Menu spécifique dans la base de donnée.

Type: Ceci est le type de liens utilisés pour cet Élément de menu. Si il s'agit d'un Article ou d'un contenu Statique vous pouvez éditer directement le contenu en cliquant le type de cet Élément de Menu.

CID: Ceci montre l'identifiant 'componentid' pour cet Élément spécifique de Menu dans la base de données.

Langues:Ceci montre la langue sous laquelle cet article du menu sera disponible. Seulement si l'utilisateur a choisi d'afficher le site en utilisant le(s) langue(s) spécifieé(s), il sera en mesure de voir le Menu. Sélectionnez 'Tous' pour le mettre à toutes les langues.
