Gestionnaire des flux de nouvelles : Nouveau/Editer

Cette page vous permet de créer ou de modifier les flux de nouvelles

 

Icône de la Barre d'Outils

Sauver: Sauvegardez les changements et retournez à l'écran précédent.

Annuler: Annule la création de menu. Aucun changements ne sera fait. Les changements ne sont pas pris en compte si vous éditez un menu.

Aide: Cet écran d'aide.

 

Détails

Catégorie: Sélectionnez ici la catégorie pour ce flux.

Nom: C'est le nom de menu de l'article, comme il sera montré sur votre site.

Titre SEO: Saisissez ici le titre de la page pour le moteur de recherche. Utilisez que des caractères latin. Pour les espaces mettez de '-' ou '_' . Ce sera a utilisé seulement si SEO Pro est activé, mais c'est une bonne habitude pour toujours le remplir.

  • Titre SEO Suggéré - Cliquez ici pour demander à Elxis de vous suggére un titre SEO. Cette suggestion de titre SEO sera basé sur le champ 'Nom' .
  • Titre SEO Validé - Après avoir saisi un titre SEO, vous devez cliquer sur ce lien pour vérifier si il est valide. Si ce n'est pas le cas trouvez une solution !

Lien: Entrez ici le lien du flux comme défini par la source.

Nombre d'Articles: Définissez le nombre maximal d'articles qui seront affichés dans ce flux.

Temps du Cache (en secondes): Saisissez ici le temps (en secondes) entre chaque rafraîchissement des flux. Entre chaques rafraîchissements, les flux seront lus à partir du cache. Cette approche améliore la performance du site.

Ordre: Sélectionnez l'ordre des listes, relatif aux autres flux de cette Catégorie. Ce paramètre sera seulement disponible après avoir enregistré ou appliqué les changements pour la première fois. Vous conservez aussi l'ordre de modification dans le Gestionnaire des Flux de Nouvelles.

Publié: Sélectionnez "Oui" pour le publier sur le site. Sélectionnez "Non" pour le cacher.
