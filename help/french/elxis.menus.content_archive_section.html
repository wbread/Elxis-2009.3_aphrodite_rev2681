Menu Archive:
Blog - Section Archive de Contenu

Cet écran, vous permet d'ajouter un nouvel élément au menu "Blog - Section Archive de Contenu". Un article de menu de ce type présente les articles archivés qui appartiennent à une ou plusieurs sections sous formes d'articles blog. De chaque article est présenté seulement le Texte d'Intro. Si l'article a aussi un Texte Principal rempli alors le lien "Lire la Suite" sera présentés. En cliquant sur ce lien vous pourrez voir l'article complet.

Vous pouvez définir plusieurs paramètres pour ce type d'élément de menu, cela contrôle le nombre, la direction de la table et les lignes présentées.

 

Icône de la barre Outils

Retour:Retour à l'écran précedent et sélectionnez un nouveau type d'élément de menu à créer.

Sauver: Sauvegarde les changements et retour à l'écran principal.

Appliquer: Sauvegarde les changements et reviens à l'écran en cours.

Annuler: Annule toutes modification, création sur le nouveau menu. Les changements ne sont donc pas enregistrés.

Aide: Cet écran d'aide.

 

Détails

Nom: Inscrivez le nom de l'élément de menu tel que vous voulez qu'il apparaisse sur votre site.

Catégorie: Sélectionnez la catégorie de contact que vous voulez montrer.

URL: Le lien pour cet élément apparaîtra après avoir cliqué "sauver/appliquer".

Elément Parent: Sélectionnez "top" si vous vous voulez que votre nouvel élément de menu apparaisse dans le menu de navigation principal (main menu). Vous pouvez choisir de mettre cet élément en sous menu d'un élément du menu principal en choisissant ce dernier comme élément parent.

Ordre: Ajoute les nouveaux liens à la suite des éléments de menu déjà existants. Après avoir sauvegardé l'élément, vous pouvez lui choisir une nouvelle place dans la liste en la sélectionnant dans le menu déroulant. Sa nouvelle place sera APRES le lien sélectionné.

Langue: Sélectionnez la langue que vous voulez avec cet élément. Si vous sélectionnez "Toutes", ce sera disponible et visualisable, sous tous les langages.

Niveau d'Accès: Choisir qui peut avoir accès à cet élément.

Publié: Choisissez si votre nouvel élément sera défini comme publié oui ou non. Par défaut, il le sera.

 

Paramètres

Image du Menu: Si vous voulez vous pouvez montrer une petite image à côté de l'article du menu. Sélectionnez cette image dans la liste déroulante.Note: Pour que l'image soit visible, les paramètres "Afficher les icône de Menu" pour le menu du module, doit être mis à "Oui".

Utiliser seulement l'image de menu: Si vous avez mis les paramètres à 'Oui', alors au lieu du nom de menu il sera affiché en utilisant l'image de menu.

Classe Suffixe de la Page: vous permet de personnaliser le style de cette page. Le suffixe que vous saisissez ici est automatiquement ajouté à la classe CSS pour cette page.

Bouton Retour: Contrôle pour activer oui ou non le bouton retour à la page précédente. Cliquez sur le lien "Retour" vous fera retourner à la page précedente. Le menu déroulant vous propose 3 options:

  • "Configuration Globale" - Montre ou cache le lien basée sur l'arrangement dans la Configuration Globale.
  • "Cacher" - Cachez le lien retour, en ignorant les paramètres Global.
  • "Afficher" - Affichez le lien retour, en ignorant les paramètres Global.

Titre de la Page: Affiche/Cache le titre de la page., tapez-le à la prochaine zone de texte et choisissez "Oui".

#En Tête: Mettez le nombre d'articles que vous voulez voir comme articles principaux (pleine largeur). Mettre à 0, n'affichera aucun article.

# Intro: Mettez le nombre d'articles que vous voulez montrer avec le texte Intro.

Colonnes: Mettez le nombre de colonnes que vous voulez pour que les articles soient triés.

# Liens: Mettez le nombre d'articles que vous voulez voir affiché comme liens.

Ordre de la Catégorie: Sélectionnez l'ordre des catégories qui seront affichées.

Ordre Primaire: Sélectionnez l'ordre des éléments de contenu qui seront affichés.

Pagination: Sélectionnez si vous voulez contrôler la pagination.

Résultats de Pagination: Masquez ou montrez les résultats de la pagination.

Description: Affiche ou cache le texte de description des catégories.

Description de l'Image: Affiche ou cache l'image qui est associée avec chaque catégorie.

MOSImages: Si les articles contiennent des indicateurs mosimage, vous pouvez choisir de les montrer ou pas dans l'affichage du blog.

Nom de la Section: Montrez ou masquez la section dans laquel l'article appartient.

Nom de Section cliquable: Si vous avez choisi de montrer le nom de section, vous pouvez le faire cliquable.

Titres Eléments: Sélectionnez si vous voulez que les titres des éléments soient visible ou pas.

Titres Cliquables: Si vous avez choisi de montrer le titre des articles, vous pouvez sélectionner ici pour agir aussi comme liens pour l'article entier.

Lire la Suite: Sélectionnez si vous voulez afficher le lien"Lire la Suite".

Votes: Sélectionnez si vous voulez afficher la possibilité de voter.

Nom de l'Auteur: Sélectionnez si vous voulez afficher le nom de l'auteur.

Date et Heure de Création: Sélectionnez si vous voulez afficher la date et heure de création.

Date et Heure de Modification: Sélectionnez si vous voulez afficher la date et heure de modification.

Icône RTF: Sélectionnez si vous voulez afficher afficher l'icone RTF.

Icône PDF: Sélectionnez si vous voulez afficher afficher l'icone PDF.

Icône Print: Sélectionnez si vous voulez afficher l'icone imprimer.

Icône Email: Sélectionnez si vous voulez afficher l'icone email.
