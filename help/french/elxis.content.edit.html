Articles: Nouveau/Éditer

Cette page permet de créer ou de modifier un élément de contenu. Sur le côté gauche, il ya deux éditeurs où vous pouvez ajouter du contenu dans l'espace de texte pour l' Intro et le corps. Sur la droite se trouve l'onglet Paramètres.

Intro c'est le texte qui est toujours affiché dans la vue du blog. Quand vous avez un long article à publier, vous pouvez le diviser dans les deux parties qui utilisent la zone principale de texte. Dans ce cas, Elxis affichera le texte Intro, suivi par un Lire la suite ou en Savoir Plus Cliquer sur le lien vous affichera l'article en entier (Intro + texte principal).

 

Icônes de la Barre d'outils

Aperçu: Ouvre une nouvelle fenêtre pour prévisualiser le contenu comme vous le verriez dans la partie publique de votre site.

Transférer: Ouvre une nouvelle fenêtre qui vous permet de mettre en ligne un fichier ou une image depuis votre ordinateur vers le dossier /media.

Sauver: Enregistre l'article et retourne au Gestionnaire.

Appliquer: Enregistre le contenu et le conserve à l'écran pour modification.

Annuler: Annule l'action. Toute modification est perdue. S'il a été ouvert pour édition, l'élément de contenu reste vérrouillée.

Aide: Cet écran d'aide

 

Détail de l'Article

Section: Choisissez ici la section à laquelle vous voulez que l'élément appartienne.

Catégorie: Choisissez ici la catégorie à laquelle vous voulez que l'élément appartienne. La liste déroulante affiche toutes les catégories disponibles dans la section choisie plus haut.

Titre: Saisissez ici le titre de l'Article. C'est le titre qui apparaitra sur la page ou dans le tableau listant cet élément.

Titre SEO: Saisissez ici le titre de la page pour les moteurs de recherches. Utilisez uniquement des caractères latin. Plutôt que d'utiliser les espaces préférez les caractères '-' ou '_' . Il ne sera utilisé que si SEO Pro est activé mais c'est une bonne habitude de toujours le remplir.

  • Titre SEO suggéré - Cliquez pour laisser Elxis suggérer un titre SEO. Le titre SEO suggéré sera basé sur le champ 'Nom' .
  • Titre SEO validé - Après avoir saisi le titre SEO, Vous pouvez cliquer sur ce lien afin de vérifier si il est valide. Si il ne l'est pas veuillez corriger!

Texte d'introduction (obligatoire): Saisissez ici votre intro texte de votre article, en utilisant l'éditeur WYSIWYG. C'est la partie affichée dans les listes de Blog, à la fois pour les Catégories et les Sections. Vous pouvez aussi choisir de cacher cette partie quand l'article est affiché..

Texte principal (facultatif): C'est le corps de votre article. Il n'est montré que quand l'article est affiché en vue complète, et est caché dans les Listes de Blog. Editez le avec l'éditeur WYSIWYG.

 

En utilisant l'éditeur WYSIWYG, il existe un certain nombre de boutons. Par défaut, il y a 'Image', 'Page Break' et 'Flash Video'. Ce sont des raccourcis pour insérer des codes. Par exemple, en cliquant sur le bouton 'Page Break', cela inséra '{mospagebreak}'. Alternativement, vous pouvez insérert des codes manuellement. L'éffet sera le même.
Vous pouvez installer d'autres Bots.

 

Onglet Paramètres

La Zone d'Attributs pour le contenu est divisée en 6 onglets:

  • Publié
  • Images
  • Paramètres
  • Info Méta
  • Lien de Menu
  • Liens connexes

 

Onglet Publication

Informations de Publication, date, auteur, langue et niveau d'accès.

Afficher sur la page d'accueil: Cochez cette case si vous voulez que ce contenu soit affecté au Composant Page d'Accueil.

Publié: Cochez cette case pour afficher cet élément sur votre site.

Langues: Sélectionner la(les) langue(s) que vous voulez attribuer à l'article. Si vous voulez que cela soit disponible pour toutes les langues sélectionnez "Tous". Vous pouvez sélectionner plus d'une langue, en enfoncant la touche CTRL et en sélectionnant les langues souhaitées.

Niveau d'accès: Choisissez qui sera autorisé à voir cet élément. Public, Membre, ou Spécial.

Alias d'Auteur: Vous pouvez publier le nom de l'auteur pour un élément de contenu. Vous pouvez choisir un Utilisateur Inscrit dans la liste déroulante ou taper n'importe quel nom dans cette zone (un auteur qui n'a pas de compte ou du contenu créé par un groupe de plusieurs auteurs).

Changer le Créateur: Permet de transférer la propriété de ce contenu à un autre utilisateur enregistré.

Ordre/Tri: hoisissez l'ordre dans lequel cet élément est listé par rapport aux autres de sa catégorie. Les choix disponibles dans la liste déroulante s'affichent une fois les changements appliqués ou enregistrés.

Écraser la date de création: Affiche la date de création du contenu. Vous pouvez la changer si nécessaire. Initiallement, la date de création actuelle est affichée.

Début de Publication: Choisissez la date à laquelle ce document doit être publié sur le site. Par défaut, les nouveaux éléments de contenus démarreront desè publication.

Fin de Publication: Si vous le souhaitez, vous pouvez définir la date à laquelle le contenu expire. Si vous voulez que cela soit toujours visible (ce qui est l'option par défaut), entrez le mot Jamais.

ID de contenu: C'est l'ID (identifiant) du contenu dans la base de données. Non modifiable.

Etat: Affiche l'état courant du document. Non modifiable.

Clics: Affiche le nombre de fois que cet élément a été vu. Le bouton re-initialiser le nombre vous permet de remettre le compteur à zéro.

Révision: Indique le nombre de révisions subies. Non modifiable.

Création: Date à laquelle le contenu a été créé. Non modifiable.

Par: Nom de l'utilisateur qui a créé l'élément. Non modifiable.

Dernière modification: Date à laquelle ce document a été modifié pour la dernière fois, s'il l'a été. Non modifiable.

Par: Nom du dernier utilisateur ayant effectué une modification, si le document l'a été. Non modifiable.

Commentaires: Ce champ est visible uniquement lorsque vous êtes en train d'éditer un contenu soumis. Affiche les observations faites par l'auteur de l'article. N'est pas visible du public.

 

Onglet Images

Ces paramètres sont utilisés uniquement si vous avez utilisé mosImages pour insérer des images dans cet élément de contenu. Les MosImages sont insérées en utilisant le bot (mosimage). Si vous ne l'avez pas, vous pouvez l'ignorer sans risque.

Sous-dossier: Choisissez le sous dossier dans lequel vous voulez prendre les images. Celles-ci sont dans images/stories.

Images de Galerie: Cette liste déroulante propose la liste des images disponibles dans le sous-dossier spécifique. Mettez un ou plusieurs noms d'image en surbrillance et cliquez sir le bouton "Ajouter". Le ou les noms seront affichés dans les Images de Contenu.

Images de Contenu: Les boutons "Haut" et "Bas" déterminent l'ordre des images si cette page en contient plus d'une. Les images seront affichées en fonction des occurrences de vos balises dans le contenu.

Modifier l'image sélectionnée: Utilisez ces commandes pour modifier les paramètres de l'image sélectionnée. Vous devez cliquer sur 'Appliquer' pour sauvegarder les changements.
Quand vous sélectionnez une image dans "Images de Contenu", ces contrôles sont activés.

Source: Affiche le nom par défaut de l'image sélectionnée. Vous pouvez modifier ce nom.

Alignement de l'image: Vous permet de choisir entre : Aucun, Centrer, Gauche, ou Droite.

Texte Alternatif: Permet de saisir un texte alternatif (Alt) pour l'image.

Bordure: Saisissez dans cette zone la largeur en pixels de la bordure que vous souhaitez autour de l'image. 0 pour pas de bordure.

Légende: Saisissez une légende pour l'image.

Alignement de la légende: Choisissez entre Aucun, Centrer, Gauche, ou Droite.

Largeur: Saisissez une valeur en pixels pour remplacer la largeur réelle de l'image.

Appliquer: Cliquez sur ce bouton pour appliquer les paramètres définis pour cette image.

 

Onglet Paramètres

Ces paramètres contrôlent ce que vous voyez quand vous affichez un élément de contenu en pleine page.

Suffixe de Classe de Page: Permet de définir des styles personnalisés pour cette page. Le suffixe que vous tapez ici est ajouté automatiquement aux classes CSS de cette page, et devra être défini dans la feuille de style CSS de votre modèle pour pouvoir être utilisé.

Suffixe Article: Permet de définir des styles pour personnaliser l'affichage de cet élément de contenu en vue de blog. Le suffixe que vous saisissez sera ajouté automatiquement à la classes CSS utilisé pour afficher des blogs, et devra être définie dans votre modèle de feuille de style CSS.

Bouton Retour: Contrôle l'affichage ou non d'un lien "Retour" qui apparaît en bas de page et renvoie le visiteur à la page précédemment affichée. La liste déroulante propose 3 solutions:

  • "Utilisation globale" - pour utiliser le paramètre défini dans la configuration globale du Site
  • "Cacher" - Masque le lien Retour, primordial du réglage Global.
  • "Afficher" - Affiche le lien Retour, primordial du réglage Global.

Afficher/cacher le titre de page: Cache ou montre le texte du titre de la page (défini dans le champ décrit ci-dessus).

Titres liés: Le titre de l'article vue du blog, peut également agir en tant que lien vers le texte intégral. Choisissez si vous voulez ce comportement.

Texte d'Intro: Si mis à Oui, lorsque vous afficherez l'Article en entier, le texte d'Intro sera caché. On peut afficher seulement le texte princnipal. Si a mis à Non, seul l'article en entier sera affiché.

Nom de Section: Cache ou montre le nom de la section à laquelle chaque élément appartient. Si cette option est activée, le nom de la section sera affiché sous le titre de chaque élément sur la page d'accueil.

Liens sur Noms de Section: Rend chaque nom de section cliquable. S'applique uniquement quand l'option "Nom de Section" (décrite ci-dessus) est active.

Nom de Catégorie: Cache ou montre le nom de la catégorie à laquelle chaque élément appartient. Si cette option est activée, le nom de la catégorie sera affiché sous le titre de chaque élément sur la page d'accueil.

Liens sur Noms de Catégories: Rend chaque nom de catégorie cliquable. S'applique uniquement quand l'option "Nom de Catégorie" (décrite ci-dessus) est active.

Pour chacun des 8 éléments ci-dessous, vous pouvez utiliser l'option choisie dans la Configuration Globale (Site -> Configuration Globale) en sélectionnant "Utiliser Global", ou vous pouvez écraser le paramètre global en sélectionnant "Montrer" ou "Cacher". Écraser un paramètre ici ne modifie pas la configuration globale et n'affecte pas les paramètres pour le reste du site.

Notes de l'Élément: Cache ou montre les notes données au contenu par les visiteurs.

Noms des auteurs: Cache ou montre les noms des auteurs des contenus.

Date et heure de création: Cache ou montre la date et l'heure à laquelle les contenus ont été créés.

Date et heure de Modification: Cache ou montre la date et l'heure à laquelle les contenus ont été modifiés.

Icône PDF: Cache ou montre l'icône et le lien vers la version PDF des contenus.

Icône RTF: Cache ou montre l'icône et le lien vers la version RTF des contenus.

Icône Imprimer: Cache ou montre l'icône et le lien vers la version imprimable des contenus.

Icône Email: Cache ou montre l'icône et le lien vers la page qui permet aux visiteurs d'envoyer des contenus par courrier électronique.

Lexique Référence: Une expression permettant de référencer le contenu (comme une aide).

 

Infos Meta

Entrez ici les informations META de l'article. Les informations META "decrivent" les contenus des articles et sont utilisés par les moteurs de recherches.

Description: Cette information est insérée dans les en-têtes de vos contenus. Les moteurs de recherche l'utilisent couramment.

Mots-Clés: Ces mots clés sont reliés aux informations d'en-tête fournies aux moteurs de recherche. A séparer par des virgules.

  • Utilisé comme des articles. Ils sont les maîtres mots qui caractérisent cet article.
  • Ces mots-clés sont la base du module What's Related qui fourni des liens automatiques vers d'autres contenus qui partagent un mot clé similaire.

Ajouter Section/Cat/Titre: Cliquer sur ce bouton est un moyen commode d'insérer les informations de l'élément dans le champ mot clés, comprenant le titre du contenu, le nom de la section et celui de la catégorie.

 

Lier au Menu

Crée un "Lien – Article" dans le menu sélectionné.

Sélectionnez un menu: Cliquez ici une fois les menus de votre site définis.

Nom de l'Élément de menu: Nom dans le menu tel qu'il apparaitra sur le site.

Lier au menu: Cliquez sur ce bouton pour créer l'élément de menu.

Éléments de menu existants: Affiche une liste de tous les éléments de menu de type "Lien – Article" référençant ce contenu.

 

Liens connexes

Vous pouvez ajouter un certain nombre de liens (jusqu'à 5) à la fin de l'article, d'autres informations, ce qui est conforme au présent article. Les liens peuvent être externes (conduisant à un autre site Web), ou internes (liens vers d'autres articles ou de même teneur).

Titres 1-5: Saisissez le texte qui devra agir en tant que lien. Par exemple, 'Pour plus d'info visitez GO UP Inc.'.

Liens 1-5: Saisissez un lien URL. Pour créer un lien externe, veuillez ajouter 'http://' devant votre url (par exp. http://www.goup.gr). Pour créer un lien interne, ne mettez pas 'http://' (par exp. index.php).
