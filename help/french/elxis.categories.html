Gestionnaire des Catégories

Cette page présente la liste des catégories définies sur votre site. Elles sont triées par section à laquelle elles appartiennent. La liste déroulante "Sélectionnez la section" vous permet d'afficher les catégories appartenant à une section spécifique.

Vous pouvez utiliser cet écran pour ajouter, éditer, supprimer, déplacer ou copier les Catégories.

Cette page est appelée à partir de:

  • Eléments de Contenus
  • Contacts
  • Flux de nouvelles
  • Liens Web

 

Barre d'outils

Publication et dépublication: Même fonction que "Publié" dans les titres. Vous pouvez sélectionner plusieurs choix.

Nouveau: Créer une nouvelle catégorie.

Déplacer: Ouvre une fenêtre "Déplacer cette catégorie" où vous pouvez déplacer la catégorie et tout son contenu dans une autre section existante. Vous pouvez en choisir plusieurs.

Copier: Ouvre une fenêtre "Copier cette catégorie" où vous pouvez copier la catégorie et tout son contenu dans une autre section existante. Vous pouvez en choisir plusieurs.(les contenus copiés conservent le même nom).

Editer: Modifier les paramètres d'une catégorie existante.

Supprimer: Vous permet de supprimer la catégorie choisie. Un "pop-up" d'alerte vous permet de confirmer ce choix. Cette action ne peut pas être effectuée si la catégorie n'est pas vide. Dans ce dernier cas, vous devrez d'abord supprimer son contenu.

Aide: Cet écran d'aide.

 

Colonnes

Nom de catégorie:C'est le nom de la catégorie. Cliquez sur le nom pour modifier les paramètres de la catégorie. Une icône représentant un cadenas vous indique qu'elle est verouillée. Pour la déverrouiller, cliquez sur son nom et validez la modification.

Publié: Indique si la catégorie est publiée ou non. Cliquez sur l'icone pour changer son statut(état).

Ordre: Ordonne les catégories dans la liste (cela n'affecte pas l'affichage sur le site). Vous pouvez modifier plusieurs positions conjointement en cliquant sur l'icône "Sauv. Ordre" dans le titre.

Accès: C'est le niveau d'accès des utilisateurs. Une fenêtre popup apparaîtra.

Section: Le nom de la section à laquelle la catégorie appartient. En cliquant sur le nom vous pouvez modifier les paramètres de la section à laquelle la catégorie appartient.

ID de la Catégorie ID: C'est l'identifiant de la catégorie tel qu'il est défini dans la base de données.

# Active: C'est le nombre d'éléments publiés dans cette catégorie.

# Corbeille: Nombre d'éléments appartenant à cette catégorie qui sont actuellement dans la corbeille.

Languages: C'est la(les) langue(s) auquelles les catégories sont ratachées.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.
