Gestionnaire de Sondage: Nouveau/Editer

Cette page vous permet de créer ou d'éditer un sondage pour votre site.

Icônes de la barre d'outils

Aperçu: Un clic sur cette icones ouvrira un pop-up avec une prévisualisation du sondage tel qu'il apparaîtra sur votre site.

Sauver: Sauvegarder le nouveau sondage (ou le sondage édité) et retourne à l'écran précédent.

Fermer: Annule les changements et retourne à l'écran précédent.

Aide: Cet écran d'aide.

 

Détails

Titre: C'est le titre de menu du sondage, comme il sera montré sur votre site.

Titre SEO: Saisissez ici le titre de la page pour le moteur de recherche. Utilisez que des caractères latin. Pour les espaces mettez de '-' ou '_' . Ce sera a utilisé seulement si SEO Pro est activé, mais c'est une bonne habitude pour toujours le remplir.

  • Titre SEO Suggéré - Cliquez ici pour demander à Elxis de vous suggére un titre SEO. Cette suggestion de titre SEO sera basé sur le champ 'Nom' .
  • Titre SEO Validé - Après avoir saisi un titre SEO, vous devez cliquer sur ce lien pour vérifier si il est valide. Si ce n'est pas le cas trouvez une solution !

Publié: Sélectionnez "Oui" pour le rendre disponible sur le site. Sélectionnez "Non" pour le cacher.

Verrouillé: Sélectionnez pour fermer ou pas le sondage.

Accès: Sélectionnez le groupe d'utilisateur qui ont accès, pour voir ce sondage.

Attente/Lag: Saisissez le temp d'attente (en secondes) entre chaque votes.

  • Pour 24 heures mettre: 86400.
  • Pour 12 heures mettre: 43200.
  • Pour 6 heures mettre: 21600.
  • Pour 1 heure mettre: 3600.

Langue: Sélectionnez la langue sous lequel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Options: Saisissez le nombre de questions possibles (maximum 12). Les champs vides ne seront pas affichés, Note: Vous pouvez ajouter des options supplémentaires à un sondage déjà publié. Mais les options effaceront les questions publiées et corromperont l'exactitude des résultats.

 

Pages / Eléments

La liste montre tous les menus disponibles ainsi que leurs articles. A partir d'eux, vous pouvez sélectionner quand le module sera visible.

  • Vous pouvez sélectionner plus d'un articles du menu, en tenant la touche CTRL enfoncé.
  • Si vous voulez que le module soit visible dans chaque page, sélectionnez "Tous."
  • Si vous voulez qu'il ne soit pas visible, sélectionnez "Aucun".
