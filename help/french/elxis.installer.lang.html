des Langues et Templates

De cette page vous pouvez installer de nouvel langues et templates pour Elxis.

Avant de commencer, veuillez noter que certains répertoires doivent être modifiable. Vérifiez la liste des répertoires et leurs permissions avant de commencer. Si le répertoire est non-modifiable au lieu d'être à modifiable, l'installation échouera.

Note de Sécurité: Après installation, veuillez remettre les permissions a non-modifiable sur les répertoires.

 

Elxis utilise 2 (deux) types de fichiers langues.

  • Langue du Site: a la responsabilité de montrer les modules et les messages à la partie public. Les Visiteurs sélectionner choisir la lange de leur choix.
  • Langue de l'Administration: a la responsabilité de montrer l'interface d'administration avec la langue sélectionnée. Uniquement disponible pour les administrateurs.

Elxis utilise 3 (ttois) types de fichiers template.

  • Templates du Site: contrôle l'apparence de votre site web (Partie-Public). C'est ce que le visiteur voit.
  • Template d'Administration: contrôle l'apparence de la partie administration (Partie-Admin). C'est ce que les administrateurs pourront voir.
  • Ecran de Connexion: contrôle l'apparence de l'écran de connexion à la partie administration (Partie-Admin).

 

Comment Installer - Transférer un fichier

C'est la plus courante et la plus facile des méthodes.

Les nouvelles langues ou templates devront toujours êtres au format .zip. Vous pouvez utiliser ce fichier pour installer le nouvel add-on.

Cliquez sur 'Parcourir...'. de l'emplacement du pack de fichiers sur votre ordinateur. Sélectionner le fichier.

Cliquez sur ''Transférer et installer'.Elxis transférera le fichier sur votre site et installera le nouveau fichier langue ou template.

 

Comment Installer - Installer à partir d'un répertoire

Cette méthode est à utiliser uniquement dans le cas d'une limite de configuration des fonctionnalités de votre serveur.

Décompressez les fichiers d'installation du fichier .zip file sur votre PC.

Créez ou utiliser un répertoire vide sur votre site web et transférez le (via FTP), extraire les fichierss (inclus les sous- répertoires).

Saisissez dans le champs 'Installation' le chemin et le répertoire de destination des fichiers.

Cliquez sur le bouton 'Installer' .

Pour terminer l'installation, veuillez suivre les instructions fournies avec l'add-on.
