Menu élément: Lien Système

Cet écran, ajoute un nouvel élément au menu de type "Système". Les liens Systèmes sont des liens pointant vers des caractéristiques spécifiques comme l'enregistrement des utilisateurs et le rappel de mot de passe dans le CMS Elxis.

 

Icônes de la Barre d'Outils

Retour: Retour à l'écran précédent et pour sélectionner un nouveau type de menu à créer.

Sauver: Sauvegarde les changements et retourne à l'écran précédent.

Appliquer: Sauvegarde les changements et reste sur cet écran.

Annuler: Annule la création d'une nouvelle option de menu. Les changements sont abandonnés si vous êtes en train d'éditer un élément de menu.

Aide: Cet écran d'aide.

 

Détails

Nom: C'est le nom de l'élément de menu. Le lien Système sont multilingues, ce nom est uniquement à usage interne.

Lien: Sélectionnez le type de lien que vous voulez créer.

Articles Parent: Vous pouvez sélectionner un élément de menu en tant que élément parent de ce nouveau menu. De cette façon, vous créez des structures hiérarchisées

Ordre: Vous sélectionnez l'ordre que vous souhaitez voir aparaître pour cet élément. Disponible uniquement après avoir sauvegardé le nouveau menu. Par défaut, les nouveaux éléments du menu seront placés à la dernière place.

Langue: Sélectionnez la langue que vous voulez avec cet élément. Si vous sélectionnez "Toutes", ce sera disponible et visualisable, sous tous les langages.

Niveau d'Accès: Sélectionnez le groupe d'utilisateurs qui pourra afficher ce menu.

Publié: Sélectionnez "Oui" pour afficher l'élément de menu sur le site. Sélectionnrz "Non" pour le cacher.
