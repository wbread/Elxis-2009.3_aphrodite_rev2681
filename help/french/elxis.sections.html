Gestionnaire de Section

Cette page vous montre une liste des sections définies sur votre site Elxis.

 

Barre d'Outils

Publié et Dépublié: Publie ou dépublie le ou les modules sélectionnés. Seuls les modules publiés sont visibles sur le site web.

Nouveau: Créer une nouvelle Section.

Copier: Copier les Sections sélectionnées (inclus les Categories et tout leur contenu), vers une nouvelle Section.

Editer: Edite la Section sélectionnée.

Supprimer: Supprime la section sélectionnée. Cette action ne peut pas être exécutée si la Section contient des Catégories ou des Articles.

Aide: Cet écran d'aide.

 

Colonnes

Nom de la Section: C'est le nom de la Section. Si vous avez l'icône cadenas à côté de lui cela montre qu'il est à vérifier. Pour ce faire, cliquez sur le nom pour l'éditer. Alors vous devez Enregistrer ou Fermer la Section.

Publié: Affiche si l'article est publié ou pas. Cliquez sur l'icône pour changer l'état.

Réordonner: Vous pouvez changer l'ordre des Sections, en utilisants les flèches. Vous pouvez aussi changer l'ordre en utilisant la boite 'Ordre' , description ci-dessous.

Ordre: Affiche l'ordre en cours des Sections. Vous pouvez le changer, en modifiant le numéro de l'ordre et en cliquant sur l'icône "Sauver l'Ordre" dans la colonne.

Accès: C'est le groupe d'utilisateurs autorisé à voir cette Section. Cliquez sur elle pour changer. un fenêtre popup s'ouvrira.

ID Section: Affiche l'ID de la Section. Non éditable.

# Catégories: Affiche combien de Catégories à cette Section.

# Active: Affiche le nombre d'articles publié dans la Section.

#Corbeille: Affiche le nombre d'articles de la Section et qui sont actuellement à la corbeille.

Langue: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

 

Pied de Page

En bas de page il y a le pied de page.

Navigation - Si le nombre de messages est plus grand que le nombre de messages affichés par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.
