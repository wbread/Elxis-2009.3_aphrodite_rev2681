Réglage de syndication

Cette page consiste à rendre les éléments de Contenus de votre site, assignés au composant FrontPage, accessibles à la syndication à travers un fichier distribuable

Pour que la syndication fonctionne, et que les visiteurs puissent accèder au nouvelles, vous devez publier le module syndication.

 

Paramètres

Cache: Choisissez le bouton radio 'oui' pour créer et afficher un nouveau fil d'actualité dépendant de la valeur configurée dans le champs durée du cache. Choisir le bouton radio 'No' affiche un nouveau fil d'actualité à chaque fois qu'un article est ajouté dans le composant FrontPage et à chaque fois qu'un autre serveur accède à ce fil d'actualité.

Durée du Cache: C'est le nombre de seconde avant que votre base de donnée du site soit vérifiée pour une mise à jour. Par défaut dans Elxis il s'agit de 3600 secondes ou 60 minutes. Une fois ce temps écoulé, le fil d'actualité se rafraîchira tout seul depuis la base de donnée et affichera les nouveaux articles que vous aurez publiés dans votre fil d'actualité au travers de la Page Accueil.

#Articles: Saisissez ici le nombre d'articles que vous souhaitez syndiquer et afficher dans votre fil d'actualité.

Titre de Syndication: Saisissez ici le titre que vous voulez afficher au dessus de vos articles dans votre fil d'actualité syndiqué pendant son affichage sur les sites et pages qui affilient votre fil d'actualité. Ce titre apparaîtra comme un lien sur votre page .

Description: Saisissez un bref résumé pour décrire votre Fil d'actualité. Si rien n'est entré, cela sera par défaut vers "Elxis site syndication".

Image: Sélectionnez une image depuis le menu déroulant pour l'inclure dans votre Fil d'actualité. Si vous n'avez pas d'image vous pouvez en transférer une depuis le Gestionnaire de Média.

Image Alt: Saisissez un texte ALT pour l'image.

Limite de Texte: Choisissez le bouton radio Oui si vous voulez limiter la longueur du champs Texte. Choisissez le bouton radio Non pour obliger la valeur dans le champs longueur du Texte à être ignorée, ceci affichera l'entier contenu pour chaque article.

Longueur du Texte: Saisissez ici le nombre de mots que vous voulez afficher par article Fil d'actualité. Pour pas de texte, entrez 0.

Ordre: Ordre dans lequel les éléments sont lister dans le Gestionnaire de la page accueil pour l'affichage. Options disponibles:

  • 'Défaut': Classe les articles dans l'ordre défini dans le gestionnaire de la page accueil.
  • 'Ordre page Accueil': Identique à ci-dessus.
  • 'Ancien en premier': Classe les articles dans l'ordre chronologique,les plus ancien en premier.
  • 'Récent en premier': Classe les articles dans l'ordre chronologique,les plus récent en premier.
  • 'Titre Alphabétique': Classe les articles par Titre dans l'ordre Alphabétique.
  • 'Titre Alphabétique-inversé': Classe les articles par Titre dans l'ordre Alphabétique inversé.
  • 'Auteur Alphabétique': Classe les articles par Auteur dans l'ordre Alphabétique.
  • 'Auteur Alphabétique-inversé': Classe les articles par Auteur dans l'ordre Alphabétique inversé.
  • 'Plus Populaire': Classe les articles en fonction de leur popularité,les plus populaire en premier.
  • 'Moins Populaire': Classe les articles en fonction de leur popularité,les moins populaire en premier.

Signet/Marque page: Activer la fonctionnalité de Signet/Marque-page de Firefox.

Fichier Marque-page: Nom de fichier spécial . Si rien n'est entré, utilisation du fichier par défaut.
