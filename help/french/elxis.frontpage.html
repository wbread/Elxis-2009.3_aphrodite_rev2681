Gestion de la page d'accueil

Cette page affiche la liste des articles gérés par le composant Page Accueil ou Frontpage. Pour obtenir de l'aide sur ce composant, cliquez sur : Aide: Composant Frontpage / Accueil.

 

Icônes de la Barre d'Outils

Publier et dépublier: Cliquez sur 'Publier' ou 'Dépublier' pour publier ou dépublier un article séléctionné (coché).

Archiver: Cliquez sur l'icône 'Archiver' pour modifier le statut des articles séléctionnés en "archivés". Ces articles restent visibles sur la page d'accueil.

Supprimer: Cliquez sur l'icône 'Supprimer' pour placer un article de la page d'accueil dans la corbeille.

Aide: Cet écran d'aide.

 

Filtre

Vous pouvez affiner la liste en utilisant la boîte Filtre pour limiter la liste des éléments de contenu. Vous pouvez également saisir une phrase ou une partie de mot, puis presser la touche . Le tableau sera mis à jour ne montrant que le contenu des éléments qui ont ces caractères dans leur titre. Si vous voulez afficher de nouveau tous les éléments de contenu, laissez vide et appuyez sur la touche .

 

Vue Simple/Etendue

Voici le filtre de champs, vous trouverez un bouton qui vous permettent de modifier le nombre de colonnes à afficher. 'Simple' ffiche uniquement les colonnes les plus importantes. 'Etendue', ffiche toutes les colonnes. Vous pouvez faire une essai si cela peut vous aider.

 

Colonne Filtres

Vous remarquerez que plusieurs colonnes ont une flèche située à côté de en-têtes. Lorsque vous cliquez sur le filtre une fenêtre pop-up s'ouvre. ous pouvez utiliser ces filtres pour sélectionner les éléments de contenu que vous voulez voir afficher. Cette fonctionnalité est adressée aux sites Web qui ont un grand nombre d'éléments de contenu. Il permet aux administrateurs de se concentrer uniquement aux éléments de contenu qui les intérêssent.

 

Colonnes

Titre: C'est le titre de l'article. Une icône cadenas après le titre indique que cet article est "bloqué". Pour "bloquer" un article, cliquez sur son titre et le modifier. Vous ne pouvez pas modifier un article "bloqué" par un autre utilisateur.

Publié: Affiche si l'article est publié ou non. Survolez cette icône avec le pointeur de la souris pour afficher les informations de publications.

Réorganiser: Permet de modifier le rang des articles avec les flèches.

Ordre: Affiche le rang d'un article. Pour le modifier, introduisez un nombre puis cliquez sur l'icône 'Valider' en haut de cette colonne. Note : Ce nouveau changement a un impact sur l'ordre d'afficahge des articles sur la page d'accueil.

Accès: Niveau d'accessibilité de l'article : Publique, Enregistré ou Spécial.

Section: Indique à quelle section l'article est rattaché. Cliquez sur la section pour la modifier.

Catégorie: Indique à quelle catégorie l'article est rattaché. Cliquez sur la catégorie pour la modifier.

Auteur: Le nom de l'auteur de l'article. Cliquez dessus pour le modifier.

Langues: C'est le ou les langues à laquelle l'Article appartient. Il ne sera affiché que si les utilisateurs/visiteurs du site Web est dans l'une des langues sélectionnées. Si il est à 'Tous', il sera visible pour toutes les langues.
