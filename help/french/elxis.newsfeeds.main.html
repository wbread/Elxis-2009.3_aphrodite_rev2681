Gestionnaire de Flux de Nouvelles

En utilisant le Gestionnaire de Flux vous pouvez définir plusieurs origines de flux RSS. Les utilisateurs de la Partie-public peuvent vérifier les flux et vous informer au sujet des dernières nouvelles diffusées sur ces sites. Ils peuvent cliquer alors sur un flux et ce transférer sur le site pour lire l'article en entier.
Les Flux de Nouvelles sont organisés en catégories.

 

Icône de la Barre d'Outils

Publié et Dépublié: Publie ou dépublie le ou les flux sélectionnés. Seuls les flux publiés sont visibles sur le site web.

Nouveau: Créer un nouveau flux.

Editer: Editer un flux sélectionné.

Supprimer: Supprimer le ou les flux sélectionné(s).

Aide: Cet écran d'aide.

 

Colonnes

Flux de Nouvelles: Le nom du flux tel qu'il apparait sur votre site. Si vous avez l'icône cadenas à côté de lui cela montre qu'il est à vérifier. Pour ce faire, cliquez sur le nom pour l'éditer. Alors vous devez Enregistrer ou Fermer le flux.

Publié: Affiche si l'article est publié ou pas. Cliquez sur l'icône pour changer l'état.

Réordonner: Vous pouvez changer l'ordre des flux, en utilisant les flèches.

Catégorie: C'est la Catégorie auquel le flux appartient. Cliquer sur le nom pour éditer les paramètres du flux.

#Articles: Ceci affiche le nombre d'Articles sur votre site est défini les paramètres du flux. Vous pouvez le changer en éditant les articles du flux.

Temps du Cache (sec): C'est le temps entre chaque renouvellement tel que défini dans les paramèters concernant le Flux de Nouvelles.

 

Pied de Page

En bas de page il y a le pied de page.

Navigation - Si le nombre de messages est plus grand que le nombre de messages affichés par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.
