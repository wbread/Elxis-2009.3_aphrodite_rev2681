SoftDisk

SoftDisk est une zone de la mémoire virtuelle pour variables et paramètres en tout genre. Vous pouvez en ajouter de nouvelles ou modifier/supprimer des entrées SoftDisk existantes sauf celles étiquetés comme intégrées au système. Aussi les entrées marquées comme "Ne pas supprimer" peuvent seulement être éditées. Pour nommer les sections de SoftDisk et les valeurs vous êtes seulement autorisé à utiliser des lettres latin, des chiffres et caractère souligné (_).

 

Barre d'Outils

Nouveau: Créer une nouvelle variable SoftDisk.

Editer: Editer la variable SoftDisk sélectionnée.

Supprimer: Supprimer la variable SoftDisk sélectionnée. Vous ne pouvez pas supprimer les variables marquées à 'Ne pas supprimer'.

Aide: Cet écran d'aide.

Mise à Jour du système: Mise à jour des valeurs spécifiques du système stockées dans SoftDisk.

 

Section Filtrer

Vous pouvez utiliser les filtres de Section pour afficher uniquement les variables SoftDisk d'une section sélectionnée.

 

Colonnes

Colonne cases à cocher: Une icône cadenas, vous informe que cette variable ne peut pas être supprimée.

Type de colonne: Affiche le type de variable SoftDisk. Types disponibles:

  • Décimale (exp. - 10.45)
  • Email (exp. - user@domain.com)
  • Image (exp. - http://www.domain.com/image01.png)
  • Nombre entier (exp. - 10)
  • Chaîne (exp. - Hello World)
  • Texte (exp. - Full Text)
  • Temps (exp. 1190459123) Le temps est entré dans format estampillé Unix (entier long qui contient le nombre de secondes entre Unix (janvier 1 1970 00:00:00 GMT) et le temps spécifiés.)
  • URL (exp. - http://www.domain.com)
  • OUINON

Section: Section d'appartenance d'une variable SoftDisk. Les Sections de SoftDisk n'ont rien à voir avec les Sections de Contenu.

Nom: Nom de la variable Softisk. Cliquez pour éditer. Note: Un survol de la souris sur l'icône d'aide et vous obtenez des infos au sujet de l'utilisation des variables SoftDisk.

Valeur: Valeur de la Variable.

Système: Informe si la variable est d'un type de système ou pas.

Ne pas Supprimer: Vous ne pouvez pas supprimer une variable marquée à Ne pas supprimer.

Dernière modification: Date et heure, la valeur a été modifiée pour la dernière fois.

Section: (A affiché seulement quand vous voyez 'Tous les élements de Contenu'). C'est la Section à laquelle l'article appartient. Le nom de section agit comme un lien pour les paramètres de section.

Catégorie: C'est la Catégorie à laquelle l'article appartient. Le nom de la catégorie est lié aux paramètres de la catégorie.

Auteur: Affiche le nom de l'auteur des éléments de Contenu. Le nom de l'auteur est lié aux détails de l'auteur.

Langues: Affiche pour quels langages du site les éléments de Contenu sont disponibles. Il sera montré seulement si les utilisateurs affichent ce site web sur une des langages disponibles. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

 

En bas de page il y a le pied de page.

At the bottom of the page there is the footer.

Navigation - Si le nombre de variable SoftDisk variables est plus grand que le nombre de variables affichées par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.
