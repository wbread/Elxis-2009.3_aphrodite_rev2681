Gestionnaire des bannières Clients

Chaque bannière est associée à un client. Utiliser ce formulaire pour ajouter ou éditer vos clients.

Icônes de la Barre d'Outils

Nouveau: Cliquez sur cet icône pour ajouter un nouveau client.

Editer: Cliquez sur cet icône pour modifier les détails d'un client.

Supprimer: Cliquez sur cet icône pour supprimer un client. Une confirmation vous sera demandée pour compléter l'opération. Sélectionnez "Oui" pour supprimer le client ou "Non" pour annuler.

Aide: Cet écran d'aide.

 

Colonnes

Nom du Client: Le nom du client. Cliquez pour éditer le client.

ID de la bannière client: L'ID du Client. Il peut être utiliser pour paramètrer le module bannière.

Contact: Personne à contacter pour votre client.

Nb. de Bannières actives: Affiche le nombre de bannières publiées pour ce client.
