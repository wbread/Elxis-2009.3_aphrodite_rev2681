Gestionnaire de Corbeille

Cet écran, vous affiche une liste des éléments envoyés dans le "Gestionnaire de Corbeille". La corbeille est l'équivalent de la corbeille Windows. Vous pouvez restaurer des éléments ou les supprimer définitivement.

Le gestionnaire de Corbeille a deux sortes de tri.
Eléments de Contenu, affiche tous les éléments, articles etc...
Eléments de Menu, affiche tous les menu, sous-menus supprimés.

 

Barre d'Outils

Restaurer: Pour restaurer un ou plusieurs éléments, cochez les cases en face du titre des éléments et cliquez sur l'icône 'Restaurer'

Supprimer: Pour supprimer DEFINITIVEMENT un élément de la base de donnée, cochez la case en face du titre et cliquez sur l'icone 'Supprimer'.

Aide: Cet écran d'aide.

 

Tables

Elements de Contenu

Titre: C'est le titre de l'élément supprimé.

Section: C'est la section dont fait partie l'élément supprimé.

Catégorie: C'est la catégorie dont fait partie l'élément supprimé.

ID: C'est l'identifiant de l'élément supprimé.

Tab Menu Items

Titre: C'est le nom du menu.

Menu: C'est le menu supprimé.

Type: C'est le type de menu supprimé.

ID: C'est l'identifiant de l'élément supprimé.
