Menus: Edit Menu Item

En cliquant sur un Elément de Menu, vous avez accès au menu Édition écran où vous pouvez modifier les détails et les paramètres. Tout élément de Menu ont des paramètres différents. Certains d'entre eux sont commun à tous.

 

Barre d'icônes

Sauver: Cliquez sur l'icône sauver enregistre les paramètres choisis et retourne sur le Gestionnaire de Menu.

Appliquer: Cliquez sur l'icône pour enregistrer les paramètres choisi et rester sur la même fenêtre pour édition.

Fermer: Cliquez sur l'icône annule toute les action sur cet écran. Vous retournerez sur la page précédente.

La fenêtre est divisée en 2 parties principales: "Détails" et "Paramètres". Ceci varie en fonction du type d'élément de menu choisi. Certains "Details" peuvent aussi être éditer par la liste dans le Gestionnaire de Menu. Certains "paramètres" peuvent êtres vu seulement après avoir cliqué sur l'icône "Appliquer" ou "Sauver" .
