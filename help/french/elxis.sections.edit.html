Gestionnaire de Sections: Nouveau/Editer

Cette page vous laisse créer ou modifier une Section.

 

Icônes de la Barre d'Outils

Transfert: Ouvre une nouvelle fenêtre qui vous laisse transférer un fichier ou une image de votre PC au dossier /media.

Sauver: Sauvegarde la Section et retourne au Gestionnaire de Sections.

Appliquer: Sauvegarde les changements et reste sur cet écran.

Annuler: Annule les changements et retourne à l'écran précédent.

Aide: Cet écran d'aide

 

Section Details

Cadre: Montre le cadre de cette Section. Valeur par défaut 'Contenu'. Non éditable.

Titre: Saisissez ici le titre de la Section que vous voulez afficher sur le site.

Nom de la Section: Saisissez ici le nom de la Section que vous voulez afficher.

Titre SEO: Saisissez ici le titre de la page pour le moteur de recherche. Utilisez que des caractères latin. Pour les espaces mettez de '-' ou '_' . Ce sera a utilisé seulement si SEO Pro est activé, mais c'est une bonne habitude pour toujours le remplir.

  • Titre SEO Suggéré - Cliquez ici pour demander à Elxis de vous suggére un titre SEO. Cette suggestion de titre SEO sera basé sur le champ 'Nom' .
  • Titre SEO Validé - Après avoir saisi un titre SEO, vous devez cliquer sur ce lien pour vérifier si il est valide. Si ce n'est pas le cas trouvez une solution !

Image: Si vous voulez, vous pouvez sélectionner une image qui sera utilisée à côté du texte de description de la section. A choisir par la liste déroulante dans le dossier images/stories.

Position de l'Image: Sélectionnez la position de cette image. Les options disponibles sont droite ou gauche.

Ordre: Sélectionnez ici l'ordre d'affichage de cette section. Cela affecte aussi les sections qui sont affichées sur la Partie-Public. Les nouvelles sections sont placées par défaut en dernière position.

Niveau d'Accès: Sélectionnez le groupe utilisateur autorisé à voir cette Section et son contentu.

Publié: Sélectionnez "Oui" pour mettre à disponible sur le site. Sélectionnez "Non" pour le cacher. Il masque aussi toutes les catégories et articles qui appartiennent à cette Section.

Langue: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Description: Vous pouvez écrire un texte pour donner une description à cette Section.

 

En-dessous de Description, il y a plusieurs boutons spéciaux. Par défaut il y a 'Image', 'Page Break' et 'Flash Video'. Ce sont des racourcis pour insérer des codes. Par exemple, cliquer sur le bouton 'Page Break' , ajoutera '{mospagebreak}'. Alternativement, vous pouvez insérer de codes manuellement. L'effet sera le même.
Vous pouvez installer et utiliser d'autres Bots.

 

Lier au Menu

Vous donne une façon alternative de créer un nouvel article de menu d'un des types suivants: "Section List", "Section Blog", "Section Archive Blog" qui seront liés aux sections.

Sélection du Menu: Sélectionnez le menu que vous voulez créé pour l'article de menu.

Type de Menu: Sélectionnez le type d'article du menu.

Nom de l'élément du Menu: Saisissez un nom d'élément.

Lier au Menu: Cliquez sur ce bouton pour créer un nouvel élément de menu.

Liens existants: Affiche une liste de tous les éléments de menu en réference avec cette Section.
