Gestionnaire de Menu: Nouveau/Editer

ous pouvez ajouter de nouveaux Menus à Elxis, en utilisant cet écran.

 

Icône de la barre Outils

Sauver: Le Nom de Menu sera mis a jour ou créé dans le gestionnaire de menu.

Annuler: Annule les changements et ne créé pas de Menu.

Aide: Cet écran d'aide.

 

Champs

Nom du Menu: C'est le nom utilisé intérieurement par Elxis pour identifier le Menu. C'est le nom affiché sur la liste du Gestionnaire du Menu.

Titre du Module: Comme chaque Menu est associé à un Module, c'est le nom du Module qui est associé au Menu. Vous pouvez changer plus tard ce nom, via le Gestionnaire du Module.
