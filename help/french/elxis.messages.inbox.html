Messages: Boite de réception

Vous pouvez utiliser cette page pour afficher vos messages privés. Les messages Privés sont envoyez par Elxis ou d'autres administrateurs. Par exemple, quand un utilisateur vous soumet un nouvel article, Elxis vous envoie un message.

 

Icônes de la Barre d'Outils

Nouveau: Créer un nouveau message.

Supprimer: Supprimer les messages sélectionnés.

Aide: Cet écran d'aide.

 

Colonnes

Sujet: Sujet du Message. Cliquez pour le lire.

De: Qui vous envoi ce message.

Date: Quand a t-il été envoyé.

Lu: 'Lu' si vous avez lu le message. 'Non-Lu' si vous ne l'avez pas encore lu.

 

Pied de Page

En bas de page il y a le pied de page.

Navigation - Si le nombre de messages est plus grand que le nombre de messages affichés par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.
