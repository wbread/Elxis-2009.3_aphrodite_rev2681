SoftDisk: Editeur

Utiliser ce formulaire vous pouvoir ajouter ou modifier des variables de softdisk.

Pour nommer les sections de SoftDisk et les valeurs vous êtes seulement autorisé à utiliser des lettres latin, des chiffres et caractère souligné (_).

 

Barre d'Outils

Sauver: Sauvegarde les changements et retourne à l'écran précédent.

Annuler: Annule les changements et retourne à l'écran précédent.

Aide: Cet écran d'aide.

 

Détails

C'est la zone où vous pouvez paramètrer les nouvelles variables de SoftDisk ou modifier une variable SoftDisk existante.

Quand vous êtes en mode édition, la Section, le Nom et et le Type de paramètres ne peuvent être changés. Vous pouvez modifier seulement la valeur de la variable.

Aussi, quand vous êtes dans un mode édition, vous voyez une description courte concernant l'usage de cette variable.

Section: Sélectionnez la Section dans laquelle la nouvelle variable SoftDisk appartiendra. Par défaut, il existe les Sections: Core, Profil Utilisateur et Utilisateurs. En cliquant sur le lien 'Nouveau' , vous pouvez ajouter une nouvelle Section. Veuillez noter que les commentaires des Sections se trouvent en haut de cette page d'aide.

Nom: Saisissez le nom pour la nouvelle variable SoftDisk. Veuillez noter que les noms des Sections se trouvent en haut de cette page d'aide.

Type: Sélectionnez le type de variable de SoftDisk. La valeur du champ est adapté au type sélectionné. Les types disponibles sont:

  • Décimale (exp. - 10.45)
  • Email (exp. - user@domain.com)
  • Image (exp. - http://www.domain.com/image01.png)
  • Nombre entier (exp. - 10)
  • Chaîne (exp. - Hello World)
  • Texte (exp. - Full Text)
  • Temps (exp. 1190459123) Le temps est entré dans format estampillé Unix (entier long qui contient le nombre de secondes entre Unix (janvier 1 1970 00:00:00 GMT) et le temps spécifiés.)
  • URL (exp. - http://www.domain.com)
  • OUINON

Valeur: Saisissez la valeur de la variable SoftDisk.

Dernière modification: Affiche la date et l'heure, cette variable a été modifiée pour la dernière fois.
