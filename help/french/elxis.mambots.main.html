Bots du Site: Gestion des Bots

Les Bots sont essentiellement de petits morceaux du code qui quand vous les appelez via plusieurs paramètres, activent un programme, écrivent ou exécutent une fonction spécifique. Ils peuvent être aussi qu'une fonction qui remplace du texte saisi en code (tel que pour afficher une image dans un article) ou (tel que les éditeurs WYSIWYG ou autres programmes).

{mosimage}, {mospagebreak}, {mosflv}, sont des exemples classiques de bots.

Chaque Bot a son propre ensemble de paramètres de configuration.

Les Bots sont divisés en 4 types/catégories et sont affichés, filtrés par une liste de menu déroulant:

  • content - Insére dans le contenu et le remplace pour être vu par la partie-public.
  • editors - Editeurs accèssible pour la partie-public et partie-admin.
  • editors-xtd - Extentions de l'Editeurs.
  • search - Activé quand les utilisateurs font des recherches sur le site Web. Retour de Résultats.

 

Filtre

Vous pouvez affiner la liste en utilisant la boîte Filtre pour limiter la liste des bots. Vous pouvez également saisir une phrase ou une partie de mot, puis presser la touche . Le tableau sera mis à jour ne montrant que les bots qui ont ces caractères dans leur titre. Si vous voulez afficher de nouveau tous les bots, laissez vide et appuyez sur la touche .

 

Colonnes Filtres

Vous remarquerez que plusieurs colonnes ont une flèche située à côté de en-têtes. Lorsque vous cliquez sur le filtre une fenêtre pop-up s'ouvre. Vous pouvez utiliser ces filtres pour sélectionner les bots que vous voulez voir afficher. Cette fonctionnalité est adressée aux sites Web qui ont un grand nombre de bots. Il permet aux administrateurs de se concentrer uniquement aux bots qui les intérêssent.

 

Icone de la Barre d'Outils

Publié/Dépublié: Cliquez sur l'icone pour le Publier ou le dépublier.

Nouveau: Cliquez sur l'icone "Nouveau" pour créer ou installer un nouveau Bot.

Editer: Cliquez sur l'icone "Editer" pour modifier les réglages souhaités.

Supprimer: Supprimer/Désinstaller le(s) bot(s) sélectionné(s).

Aide: Cet écran d'aide.

 

Colonnes

Nom du Bot: Cela liste les noms des Bots installés.

Publié: Cela montre si le Bot est publié ou non. Cliquez sur l'icône si vous souhaitez changer son état.

Réordonné: Cliquez sur les icônes flèche pour monter l'article ou le descendre dans la liste. Cela n'a aucun effet sur la partie public de votre site.

Ordre: Cela affiche la commande des Bots par rapport à d'autre Bots de même type. Tapez le nouveau chiffre d'ordre dans votre bot pour le voir apparaitre dans l'administration.

Accès: Cela affiche qui peut voir ou utiliser le Bots sur la partie public. Choisissez le niveau d'accès en cliquant sur le bouton activer le niveau d'accès. Une fenêtre popup apparaîtra.

Type: Cela affiche le type de Bot.

Fichier: Cela affiche le nom de fichier du Bot.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.
