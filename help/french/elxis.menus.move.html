Menus: Déplacer

Cette page permet de de déplacer un ou plusieurs Éléments dans un autre menu existant. Si vous choisissez de déplacer un élément Parent de menu et ses sous-menus ils seront tous déplacé gardant la relation Parent-Enfant.

 

Barre d'Outils

Déplacer: Cliquer sur l'icône pour déplacer les éléments de menu.

Annuler: Cliquer sur l'icone pour annuler le processus.

Aide: Cet écran d'aide.

 

Contrôles

Déplacer au Menu: Listes des menus disponibles. Sélectionnez le menu que vous souhaitez déplacer vers l'élément du menu.

Eléments de Menu déplacés: Dresse la liste des éléments du menu qui sont déplacés.
