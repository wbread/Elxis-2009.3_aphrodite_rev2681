Le Gestionnaire d'articles

Le gestionnaire d'articles affiche la liste des articles d'une ou de toutes les sections dans le CMS Elxis. Il vous permet d'ajouter, éditer, supprimer des éléments de contenus (articles) pour votre site.

Elle peut être consultée à partir des deux menus d'administration. Basée sur la manière dont il est appelé, indique une autre série d'éléments du contenu. La liste suivante explique que:

  • Appelé à partir de Contenu -> Contenu par Section -> [Nom de Section] -> [Nom de Section].
    Affiche les éléments de contenu qui appartiennent à cette section.
  • Appelé à partir de Contenu -> Tous les Contenus
    Affiche tous les éléments de contenus.

 

Barre d'outils

Nouveau: Pour créer un nouvel article.

Editer: Pour modifier un article.

Publié et dépublié: Pour publier ou dépublier les articles séléctionnés. Seuls les éléments publiés sont visibles sur le site.

Déplacer: Pour déplacer un article vers une autre Section ou une autre catégorie.

Copier: Pour copier un article dans une section ou une catégorie.

Traduire: Cliquer sur l'icone '"Tranduire" pour traduire l'article de contenu sélectionné et créé un nouvel article. A Appliquer à un article à la fois seulement.

Archive: Pour archiver des articles. Ils restent pourtant visibles depuis la page d'accueil s'ils sont publiés.

Corbeille: Pour mettre des articles à la poubelle. Même s'ils ne sont plus visibles, ils ne sont pas définitivement supprimés. Vous pouvez restaurer les articles mis à la poubelle dans le Gestionnaire de corbeille.

Aide: Cet écran d'aide.

 

Filtre

Vous pouvez affiner la liste en utilisant la boîte Filtre pour limiter la liste des éléments de contenu. Vous pouvez également saisir une phrase ou une partie de mot, puis presser la touche <Entrée>. Le tableau sera mis à jour ne montrant que le contenu des éléments qui ont ces caractères dans leur titre. Si vous voulez afficher de nouveau tous les éléments de contenu, laissez vide et appuyez sur la touche <Entrée>.

 

Affichage Simple/Etendu

Voici le filtre de champs, vous trouverez un bouton qui vous permettent de modifier le nombre de colonnes à afficher. 'Simple' affiche uniquement les colonnes les plus importantes. 'Etendue', affiche toutes les colonnes. Vous pouvez faire une essai si cela peut vous aider.

 

Colonnes Filtres

Vous remarquerez que plusieurs colonnes ont une flèche située à côté de en-têtes . Lorsque vous cliquez sur le filtre une fenêtre pop-up s'ouvre. Vous pouvez utiliser ces filtres pour sélectionner les éléments de contenu que vous voulez voir afficher. Cette fonctionnalité est adressée aux sites Web qui ont un grand nombre d'éléments de contenu. Il permet aux administrateurs de se concentrer uniquement aux éléments de contenu qui les intérêssent.

 

Colonnes

Titre: Titre de l'article. Cliquez sur ce titre pour le modifier. Une icône cadenas après l'article signifie qu'il est "Bloqué". Pour le débloquer, cliquez sur l'article pour le modifier puis cliquez sur 'Enregistrer' ou 'Annuler'.

Publié: Indique si l'article est publié ou non. Survolez avec le pointeur de la souris cette icône pour afficher les renseignements sur sa publication. Cliquez dessus pour modifier son état.

Page Accueil: Permet de définir si l'article sera visible ou non sur la page d'accueil. S'il est publié avec le composant Frontpage, cette colonne est cochée par défaut.

Réorganiser: Permet de faire monter ou descendre un article dans la liste.

Ordre: Affiche le rang de l'artcle dans la liste. Vous pouvez le modifier en tapant un nombre puis en cliquant sur l'icône 'Valider' de la barre des titres.

Accès: Définit le niveau d'accès à l'article. Cliquez sur un niveau d'accès pour modifier sa valeur.

ID: Affiche l'ID de l'élement de contenu. Non modifiable.

Section: Affiche la section à laquelle l'article est rattaché. Cette colonne est disponible sur la page 'Tous les articles'. Cliquez sur la section pour ouvrir sa page de modification.

Catégorir: Affiche la catégorie à laquelle l'article est rattaché. Cliquez sur la catégorie pour ouvrir sa page de modification.

Auteur: Affiche le nom du rédacteur. Cliquez sur l'auteur pour ouvrir la page de modification de ses informations.

Langue: C'est la langue de l'article dans laquelle elle est affichée. Vous pouvez utiliser le Filtre, pour seulement sélectionner les Articles avec la langage qui vous intéresse. Si vous sélectionnez "Tous", alors tous les Articles seront montrés, sans souci des langages supportés. "Tout" la signification de cet Article de Contenu sera affichée dans toutes les langues. Sans tenir compte de la langue sélectionné.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.

Légende - Affiche et explique les différents états de publication d'un élément de contenu. Ces icônes sont affichées dans la colonne 'publié'.
