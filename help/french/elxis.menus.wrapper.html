Menu élément: IFrames/Wrapper

Cet écran, vous permet d'ajouter un élément "IFrames/Wrapper". Utilise un IFrame pour montrer un autre site web, une application externe, ou d'une page web externe. Pour les flux RSS, c'est une méthode très pratique si vous voulez les inclure dans votre site, cela concerne aussi les informations d'autres origines.

 

Icones de la Barre d'Outils

Retour: Retournez à l'écran précédent et sélectionnez un nouveau type de menu article à créer.

Sauver: Sauvegardez les changements et retournez à l'écran précédent.

Appliquer: Sauvegardez les changements et restez sur cet écran.

Annuler: Annule la création de menu. Aucun changements ne sera fait. Les changements ne sont pas pris en compte si vous éditez un menu.

Aide: Cet écran d'aide.

 

Détails

Nom: C'est le nom de menu de l'article, comme il sera montré sur votre site.

Titre SEO: Saisissez ici le titre de la page pour le moteur de recherche. Utilisez que des caractères latin. Pour les espaces mettez de '-' ou '_' . Ce sera a utilisé seulement si SEO Pro est activé, mais c'est une bonne habitude pour toujours le remplir.

  • Titre SEO Suggéré - Cliquez ici pour demander à Elxis de vous suggére un titre SEO. Cette suggestion de titre SEO sera basé sur le champ 'Nom' .
  • Titre SEO Validé - Après avoir saisi un titre SEO, vous devez cliquer sur ce lien pour vérifier si il est valide. Si ce n'est pas le cas trouvez une solution !

Lien du Wrapper: Saisissez ici l'adresse URL du lien vers la page afficher dans l'IFrame/wrapper.

URL: L'URL pour cet élément sera visible après avoir Appliquer/Sauver.

Elément Parent: Vous pouvez sélectionner un article du menu existant comme parent de ce nouveau menu article. Cette direction vous créez des structures hiérarchisées.

Ordre: Vous sélectionnez la commande que vous voulez pour que cet menu article paraisse. Disponible seulement après avoir enregistré le nouveau menu article. Les nouveaux menus articles sont par défaut, à la dernière place.

Langue: Sélectionnez la langue sous laquel ce menu article sera disponible. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Niveau d'Accès: Sélectionnez le groupe d'utilisateurs auquel l'utilisateur doit appartenir pour afficher cet article du menu.

Publié: Sélectionnez "Oui" pour rendre disponible le menu sur votre site. Sélectionnez "Non" pour le cacher.

 

Paramètres

Image de Menu: Si vous le souhaitez, vous pouvez afficher une petite image à côté du menu article. Sélectionnez cette image dans la liste déroulante.

Utiliser uniquement l'image de Menu: Si vous avez réglé les paramètres à 'Oui', alors au lieu du nom du menu, sera afficher uniquement l'image de menu.

Classe Suffixe de la Page: vous permet de personnaliser le style de cette page. Le suffixe que vous saisissez ici est automatiquement ajouté à la classe CSS pour cette page.

Bouton Retour: Contrôle pour activer oui ou non le bouton retour à la page précédente.

  • "Utilisation Globale" - Affiche ou cache le lien Retour basé sur la Configuration Globale.
  • "Cacher" - Cache le lien Retour, indépendement du régalge de la Configuration Globale.
  • "Afficher" - Affiche le lien Retour, indépendement du régalge de la Configuration Globale.

Afficher le titre de la Page: Choisissez si le titre de la page sera affiché ou pas.

Titre de la Page: Si vous souhaitez afficher un texte spécial comme le titre de cette page, saisissez-le ici.

Barres de défilement: Décidez comment vous voulez que les barres de défilement se comportent dans l'iFrame. Les options disponibles sont:

  • "Non" - N'utilise pas de barres de défilement. Si la largeur ou la hauteur du contenu du wrappeur est plus grande que ses dimensions, il faudra les redimenssionner.
  • "Oui" - Affichera toujours les barres de défilement, même si elles ne sont pas indispensables.
  • "Auto" - N'affichera les barres de défilement que si elles sont indispensables.

Largeur: Spécifiez la largeur de la fenêtre de l'IFrame/Wrapper. Vous pouvez utiliser les valeurs en pixels (expl. 400), ou utiliser les valeurs relatives (exp. 80%).

Hauteur: Spécifiez la hauteur de la fenêtre de l'IFrame/Wrapper en pixels.

Hauteur Auto: Mettez à "Oui", si vous voulez que la hauteur de l'IFrame/Wrapper soit calculée automatiquement par rapport à la page externe. Ne fonctionne que avec certaines pages.

Auto: Par défaut, si "http://", ou "https://" est trouvé, les "http://" seront insérés automatiquement dans l'adresse URL entrée dans le lien de"IFrame/Wrapper". Vous pouvez désactiver cette fonctionnalité pour avoir le contrôle total de l'adresse URL.
