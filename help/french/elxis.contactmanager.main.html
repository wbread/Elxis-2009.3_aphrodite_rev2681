Gestionnaire de contacts

Vous "sélectionnez" les contacts en cochant la case qui se trouve au début de chaque ligne.

 

Icones de la barre d'outils

Publié et Dépublié: Cliquez sur l'icône 'Publié' ou 'Dépublié' pour publier ou dépublier n'importe quels contacts sélectionnés.

Nouveau: Cliquez sur l'icône 'Nouveau' pour créer un nouveau contact.

Editer: Cliquez sur l'icône 'Editer' pour éditer le contact.

Supprimer: Cliquez sur l'icône 'Supprimer' pour supprimer le contact de la base de données.

Aide: Cet écran d'aide.

 

Filtre

Vous pouvez utiliser le Champ Filtre pour limiter la liste des contacts ou la recherche spécifique des contacts. Vous pouvez taper une phrase ou une partie de mot et appuyez sur <Entrée>. Le tableau sera mis à jour montrant que les contacts qui ont ces caractères. Si vous voulez afficher de nouveau tous les contacts, laisser vide et appuyez sur <Entrée>.

 

Colonne Filtre

Vous remarquerez que la colonne Catégorie a une flèche située à côté de l'en-tête. Lorsque vous cliquez sur filtre une fenêtre popup s'ouvre. Vous pouvez utiliser ce filtre pour n'afficher que les contacts qui appartiennent à la catégorie sélectionnée.

 

Colonnes

Nom: Nom du ou des Contacts.

Publié: Cliquez sur l'icône pour basculer entre Publié/Dépublié et décider si le contact sera disponible ou non sur votre site.

Réordonner: Cliquer sur les icônes pour déplacer l'enregistrement vers le haut ou le bas de la liste. L'ordre est utilisé pour l'affichage dans le gestionnaire de contacts mais également pour la page visible par les visiteurs.

Catégorie: Vous pouvez attribuer aux contacts différentes catégories. Cette colonne affiche la catégorie à laquelle le contact appartient. Cliquez pour éditer la catégorie.

Lié aux utilisateurs: Si le contact est lié à un utilisateur enregistré sur le site, le nom de l'utilisateur apparaitra ici. Cliquez pour éditer l'utilisateur.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.
