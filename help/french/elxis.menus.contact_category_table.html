Menu Contact:
Table - Catégorie de Contact

Cet écran, vous permet d'ajouter un nouvel élément au menu "Table - Catégorie de Contact".

Vous pouvez définir plusieurs paramètres pour ce type d'élément de menu, cela contrôle le nombre, la direction de la table et les lignes présentées.

 

Icône de la barre Outils

Retour:Retour à l'écran précedent et sélectionnez un nouveau type d'élément de menu à créer

Sauver: Sauvegarde les changements et retour à l'écran principal.

Appliquer: Sauvegarde les changements et reviens à l'écran en cours.

Annuler: Annule toutes modification, création sur le nouveau menu. Les changements ne sont donc pas enregistrés.

Aide: Cet écran d'aide.

 

Détails

Nom: Inscrivez le nom de l'élément de menu tel que vous voulez qu'il apparaisse sur votre site.

Catégorie: Sélectionnez la catégorie de contact que vous voulez montrer.

URL: Le lien pour cet élément apparaîtra après avoir cliqué "sauver/appliquer".

Elément Parent: Sélectionnez "top" si vous vous voulez que votre nouvel élément de menu apparaisse dans le menu de navigation principal (main menu). Vous pouvez choisir de mettre cet élément en sous menu d'un élément du menu principal en choisissant ce dernier comme élément parent.

Ordre: Ajoute les nouveaux liens à la suite des éléments de menu déjà existants. Après avoir sauvegardé l'élément, vous pouvez lui choisir une nouvelle place dans la liste en la sélectionnant dans le menu déroulant. Sa nouvelle place sera APRES le lien sélectionné.

Langue: Sélectionnez la langue que vous voulez avec cet élément. Si vous sélectionnez "Toutes", ce sera disponible et visualisable, sous tous les langages.

Nivau d'Accès: Choisir qui peut avoir accès à cet élément.

Publié: Choisissez si votre nouvel élément sera défini comme publié oui ou non. Par défaut, il le sera.

 

Paramètres

Image de Menu: vous permet de placer une image qui apparaîtra à la gauche ou à la droite de l'élément de menu. Sélectionnez l'image dans la liste par le menu déroulant. Note: En outre, pour que l'image apparaisse dans Modules -> Site Modules -> [nom-de-menu], l'option " afficher les éléments de menu" doit être paramétrée à "OUI" dans les paramètres.

Utiliser seulement l'image de menu: Si vous avez mis les paramètres à 'Oui', alors au lieu du nom de menu il sera affiché en utilisant l'image de menu.

PClasse Suffix de la Page: Vous permet de mettre des styles personnalisés pour cette page. Le suffixe que vous tapez ici est ajouté automatiquement aux classes CSS pour cette page et devra être défini dans la feuille de style CSS de votre Template.

Bouton Retour: Contrôle pour activer oui ou non le bouton retour à la page précédente.

Afficher le Titre de la Page: Sélectionnez si vous voulez afficher ou pas le titre de la page.

Titre de la Page: Si vous souhaitez montrer un texte spécial comme le titre de cette page, tapez-le à la prochaine zone de texte et choisissez "Afficher".

Autres Catégories: Montre ou cache une liste de Catégories de Contact disponibles en bas de page. Les deux paramètres suivants contrôlent son comportement.

Description de la Catégorie: Choisissez si vous voulez afficher aussi le texte de la description pour chaque lien de la catégorie.

# Catégorie: Choisissez si vous voulez montrer le nombre d'articles de contact à chaque catégorie.

Titres des Tableaux: Choisissez si vous voulez que les titres de la table soient visible. Le "Nom", "Position", "Email", "Téléphone", "Fax", les en-tête seront visibles ou pas. Les prochains quatre paramètres vous permettent de sélectionner si vous voulez le voir sur plusieurs colonnes.

Colonne Position: Sélectionnez montrer ou masquer la colonne postion.

Colonne Email: Sélectionnez montrer ou masquer la colonne email.

Colonne Téléphone: Sélectionnez montrer ou masquer la colonne téléphone.

Colonne Fax: Sélectionnez montrer ou masquer la colonne fax.
