Menu : Nouvel Élément de Menu

Il existe plusieurs types d'éléments de Menu. Chaque type a un rôle spécial. Lorsque nous créons un Nouvel élément de Menu nous devons choisir son type.

Pour créer un nouvel élément de Menu, vous pouvez:

  • Cliquer sur son nom. Un nouvel écran apparaît où vous pourrez modifier les paramètres.
  • Sélectionner un type et cliquer sur l'icône 'Nouveau' . Un nouvel écran apparaît où vous pourrez modifier les paramètres.

 

Icônes de la Barre d'Outils

Suivant: après avoir sélectionné le bouton radio pour le type d'Éléments de menu que vous souhaitez créer cliquez sur l'icône uivant dans la barre des tâches. Une nouvelle fenêtre s'ouvrira avec le nom Ajouter un Élément de Menu :Nouveau Lien.

Annuler: Cliquez sur l'icone "Annuler" si vous voulez arrêter la création de l'élément de Menu.

Aide: Cet écran d'aide.

 

Note:Vous trouverez des informations détaillées sur chaque type, en cliquant sur l'icône 'Aide' .
