Menu élément:
Table - Contenu de Section

Cet écran, ajoute un nouvel élément de menu dans "Table - Contenu de Section". Un menu de ce type, présente des catégories qui appartiennent à un article sous forme de liens. Vous pouvez cliquer sur le titre de la catégorie afin d'afficher son contenu. Le Contenu des articles est affiché dans une liste.

Vous pouvez définir un certain nombre de paramètres pour cet élément de menu, qui contrôle le nombre et la manière dont les liens sont affichés par catégorie.

 

Icones de la barre d'outils

Retour: Retour à l'écran précedent et sélectionnez un nouveau type d'élément de menu à créer.

Sauver: Sauvegarde les changements et retour à l'écran principal.

Appliquer: Sauvegarde les changements et reviens à l'écran en cours.

Annuler: Annule toutes modification, création sur le nouveau menu. Les changements ne sont donc pas enregistrés.

Aide: Cet écran d'aide.

 

Détails

Nom: Inscrivez le nom de l'élément de menu tel que vous voulez qu'il apparaisse sur votre site.

Section: Sélectionnez le contenu de l'article que vous voulez afficher. Vous ne pouvez sélectionner qu'une seule section.

URL: Le lien pour cet élément apparaîtra après avoir cliqué "sauver/appliquer".

Elément Parent: Sélectionnez "top" si vous vous voulez que votre nouvel élément de menu apparaisse dans le menu de navigation principal (main menu). Vous pouvez choisir de mettre cet élément en sous menu d'un élément du menu principal en choisissant ce dernier comme élément parent.

Ordre: Ajoute les nouveaux liens à la suite des éléments de menu déjà existants. Après avoir sauvegardé l'élément, vous pouvez lui choisir une nouvelle place dans la liste en la sélectionnant dans le menu déroulant. Sa nouvelle place sera APRES le lien sélectionné.

Langue: Sélectionnez la langue que vous voulez avec cet élément. Si vous sélectionnez "Toutes", ce sera disponible et visualisable, sous tous les langages.

Niveau d'Accès: Choisir qui peut avoir accès à cet élément.

Publié: Choisissez si votre nouvel élément sera défini comme publié oui ou non. Par défaut, il le sera.

 

Paramètres

Image du Menu: Si vous voulez vous pouvez montrer une petite image à côté de l'article du menu. Sélectionnez cette image dans la liste déroulante.Note: Pour que l'image soit visible, les paramètres "Afficher les icône de Menu" pour le menu du module, doit être mis à "Oui".

Utiliser seulement l'image de menu: Si vous avez mis les paramètres à 'Oui', alors au lieu du nom de menu il sera affiché en utilisant l'image de menu.

Titre de Section: Si vous voulez afficher le titre de la Section sélectionnez "Oui".

Classe Suffixe de la Page: vous permet de personnaliser le style de cette page. Le suffixe que vous saisissez ici est automatiquement ajouté à la classe CSS pour cette page.

Bouton Retour: Contrôle pour activer oui ou non le bouton retour à la page précédente. Cliquez sur le lien "Retour" vous fera retourner à la page précedente. Le menu déroulant vous propose 3 options:

  • "Configuration Globale" - Montre ou cache le lien basée sur l'arrangement dans la Configuration Globale.
  • "Cacher" - Cachez le lien retour, en ignorant les paramètres Global.
  • "Afficher" - Affichez le lien retour, en ignorant les paramètres Global.

Trier Par: Sélectionne l'ordre dans lequel vous voulez vour les éléments.

Liste de la Catégorie - Section: Ce paramètre contrôle si la liste des catégories disponibles sera visible ou non dans la section. Si vous sélectionnez 'Cacher' (non recommandé), Vous serez en mesure de voir la description de la Section. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Description: Obsolète. La description sera Toujours Visible.

Description Image: Obsolète. La description des catégories en mosImages est toujours affichée. Seules les images sont normalement visibles.

Liste de la Catégorie - Catégorie: Ce paramètre contrôle, si la liste des catégories disponibles sera visible lorsque vous aurez cliqué sur une catégorie et affichez ses éléments de contenu. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Caché.

Catégories Vides: Obsolète. N'a aucun effet. Les catégories vides ne sont plus affichées.

# Eléments de la Catégorie: Sélectionnez si vous voulez afficher le nombez d'éléments de contenu après le titre de la catégorie. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Description de la Catégorie: Select if you want to display also the description text for each category link. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Format de la Date: Choisissez le format avec lequel vous souhaitz voir les dates affichées. Si laissée vide, il utilisera le format du fichier de langue. Ce format est basé sur la fonction strftime de PHP. Pour plus d'info sur ces paramètres, consultez le site: . Ces opérations dépendent des réglages des 'URL SEF (Search Engine Friendly)' .

  • URL SEF = 'Non' or 'SEO Basic' - Effets format de date.
  • URL SEF = 'SEO Pro' - Aucun effet.

Colonne Date: Affiche ou cache la date d'infos. Vous pouvez utiliser le paramètre global ou la contourner localement. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Colonne Auteur: Affiche ou cache l'auteur. Vous pouvez utiliser le paramètre global ou la contourner localement. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Colonne Clics: Affiche ou cache les clics. Vous pouvez utiliser le paramètre global ou la contourner localement. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Tableau En-Tête: Obsolète. N'a aucun effet.

Barre de Navigation: Normalement, il ya une barre de navigation en bas du tableau. Vous pouvez choisir si vous voulez qu'elle soit visible ou non. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours visible (5 éléments par page).

Ordre: Si vous sélectionnez cette option, vous pouvez donner aux visiteurs la possibilité d'ordonner les lignes du tableau en fonction de leurs préférences. Une liste déroulante apparaît, voire au-dessus du tableau toutes les options disponibles. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Affichage: Vous pouvez laisser l'utilisateur modifier le nombre d'articles par page. Une liste déroulante s'affiche avec les options disponibles. Vous pouvez définir la valeur initiale des paramètres. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Non visible.

Nombre d'affichage: Sélectionnez le nombre d'articles par page pour ce menu.

Filtre: Vous avez la possibilité de montrer une zone de texte filtre au-dessus du tableau. Les utilisateurs peuvent saisir un mot pour filtrer les éléments de contenu qui sont affichés. Cette opération dépend des réglages des 'URL SEF' .

  • URL SEF = 'Non' or 'SEO Basic' - Fonctionne comme prévu.
  • URL SEF = 'SEO Pro' - Toujours Visible.

Champs Filtre: Sélectionnez le champs que vous voulez appliquer au filtre.
