Le Blog Elxis

Elxis Blog (eBlog) is a full-featured blog system for Elxis CMS. You can create unlimited number of blogs and assign an owner to each one. Each blog has its own settings and style. You can upload images, display YouTube and Yahoo videos, post comnments, etc.

 

Icônes de la Barre d'Outils

Publier et dépublier: Cliquez sur 'Publier' ou 'Dépublier' pour publier ou dépublier un article séléctionné (coché).

Nouveau: Crée un nouveau blog.

Editer: Edite le blog sélectionné.

Supprimer: Supprime le blog sélectionné.

Aidep: Cet écran d'aide.

 

Colonnes

Titre: Ceci est le titre du blog. Cliquer sur le titre pour l'editer.

Publié: Montre si le blog est publié ou non. Survoler cette icône pour voir un résumé de la publication de l'information. Cliquez sur cette icône pour modifier son état. Agit comme un interrupteur.

Blog owner: Affiche le nom du propriétaire (auteur) du blog. Cet utilisateur est capable de gérer son blog (poster des articles, modifier les paramètres, etc) à partir de la partie-public.

Postes: Nombre total de postes dans votre blog.

 

Pied de Page

En bas de page se trouve le pied de page

Navigation - Si le nombre de catégories est plus grand que le nombre d'articles par page, alors les contrôles de navigation seront activés. Vous pouvez les utiliser pour afficher les pages.

Affichage # - Utilisez cette commande pour modifier le nombre d'articles affichés par page. La valeur par défaut est contrôlée à partir de 'Longueur Liste' paramètres définis dans la configuration globale.
