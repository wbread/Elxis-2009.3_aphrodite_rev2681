Template Editeur HTML

Permet d'éditer le fichier templates/votre-template/index.php. Ce fichier est chargé en FrontEnd (partie publique)lorsqu'un utilisateur visite votre site. Veuillez vous assurer que le fichier est modifiable.

Vérifier que l'option "Rendre non-modifiable après sauvegarde" soit bien cochée, si vous souhaitez ne plus modifier ce fichier après changement.

 

Barre d'Outils

Sauver: Cliquez sur l'icone pour apporter vos changements.

Annuler: Si vous voulez annuler l'opération en cours cliquez sur l'icone "Annuler".

Aide: Cet écran d'aide.
