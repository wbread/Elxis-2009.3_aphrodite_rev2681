Blog Elxis (Eblog): Nouveau/Editer

Cette page est affiché lorsque vous créez ou modifiez un blog dans Elxis.

 

Icones de la barre d'outils

Sauver: Sauvegardez les changements et retournez à l'écran précédent.

Annuler: Annule la création de menu. Aucun changements ne sera fait. Les changements ne sont pas pris en compte si vous éditez un menu.

Aide: Cet écran d'aide.

 

Détails du Blog

Titre: Saisissez ici le titre de l'Article. C'est le titre qui apparaitra sur la page ou dans le tableau listant cet élément.

Titre SEO: Saisissez ici le titre de la page pour les moteurs de recherches. Utilisez uniquement des caractères latin. Plutôt que d'utiliser les espaces préférez les caractères '-' ou '_' . Il ne sera utilisé que si SEO Pro est activé mais c'est une bonne habitude de toujours le remplir.

  • Titre SEO suggéré - Cliquez pour laisser Elxis suggérer un titre SEO. Le titre SEO suggéré sera basé sur le champ 'Nom' .
  • Titre SEO validé - Après avoir saisi le titre SEO, Vous pouvez cliquer sur ce lien afin de vérifier si il est valide. Si il ne l'est pas veuillez corriger!

Slogan: Si vous le souhaitez, vous pouvez entrer un slogan qui sera affiché dans l'en-tête du blog. Notez que vous pouvez écrire en HTML dans ce champ. Par l'écriture HTML, vous pouvez personnaliser l'apparence de l'en-tête du blog.

Propriétaire du Blog: Sélectionnez le propriétaire du blog. L'utilisateur sera en mesure de gérer ce blog et de poster des articles sur la partie public.

Fichier CSS/RTL: Les fichiers CSS sont utilisés pour fournir des informations de style. Le fichier CSS/RTL est utilisé pour les langues dont la direction du texte est de droite à gauche. Chaque blog peut avoir son propre style et caractère. Sélectionnez ici la feuille de style que vous souhaitez utiliser pour le blog. Vous pouvez transférer d'autres feuilles de style (. Fichiers CSS) à l'intérieur du dossier components/com_eblog/css/.

Transfert: Choisissez ici si vous souhaiter que le propriétaire du blog pouisse être en mesure de transférer des fichiers multimédias tels que des images. Si cette option est activée, un dossier spécial sera automatiquement créé pour que l'utilisateur puisse enregistrer les fichiers téléchargés. L'utilisateur sera en mesure d'accéder aux fichiers de son dossier personnel, uniquement celui-ci. Les dossiers sont créés à l'intérieur du répertoire images/userfiles/, soyez sur que ce répertoire a les permissions en écriture (777). Les dossiers sont désignés par le nom d'utilisateur afin d'être en mesure de facilement savoir à qui ils appartiennent.

pour toutes les autres options d'aide intégré. Placez la souris sur l'icône pour voir les informations de chaque option.
