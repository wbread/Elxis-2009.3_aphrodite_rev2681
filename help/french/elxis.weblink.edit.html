Gestionnaire de Liens Web: Nouveau/Editer

Cette page vous permet d'ajouter ou modifier les détails pour un lien web.

 

Icônes de la barre d'outils

Transfert: Ouvre une nouvelle fenêtre contextuelle qui vous laisse téléchargez un fichier ou une image de votre ordinateur dans le dossier /media .

Sauver: Enregistre le lien web et retourne au gestionnaire.

Fermer: Abandonne l'action en cours. Les modifications seront perdues.

Aide: Cet écran d'aide.

 

Détails

Name: Entrez ici le nom du lien web tel qu'il apparaitra sur votre site.

Category: Sélectionnez la catégorie souhaitée. Il ne peut appartenir qu'à une seule catégorie.

URL: Entrez ici l'adresse URL du lien web.

Description: Ajoute un texte de description au Lien Web.

Ordering: Choisissez ici l'ordre du Lien Web dans une catégorie spécifique tel qu'il va apparaître sur le site lorsqu'il sera affiché dans la table. Lien Catégorie Menu Element. Ce paramètre ne sera seulement disponible qu' après avoir enregistrer ou appliquer les changements pour la première fois.

Language: Sélectionnez le ou les langages sous lesquels ce lien web sera disponible. Si vous sélectionnez "Tous", il sera disponible et affichable sous toutes les langues.

Approved: Décider ici si un lien proposé est approuvé par l'administrateur et peut être publié.

Published: C'est si le lien est publié ou non. Sélectionnez "Non" pour le cacher.

Screenshot: Vous pouvez assigner une capture d'écran du lien web en question. Vous avez trois options:

  • Ne pas afficher la capture: N'affiche pas la capture image du lien web.
  • Utiliser une image en local: Sélectionnez une image à partir du répertoiren /images/screenshots. Au-dessous de ce paramètre, il y a une liste déroulante qui liste toutes les images disponibles.
  • Charger une image à partir d'internet: Vérifie sur le web si des captures sont disponibles. Cliquez sur le bouton disponible et attendez l'extraction de l'image. Ce n'est pas certain que le sites web ont tous des captures disponibles à ajouter. Le répertoire /images/screenshots doit être modifiables pour stocker l'image trouvée.

 

Paramètres

Target: Avise si en cliquant sur le lien celui ci s'ouvrira dans la même fenêtre. Options disponibles:

  • Dans la même fenêtre avec le Navigateur
  • Dans une nouvelle fenêtre avec le Navigateur
  • Dans une nouvelle fenêtre sans le Navigateur
