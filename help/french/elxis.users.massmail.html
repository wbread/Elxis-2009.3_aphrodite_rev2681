Envoi groupé d'un message

Cette page vous permet d'envoyer un email ou un message à un ou plusieurs groupe d'utilisateurs.

NOTE: Seuls les utilisateurs du groupe Super-Administrateur ont autorité d'utiliser cette fonction.

 

Barre d'Outils

Envoyer: Cliquez pour envoyer les emails.

Annuler: N'envoye pas l'email et retourne à l'écran précédent.

Aide: Cet écran d'aide.

 

Détails

Groupe: Sélectionnez un groupe dans la liste. Pour envoyer un message à tous les utilisateurs, sélectionnez « Tous les groupes ». Vous ne pouvez faire qu'une seule sélection à la fois.

Message aux sous-groupes: Cliquez dans cette case à cocher pour inclure automatiquement tous les sous-groupes du groupe sélectionné.

Envoye en mode HTML: Le message sera envoie en mode HTML, en gardant les indicateurs html que vous avez inclus dans le 'Message'.

Subject: Saisissez le sujet de votre message.

Message: Saisissez votre message. Vous pouvez utiliser d'envoye en html, vérifiez que vous avez sélectionné "Envoye en mode HTML".
