Gestionnaire de Contenu Statique

Le gestionnaire de contenu statique montre une liste des objets statiques de votre site. Les articles statiques ne font pas partie d'une section ou d'une catégorie. Ils ne peuvent pas être ajoutés à un blog, à une catégorie ou une section. En outre, ils ne disposent que d'une Intro de texte et non pas de texte principal.

La seule façon de les afficher est de le faire à partir du menu.

 

Barre d'Outils

Nouveau: Créer un nouveau contenu statique.

Editer: Editer la page de contenu statique.

Publié et Dépublié: Publiez ou Dépubliez pour afficher ou non les contenus statiques sélectionnés. Seules les pages publiées seront visibles sur le site.

Traduction: Sélectionnez un contenu statique et cliquez sur cette icône pour traduire. Après la validation de la traduction, une nouvelle page sera créé en incluant le texte traduit.

Corbeille: Supprime les pages sélectionnées en les déplaçant à la corbeille. Ces pages de contenu statique ne sont pas visibles sur ce site. Ces pages ne sont actuellement pas supprimées. Elles peuvent être restaurées via le 'Gestionnaire de Corbeille'.

Aide: Cet écran d'aide.

 

Filtre

Vous pouvez utiliser le Champs Filtre pour limiter la liste de recherche des contenus statiques. Vous pouvez saisir une phrase ou un mot et appuyer sur la touche <Entrée>. Le tableau sera mis à jour et affichera uniquement les contenus statiques avec les caractères compris dans le titre. Si vous voulez tout afficher, laissez le champs vide et appuyer sur la touche <Entrée>.

 

Vue Simple/Etendue

Voici le champs filtre, vous trouverez une icône qui vous permettra de modifier le nombre de colonnes à afficher. 'Simple' Montre que les colonnes les plus importantes colonnes seront montrées avec le plus grand nombre d'entrées. 'Etendue', affiche toutes les colonnes. Vous pouvez essayer d'utiliser l'aide.

 

Colonne Filtres

Vous remarquerez que plusieurs colonnes ont une flèche à côté de l'en-têtes. Vous pouvez cliquer sur filtre. Vous pouvez utiliser ces filtres pour sélectionner les contenus statiques que vous voulez voir afficher. Cette fonctionnalité est adressée aux sites Web qui ont un grand nombre de contenus statiques. Il permet aux administrateurs de se concentrer uniquement aux contenus auquels ils s'intéressent.

 

Colonnes

Titre: C'est le titre de l'objet. Cliquer ici pour éditer son contenu. Une icône cadenas à côté d'elle montre qu'elle est à vérifier, cliquez sur le nom pour l'éditer. Puis cliquez sur Fermer ou Sauver pour fermer la page.

Titre SEO: SEO title used for this page.

Publié: Cela montre si l'élément est publié. Restez au dessus de cette icône pour lire un sommaire de l'information à publier. Cliquez dessus si vous voulez faire une modification de son état.

Ordre: Montre l'ordre des éléments du contenu statique. Vous pouvez le modifier en editant 2 # et en cliquant sur l'icone "Sauvegarder l'ordre" situé à coté de l'en-tête de colonne "Ordre".

Accès: Ceci est le niveau d'accès de l'utilisateur pour l'item du menu. Cliquez sur Niveau d'accès, si vous voulez modifier sa valeur.

ID: Ceci est l'identifiant ID de l'élément dans la base de données.

Liens: Le nombre d'éléments de menu qui sont liés avec cettes page.

Auteur: Cela indique le nom de l'auteur du contenu statique. Cliquez dessus pour visualiser l'information sur l'auteur.

Langue: Cela indique que l'article de contenu statique est publié sous différentes langues. Si vous sélectionnez "Tous", il sera disponible et visible dans toutes les langues.

Date: Date de création.

 

Pied de Page

En bas de page il y a le pied de page.

Navigation - Si le nombre de messages est plus grand que le nombre de messages affichés par page, alors les contrôles de navigation sont activés. Vous pouvez les utiliser pour changer de pages.

Affichage # - Utilisez ce contrôle pour changer le nombre de messages affichés par page. La valeur par défaut est définie par le paramètre 'Longueur de Liste' dans la Configuration Globale.

Légende - Affiche et explique les différents états d'une publication statique. Ces icônes sont affichées dans la colonne 'publié' .
