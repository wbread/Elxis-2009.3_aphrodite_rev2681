Menus: Copier éléments

Cette page vous laisse copiez un Menu et tous les Articles du Menu. Une liste affiche le nom du Menu et les Articles du Menu copiés.

 

Barre d'Outils

Copier: Cliquez sur cet icône pour copier un élément de menu à sélectionner dans le menu.

Annuler: Cliquez sur cet icône si vous voulez arrêter et annuler cette action.

Aide: Cet écran d'aide.

 

Contrôles

Copier au Menu: Listes des menus disponibles. Sélectionnez le menu que vous souhaitez copier.

Eléments de Menu copiés: Dresse la liste des éléments du menu qui sont copiés.
