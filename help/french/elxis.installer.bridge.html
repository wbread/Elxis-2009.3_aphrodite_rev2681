Installer un Pont

A partir de cette page, vous pouvez installer de nouveaux ponts dans Elxis.

 

Avant de poursuivre, veuillez noter que certains répertoires doivent être modifiables. Veuillez consulter la liste des répertoires et de leurs autorisations avant de continuer. Si un répertoire est non-modifiable au lieu d'être modifiable, l'installation échouera.

Notice de Sécurité: Après l'installation, n'oubliez pas de redéfinir les permissions des répertoires en non-modifiable.

 

Comment Installer - Transférer un fichier

C'est la plus courante et la plus facile des méthodes.

Les nouveaux Ponts devront toujours êtres au format .zip. Vous pouvez utiliser ce fichier pour installer le nouveau pont.

Cliquez sur 'Parcourir...'. ' de l'emplacement du pack de fichiers sur votre ordinateur.

Cliquez sur 'Transférer et installer. Elxis transférera le fichier sur votre site et installera le nouveau Pont. Pour terminer l'installation, veuillez suivre les instructions fournies avec le Pont.

 

Comment Installer - Installer à partir d'un répertoire

Cette méthode est à utiliser uniquement dans le cas d'une limite de configuration des fonctionnalités de votre serveur.

Décompressez les fichiers d'installation du fichier .zip file sur votre PC.

Créez ou utiliser un répertoire vide sur votre site web et transférez le (via FTP), extraire les fichierss (inclus les sous- répertoires).

Saisissez dans le champs 'Installation' , le chemin et le répertoire de destiantion des fichiers.

Cliquez sur le bouton 'Installer' .

Pour terminer l'installation, veuillez suivre les instructions fournies avec le Pont.

 

Comment désinstaller

Si vous voulez désinstaller un Pont aller dans Composants -> Ponts, sélectionnez dans la listé le Pont à désinstaller et cliquez sur l'icône "Désinstaller" .
