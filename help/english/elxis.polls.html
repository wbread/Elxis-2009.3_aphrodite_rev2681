Poll Manager

This page shows a list of the Polls you may display on this site. You can add new polls or edit existing ones.

 

Toolbar Icons

Publish and Unpublish: Publishes or unpublishes the selected poll(s). Only published polls are visible in this web site.

New: Creates a new poll.

Edit: Edits the selected poll.

Delete: Delete selected poll.

Help: This Help Screen.

 

Columns

Poll Title: This is the name of the poll. Click to edit. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the poll.

Published: Shows if the poll is published or not. Click to change state. Acts as a toggle switch.

Locked: Shows if the poll is locked to public (users can view results but not vote). Click to change state. Acts as a toggle switch.

Options: Not used.

Lag: Is the minimum time (in seconds) that has to pass before a user is allowed to vote again in a poll. Default time is 86.400 seconds (24 hours).
