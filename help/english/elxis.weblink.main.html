Weblinks Manager

Weblink Manager is the place from where you can add or administer the web links presented on this site. You can create new web links, publish, unpublish or delete them. This page shows a list of the web links.

 

Toolbar

Upload: Opens a new popup window which lets you upload a file or an image from your computer to /media folder.

Publish and Unpublish: Publishes or unpublishes selected web link(s). Only published web links are visible in this web site.

New: Creates a new web link.

Delete: Deletes selected web links.

Help: This Help Screen.

 

Filter

You can use the Filter field to limit listed web links or search for specific web link. You can type a phrase or part of a word and press <Enter>. The table will be updated showning only the web links that have these characters in their title. If you want to show again all web links, leave empty and press <Enter>.

 

Columns

Title: This is the title of the web link. Click to edit. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the item.

Published: Shows if the web link is published or not. Click to change state. Acts as a toggle switch.

Approved: Shows if the web link is approved or not. Generaly, web links submitted by users need approval prior being published in this site.

Reorder: You can change the order of web links, by using the arrows.

Category: This is the category the web link belongs. Category name acts as a link to category parameters.

Hits: Shows the number of times a link was clicked.

Language: Shows for which site languages this web link is available. It will be shown only if users are viewing this web site in one of the listed languages. If it is set to 'All', then it will be visible for all languages.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of web links is larger than the number of web links displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed web link per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.

 

Title: This is the name of the Weblink. Click the name to edit the Weblink.

Category: This is the category that the Weblink is assigned to.

Hits: This is the number of hits that the Weblink has received.

Checked Out: This shows the name of a user if this Web Link item is "Checked Out". You will not be able to edit an item if it is checked out by another user. It must be checked back in by an administrator by going to System -> Global Checkin. It can also be checked back in by the user who has checked the item out by clicking the "Edit" icon of the Weblink item and then clicking the the "Save" or "Cancel" icon.

Approved: This is whether the Weblink has been approved for publication. Remember that user submitted links are not automatically approved.

Reorder: Click the icons to move the record up or down in the list.

To Publish or Unpublish one or more Weblinks, select the checkbox(es) next to the name of the Weblink(s) and click either the "Publish" or "Unpublish" icon.

Click the "New" icon to add a new Weblink.

Click the name of the Weblink to edit the details, or select the check box next to the name and click the "Edit" icon.

To delete a Weblink, select the check box next to the name and click the "Delete" icon. You will be asked to confirm the delete operation. Select "Yes" to delete the Weblink or "No" to cancel.
