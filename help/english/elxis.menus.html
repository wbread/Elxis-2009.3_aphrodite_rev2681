Menus Manager / Main Menu

This page shows a list of the Menu Items associated with the menu you have just chosen.

Max Levels: Select the maximum number of levels depth that the menu tree will display.

Filter: Type in the keyword to search for text in the Menu Item titles.

 

Toolbar Icons

New: Click this icon to create a new Menu Item

Edit: Click this icon to edit an existing Menu Item.

Publish and Unpublish: Click 'Publish' to publish selected menu items. Click 'Unpublish' to unpublish selected menu items. Only published items are visible to the web site.

Move: Click this icon to move selected menu items to another menu.

Copy: Click this icon to copy selected menu items to another menu.

Trash: Click this icon to send selected menu items to trash bin. The menu items may be recovered through the Admin Menu: Site->Trash Manager.

Help: This Help Screen.

 

Columns

Menu Item: This is the name of the link as it will appears at this site. You can click it, to edit menu item properties. A padlock icon next to it shows that it is checked-out. To check-in, click it to open properties screen and click 'Save' or 'Close' toolbar icons.

Published: This shows whether the Menu Item is published or not. Click the icon to switch its state.

Reorder: Use arrows to move Menu Items up and down in the list.

Order: This shows the order of Menu Items in the menu. To change the order, type in the order number you wish to have. Then click the "Save Order" icon next to the column heading.

Access: If you want to change the "Access Level" for a Menu Item, click it. A pop up window will appear showning all available user groups. Select the one that you wish. Only members that belong to that or higher groups will be able to see it.

Itemid: This shows the Menu Item's "id".

Type: Shows the type of the Menu Item. If the type is one of "Link – Content Item" or a "Link – Autonomous Page", act also as links to the content.

CID: This shows the Menu Item's "componentid".

Language: Shows the language(s) for which the Menu Item will be visible. Only if user has selected to view the site using the language(s) specified here, he will be able to see the Menu Item. Select 'All' in order to make it available for all languages.
