Menus: Copy Menu Items

This page is shown when you have selected to copy one or more menu items to another menu. If you have selected to copy a parent menu item, then all its child items are copied with it

 

Toolbar

Copy: Click this icon tocopy the menu item(s) to the selected menu.

Cancel: Cancel action and return to previous screen.

Help: This Help Screen.

 

Controls

Copy to Menu: Lists the available menus. Select the menu you want to copy menu items to.

Menu Items being copied: Lists the menu items that are being copied.
