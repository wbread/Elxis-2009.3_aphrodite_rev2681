SoftDisk

SoftDisk is a virtual storage area for variables and parameters of any kind. You can add new or edit/delete existing SoftDisk's entries except the ones labeled as owned by the system. Also entries marked as "Undeletable" can only be edited. For naming SoftDisk's sections and value names you are only allowed to use capital latin letters, digits and underscore (_).

 

Toolbar

New: Create a new SoftDisk variable.

Edit: Edit selected SoftDisk variable.

Delete: Delete selected SoftDisk variable. You can not delete variables marked as 'Undeletable'.

Help: This Help Screen.

Update system: Update system specific values stored in SoftDisk.

 

Section Filter

You can use the Section filter to display only SoftDisk variables that belong to the selected section.

 

Columns

Checkboxes column: A padlock icon, informs that this variable cannot be deleted.

Type column: Shows the type of SoftDisk variable. Available types are:

  • Decimal (i.e. - 10.45)
  • Email (i.e. - user@domain.com)
  • Image (i.e. - http://www.domain.com/image01.png)
  • Integer (i.e. - 10)
  • String (i.e. - Hello World)
  • Text (i.e. - Full Text)
  • Time
  • URL (i.e. - http://www.domain.com)
  • YESNO

Section: The section a SoftDisk variable belongs. SoftDisk section have nothing to do with Content Sections.

Name: Softisk variable name. Click to edit. Note: Hover the mouse over help icons to get info about the use of SoftDisk variables.

Value: Variable's value.

System: Informs if variable is of a system type or not.

Undeletable: You cannot delete variables marked as undeletable.

Last modified: Date and time, value was modified for the last time.

Section: (Displayed only when you view 'All Content Items'). This is the Section the item belongs. Section name acts as a link to section parameters.

Category: This is the Category the item belongs. Category name acts as a link to category parameters.

Author: Shows Content Item's author name. Author names act as links to author details.

Language: Shows for which site languages this Content Item is available. It will be shown only if users are viewing this web site in one of the listed languages. If it is set to 'All', then it will be visible for all languages.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of SoftDisk variables is larger than the number of variables displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed items per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
