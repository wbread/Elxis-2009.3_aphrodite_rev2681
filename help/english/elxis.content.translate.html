Automatic Translation

With Elxis CMS you can translate content items and autonomous pages from their current language to another. Translated content forms a new content item or autonomous page.

Translation service is available only when Elxis is hosted on a real web server. Translation service does not work in local installation.

Directory /tmpr must be writable.

 

Toolbar

Translate: Click to translate content. First you have to select source and destination languages.

Cancel: Cancel and return to previous screen without performing the operation.

Help: This Help Screen.

 

Select Source and Destination Languages

Select the source and destination language pair. Please note that Elxis cannot check if the source language that you have selected is the actual language used by the article. It is up to you make the right choice.

 

Item Being Translated

Shows the article that has been selected for translation.
