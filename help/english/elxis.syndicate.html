Syndicate

This page controls syndication settings for this web site. By the term 'syndication' we mean that you will allow other web sites to display and use news from your site. Syndicated news are displayed in other web sites as Titles followed by a short description. Titles act as links to this site. People have to visit your site in order to view the full article. Only content items that are displayed in the Frontpage are used for syndication.

For syndication to work, and people being able to get news from your site, you have to publish the Syndicate module.

 

Parameters

Cache: Select if syndicated files will be cached or not. If you enable cache, syndicated info will be displayed faster in other web sites and your site will perform better. The drawback is that syndicated news will be updated only every 'Cache Time' seconds. If you do not enable cache, syndicated news will be made immediately available to other web sites, but your site's performance will be degraded, because your site will be queried every time someone visits any site where your News Feed is syndicated.

Cache Time: This is the number of seconds before your site's database is checked for updates. The default in Elxis is 3600 seconds, or 60 minutes. After this time has elapsed, the News Feed will refreshes itself from your database thus displaying any new items you have published through the Frontpage.

#Items: The number of items you want to syndicate and display in your News Feed.

Suyndication Title: Type the text that you want to be used as title above your syndicated articles, when they are displayed in other web sites. Title will also act as a link to your site.

Description: Type a brief statement to describe your News Feed. If none is entered, it will default to "Elxis site syndication".

Image: Select an image, from the drop-down menu to be included in your News Feed. Used images are stored in /images/M_images directory.

Image Alt: Type the ALT text for the image.

Limit Text: Select if you want to limit the text displayed bellow syndicated content titles. If you select 'Yes', then check the following parameter, otherwise ignore it.

Text Length: Select the number of words that you want to be displayed bellow syndicated content titles. For no text, enter 0.

Order: Select in which order syndicated content will be displayed in other web sites. Available options are:

  • 'Default': Arranges items according to the order set in the FrontPage Manager.
  • 'FrontPage Ordering': Same as above.
  • 'Oldest first': Arranges items in chronological order, with the oldest items first.
  • 'Most recent first': Arranges items in chronological order, with the newest items first.
  • 'Title Alphabetical': Arranges items by title in alphabetical order.
  • 'Title Reverse-Alphabetical': Arranges items in reverse-alphabetical order.
  • 'Author Alphabetical': Arranges items by author in alphabetical order.
  • 'Author Reverse-Alphabetical': Arranges items by author in reverse alphabetical order.
  • 'Most Hits': Arranges items according to the number of hits, with the items with the most hits first.
  • 'Least Hits': Arranges items according to the number of hits, with the items with the least hits first.

Live Bookmarks: Activate support for Firefox Live Bookmark functionality.

Bookmark File: If you wish, you can enter a special file name. If none is entered, the default file will be used.
