Welcome to Elxis
The Professional Open Source CMS

Elxis is a revolutionary Open Source CMS that is fully UTF-8 compliant, builds multilingual web sites and is Search Engines Friendly.

It is designed to address the needs of millions of users and businesses that need to have a simple to use, and powerful Content Management System, that has all the needed functionality.

Elxis CMS is developed and supported by Elxis Team and volunteers around the world.

 

Elxis 2008

Elxis 2008 has a number of unique features that make it ideal for use by SMBs (Small and Medium Business), schools, public sector, tourism industry, non-profits, personal, etc. We are sure that you are going to love it because it is flexible, stable, expandable, easy to use and well and professionaly supported.

 

Administration Interface (Back-End)

Using Administration interface you build and edit your site. It is also called, Back-End. Administration interface follows a user friendly design with a number of unique features:

  1. Open Source. All code is open and accessible.
  2. User Friendly. Elxis grows with your knowledge and experience. Easy for new users. Powerful for experienced administrators.
  3. Multilingual. You can view administration interface in your preferred language.
  4. Simple and Extended Interface View. Select the way interface will look. Use Extended, for detailed info, Simple for everyday use.
  5. Easy Interface Administration interface is easy to follow.
  6. Expandable. Install new add-ons (Components, Modules, Bots, Templates) and expand the functionality of Elxis CMS.
  7. Language Manager. Language manager serves Front-End and Back-End languages. Elxis is multilingual both in Front and Back end.
  8. UTF-8. Elxis uses the UTF-8 encoding.
  9. Strong SEF URLs support. SEF stands for Search Engines Friendly URLs. Elxis has an advanced mechanism that produces SEF enabled URLs.
    • SEO Pro: produces human-readable URLs. Administrators can define the human-readable text used.
      For example:
      faqs/managing_content/editing-content.html
    • SEO Basic: produces SEF URLs that are not human readable.
      For example:
      content/category/3/18/33/
    • No SEF: produces normal URLs.
      For example:
      index.php?option=com_content&task=category§ionid=3&id=18&Itemid=32
  10. SEO. SEO stands for Search Engines Optimization. Define your META Keywords and META description for each content item.
  11. Ajax enabled. Ajax is used in crucial parts of Elxis Administration interface to easy administrators life and speed up things.
  12. Automatic Translations. A great feature for multilingual web sites. Translate content (content items and autonomous pages) from one language to another. Translated content retains formating.
  13. Optimized Cache. Use the optimized Cache to speed up your web site.
  14. Templates. Site's look is controlled by templates, separating data from the appearance. Elxis uses 3 types of templates. Site templates, Administration templates, Login Screen templates.
  15. WYSIWYG editor. Add or Edit content using the embedded WYSIWYG editor. Easy as using well known desktop word processors. You can add or remove functions.
  16. Access Manager. Manage user group rights. Create new user groups. Define what are allowed and what are not allowed to do. Access Manager is a must-have system on professional web sites.
  17. New Media Manager. Use this poweful feature to manage all your site's media files. Upload, move, delete media files. Supports the following file types:
    • Images (gif, jpg, png, bmp, jpeg),
    • Video (mpg, avi, wmv, mov, flv, mpeg, asf, rm, nsv, xvid),
    • Audio (mp3, wav, mid, ram, wma, ogg, aac),
    • Other (doc, xls, csv, ppt, swf, pdf, txt, zip, tar, rar, gz, tgz),
  18. SoftDisk. Control several aspects of Elxis functionality. SoftDisk is a unique system designed with administrators and developers in mind. You can define your own variables and use them in Elxis.
  19. Bridges. Bridges are another unique feature of Elxis CMS that allows it to use third party applications as being part of Elxis.
  20. Logins Recorder. Logs the users that log into your administration interface.
  21. Database Manager. Administer and optimize your database directly from Elxis. Time saver for every day use. Ideal tool for system administrators. Import, Export, Backup database data.
  22. Custom User Profiles. Each user has its own profile that is composed from predefined and extra fields. You can specify any number of extra fields to be included in a user's profile. You can set which fields are required and which are required during the registration process. Feature is available through the User Manager.
  23. Filters. Elxis provides a flexible mechanism for filtering displayed content while in Back-End. You can filter almost every aspect of content and focus on the items that you need to check or edit.
  24. Tools. Special utilities available only to administrators that help them, among other things, secure, maintain and update their sites.
  25. Statistics. Provides basic statistic data for your site.
  26. Trash Manager. Deleted content is first moved to trash. Then it can safely be permanently deleted.
  27. Menus Manager. Create menus the easy way. Multiple menu levels are supported.
  28. Archive. Archive your old articles. Present them using special ways.
  29. Content Organization. Content is organized in Sections, Categories, Content Items and Autonomous Pages for easy navigation and maintenance.
  30. Banners Management. Create and manage yours or your customers banners.
  31. Contacts Management. Define and manage your contacts. Let users contact the right department of your company.
  32. Mass Mail. Mass Mail news to your registered users.
  33. News Feeds. Get connected with sources of information and present them on your site.
  34. RSS Feeds. Be the source of information for other sites.
  35. Polls. Define and administer polls.
  36. Web Links. Create lists of links in a categorized form. Get web sites screenshots directly from internet.
  37. Manage Submitted content. Administer content that is submitted by users.
  38. Modules Manager. Manage installed modules. Create your own. Install new ones.
  39. Bots. Manage installed bots. Install new ones.
  40. Multilingual Online Help. Online context help screens and tip icons.

 

Front-End

Front-End is what visitors see. It is your web site. All advanced Elxis administration features exist to make your site look great and safe.

  1. Multilingual Content. Elxis is a true multilingual CMS that uses UTF-8 encoding. You can create content for all installed languages. You can have different menus and menu items based on the language used. You have the freedom, to select the structure of your contents based on the language used. For example in English you can have a menu with 5 menu items that lead to the corresponding content in English. When you switch to German, you have the freedom to present a different 3 menu items menu, that lead to content that is suitable to German speaking users. That is a lot of freedom and versatility!
  2. Search Engines Optimized. Elxis based web sites are Search Engines Optimized.
  3. Search Engines Friently URLs. Forget the ugly URLs. Elxis 2008 based web sites, can have human-readable URLs.
  4. Fast. Web sites load faster.
  5. Secure. Take advance of Elxis tools to secure your site. (Security is a very complex thing. Other parameters, beyond Elxis may affect it).
  6. Template Based. Web site appearance is controlled by templates. You can change the look of your site with just a few clicks.
  7. CSS Based Designs. Distributed templates are CSS based.
  8. Frontpage Module. New Frontpage module for displaying important content in an attractive way. Please do not confuse it with Frontpage component.
  9. YAML Template Support. Elxis supports YAML based template layouts.
  10. Flexible Modules. Show modules where ever you need them.
  11. Easy Navigation. Publish as many menus as you need. Supports dynamic menus.
  12. Related Content. Each article can have a list of related links.
  13. Export Articles as RTF. You can create RTF files out of articles and save them in your computer.
  14. Print and Mail Articles. You can print and/or email articles.
  15. Polls. Display polls.
  16. Banners. Display banners on your web site.
  17. Improved Access Level Based Content. You can present content based on user's access level.
  18. Wrapper. Display content from other applications or from other web sites using a wrapper.
  19. Video. Display flv video clips as part of your content.
  20. Content Rating. Let users rate articles.
  21. Intro and Main Text. Articles are divided in Intro and Main Text. You can show Intro text and let users decide if they are interested to see Main text.
  22. Flexible Content Display. You can display available content in various forms that are directly related with the way content is organized in Elxis. Some of the most common options are:
    • Blog view
    • Table view
    • Section view
    • Category view
    • Single Item view
    • Component view
    • Autonomous Pages
  23. Contact Forms. Use contact forms to let users contact, you or departments of your company.
  24. Web Links. Present lists of web links organized in categories.
  25. Embedded Search Engine. Users can search your web site for specific words.
  26. Who Is Online. Display users that are currently visiting your site.
  27. Let Users Submit Content. Users can submit content. Content is checked from administrators prior to publishing.
  28. Show Member Profiles. You can show random member profiles, or you can show list of available members.
  29. Manage Own Profile. Members can manage their own profiles.
  30. Custom Footer. You can customize the footer of your site, by editing a single file.
  31. Login Screens. Elxis supports custom login screens. You can design your own login screens and select which one will be used to login to Back-End.

 

Compatibility!

As first version of Elxis was based on Mambo 4.5.3, current version maintains a level of compatibility with it. As Joomla was also based on Mambo, Elxis maintains also a compatibility with it.

Components, Modules and Bots written for Mambo, can be modified to become compatible with Elxis CMS. The same is true for those that are compatible with Joomla.

Most of the Templates written for Mambo or Joomla, are compatible with Elxis or they require minor adjustments.
