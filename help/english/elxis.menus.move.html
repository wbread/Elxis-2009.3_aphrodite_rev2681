Menus: Move Menu Items

This page is shown when you have selected to move one or more menu items to another menu. If you have selected to move a parent menu item, then all its child items are moved with it

 

Toolbar

Move: Click this icon to move the menu item(s) to the selected menu.

Cancel: Cancel action and return to previous screen.

Help: This Help Screen.

 

Controls

Move to Menu: Lists the available menus. Select the menu you want to move menu items to.

Menu Items being moved: Lists the menu items that are being moved.
