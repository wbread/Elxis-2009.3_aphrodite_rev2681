Banners Manager

Using Banners Manager you can manage the banners displayed in this site.

Toolbar Icons

Publish and Unpublish: Publishes or unpublishes selected banners. Only published banners are visible in this web site.

Upload: Opens a new popup window which lets you upload a file or an image from your computer to /images/banners folder.

New: Creates a new banner.

Edit: Edits the selected banner.

Delete: Deletes the selected banner(s).

Help: This Help Screen.

 

Column Filters

You will notice that several columns have an arrow next to there headers. When you click them a filter window popups. You can use this filters to select the banners that you want to be displayed.

 

Columns

Banner Name: This is the name of the banner. Click to edit banner.

Client Name: Each banner is associated with a client. This is the client's name.

Publish: Shows if the banner is published or not. Click to change state. Acts as a toggle switch.

Impressions Made: Tells you how many times a banner was displayed in this site.

Impressions Left: Shows how many impressions are left. 'Unlimited' means that there is no limit.

Clicks: Shows how many times a banner was clicked.

% Clicks: Measures the success of this banner.
