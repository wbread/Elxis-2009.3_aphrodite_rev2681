Bridge Manager

Bridges are a unique feature of Elxis CMS that allows it to use third party applications as being part of Elxis. This page shows installed Bridges in your site.

Bridges are installed separately.

 

Toolbar Icons

Publish/Unpublish: Publishes or unpublishes selected Bridges.

Edit: Click this icon to edit selected Bridge.

Uninstall: Removes selected Bridge.

Enable/Disable Bridge System: Click this icon to enable or disable Bridge System. When Bridge System is disabled all Bridges functions, modules and APIs are stopped. Use this option if you do not use any Bridge in your site. You can re-enable it any time you wish.

Help: This Help Screen.

 

Columns

Bridge: Bridge name.

Published: Shows if the Bridge is published or not. Click to change state. Acts as a toggle switch.

Description: A short description of Bridge's functionality.

Author: Author's name.

Version: Bridge's version.

Date: Date info.

 

Registry

Registry keeps a list of all available Bridges for Elxis and a unique identification number for each of them. Do not modify Registry by hand! Keep your registry file up to date. If you experience problems saving new registry make sure that /administrator/components/com_bridge/registry/registry.ini file is writable.

 

Editing Bridges

Each Bridge is unique. It has its own set of parameters. By editing a Bridge you can change the way Elxis and the third party application interacts. Note that in some cases a Bridge can be bridged to itself! In that case the third party application is the Bridge itself!
