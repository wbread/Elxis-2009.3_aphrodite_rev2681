Bots: New/Edit

The Bots: New/Edit page lets you edit the Site Bots.

 

Toolbar Icons

Save: Save changes and return to Bots Manager.

Apply: Save changes and remain to this screen.

Cancel: Cancel changes and return to Bots Manager.

 

Bot Details

Name: Type the name of the Bot as it will appear in Bots Manager. This has no impact on your site.

Folder: Shows the folder bots files are stored.

Bot file: This is the name of the Bot file as it appears in the folder.

Bot Order: Choose the order Bot as listed.

Access Level: Select the user group a user must belong, in order to view this Bot.

Published: Select "Yes" to make it available to this site. Select "No" to hide it.

Description: This is a short description of Bot's function.

 

Parameters

If available, they will be displayed here.
