Elxis Blog

Elxis Blog (eBlog) is a full-featured blog system for Elxis CMS. You can create unlimited number of blogs and assign an owner to each one. Each blog has its own settings and style. You can upload images, display YouTube and Yahoo videos, post comnments, etc.

 

Toolbar

Publish and Unpublish: Publishes or unpublishes selected blogs(s). Only published blogs are visible in the site front-end.

New: Creates a new blog.

Edit: Edits the selected blog.

Delete: Deletes the selected blog.

Help: Shows this help screen.

 

Columns

Title: This is the title of the blog. Click Title to edit blog.

Published: Shows if the blog is published or not. Hover over this icon to see a summary of publishing information. Click this icon to change publish state. Acts as a toggle switch.

Blog owner: Shows Blog's owner (author) name. This user is able to manage his blog (post articles, change settings, etc) from the site front-end.

Posts: Blog's total number of posts.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of blogs is larger than the number of items displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed blogs per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
