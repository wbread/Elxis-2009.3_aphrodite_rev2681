Access Manager - Add/Edit User Group

Use this screen to add a new user group or edit an existing one. You can change user group's name and select the parent user group it belongs.

Important Note: Please use this screen with caution. Failing to do so, may lead to a non functional web site.

 

Toolbar Icons

Save: Saves changes.

Cancel: Cancel changes.

Help: This Help Screen.

 

Group Details

Group Name: Type user group name.

Parent Group: Select from the drop down list the parent group.

Existing User Groups: Shows the hierarchy of existing groups.
