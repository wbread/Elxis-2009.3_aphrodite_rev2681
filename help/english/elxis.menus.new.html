Menus: New Menu Item

There are several different types of Menu Items. Each type has a special role. When we create a New Menu Item we must choose its type.

To create a new Menu Item, you can:

  • Click on its type name. A new screen will appear where you can modify its parameters.
  • Select a type and click the 'New' toolbar icon. A new screen will appear where you can modify its parameters.

 

Toolbar Icons

Next: Select the menu item type you wish and click 'Next'. A new window will open with the name "Add Menu Item :: The-type-of-link-chosen", where you can define its properties.

Cancel: Cancels the action and returns to previous screen.

Help: This Help Screen.

 

Note: You can find detailed information about each type, by clicking the 'Help' icon on its own parameters screen.
