Weblinks Manager: New/Edit

This page allow you to add a new weblink or edit an existing one.

 

Toolbar Icons

Upload: Opens a new popup window which lets you upload a file or an image from your computer to /media folder.

Save: Save changes and return to previous screen.

Close: Aborts the creation of the new weblink and return to previous screen.

Help: This Help Screen.

 

Details

Name: Type here the name of the web link as it will appear on this site.

Category: Select the category this web link will belong. It can belong to only one category.

URL: Type here the URL address of the web link.

Description: Provide a description for the web link.

Ordering: Select the order in which this web link is listed, relative to other web links in this Category. This parameter will be available only after saving or applying changes for the first time.

Language: Select the language(s) under which this web link will be available. If you select "All", it will be available and viewable, under all languages.

Approved: Mark web link as approved or not. Only approved web links are visible in this site.

Published: Select "Yes" to make it available to this site. Select "No" to hide it.

Screenshot: You may assign a screenshot image, to this web link. You have three options:

  • Don't show screenshot: Don't show an image for this weblink.
  • Use a local image: Select one of the images that exists in /images/screenshots directory. Use list to pick an image.
  • Load new image from internet: Checks the web for available screenshots. Click available buttons and wait for image retrieval. It is not certain that all available sources will have a screenshot for the web site that you are adding. Directory /images/screenshots must be writable in order to store retrieved image.

 

Parameters

Target: Select from the list how you want the linked page to open. Available options are:

  • Parent Window with Browser Navigation - Linked page replaces this site.
  • New Window with Browser Navigation - Use a new window to show the linked page. The browser window will have all navigation controls.
  • New Window without Browser Navigation - Use a new window to show the linked page. The browser window will have no navigation controls.
