NewsFeeds Manager: New/Edit

From this page you create or edit a newsfeed.

 

Toolbar Icons

Save: Save changes and return to previous screen.

Cancel: Cancel changes and return to previous screen.

Help: This Help Screen.

 

Details

Category: Select here the category for this newsfeed.

Name: Type here the name of the newseed as it will appear on this web site.

SEO title: Type here a Search Engines and Human-Friendly page title. Use only latin characters. Instead of spaces use '-' or '_' characters. It will be used only if SEO Pro is enabled, but it is a good habit to always fill it.

  • Suggested SEO title - Click it in order to let Elxis suggest a SEO title. Suggested SEO title will be based on 'Name' field.
  • Validate SEO title - After typing a SEO title, you can click this link in order to check if it is valid. If it is not please fix it!

Link: Enter here the link to the newsfeed as provided by the source.

Number of Articles: Define the maximum number of articles that will be displayed from this newsfeed.

Cache Time (in seconds): Type here the time (in seconds) between newsfeed refreshes. Between refreshes, newsfeeds will be read from cache. This approach improves site performance.

Ordering: Select the order in which this newsfeed is listed, relative to other newsfeeds in this Category. This parameter will be available only after saving or applying changes for the first time. You can also change order from News Feed Manager.

Published: Select "Yes" to make it available to this site. Select "No" to hide it.
