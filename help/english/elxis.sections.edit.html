Sections Manager: New/Edit

This page is shown when you are adding or editing a Section.

 

Toolbar Icons

Upload: Opens a new popup window which lets you upload a file or an image from your computer to /media folder.

Save: Saves the Section and returns to previous screen.

Apply: Saves changes and remains to this screen.

Cancel: Cancels changes and returns to previous screen.

Help: This Help Screen

 

Section Details

Scope: Shows the scopr of this Section. Default value 'Content'. Not editable.

Title: Type here the title of the Section as you want to be displayed in this site.

Section Name: Type here the name of the Section as you want to be displayed as heading.

SEO title: Type here a Search Engines and Human-Friendly page title. Use only latin characters. Instead of spaces use '-' or '_' characters. It will be used only if SEO Pro is enabled, but it is a good habit to always fill it.

  • Suggested SEO title - Click it in order to let Elxis suggest a SEO title. Suggested SEO title will be based on 'Section Name' field.
  • Validate SEO title - After typing a SEO title, you can click this link in order to check if it is valid. If it is not please fix it!

Image: If you want, you can select an image that will be used next to Section's description text. The dropdown lists images from the images/stories folder.

Image position: Select the position of this image. Available options are Left or right.

Ordering: Select here the order this Section is listed. Also affects the order Sections are displayed in the Front-End. New Sections by default are placed to the last position. Ordering can be changed after applying or saving changes.

Access Level: Select the user group a user must belong, in order to view this Section and its contents.

Published: Select "Yes" to make it available to this site. Select "No" to hide it. It also hides all categories and items that belong to this Section.

Languages: Select the language(s) under which this Section will be available. If you select "All", it will be available and viewable, under all languages.

Description: You can type a descriptional text for this Section.

 

Bellow Description, there are a number of special buttons. By default there are 'Image', 'Page Break' and 'Flash Video'. These are shortcuts for inserting Bots code. For example, clicking the 'Page Break' button, will insert '{mospagebreak}'. Alternatively, you can insert by hand Bots code. Effect is the same.
You can install and use more Bots.

 

Link to Menu

Provides an alternative way to create a new menu item of one of the following types "Section List", "Section Blog", "Section Archive Blog" that links to this Section.

Select a Menu: Select the menu that you want the menu item to be created in.

Menu Type: Select the type of the menu item.

Menu Item Name: Type the name of the menu item.

Link to Menu: Click this button to create the new menu item.

Existing Menu Links: Displays a list of all menu items that reference this Section.
