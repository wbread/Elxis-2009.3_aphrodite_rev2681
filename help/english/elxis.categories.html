Categories Manager

This page shows a list of the Categories defined on your site. They are ordered by the Section they belong to.

You can use this screen to add, edit, delete, move or copy Categories.

This page is called from:

  • Content Items
  • Contacts
  • Newsfeeds
  • WebLinks

 

Toolbar

Publish and Unpublish: Publishes or unpublishes the selected categories. Only items contained in published categories are visible in this web site.

New: Creates a new Category.

Move: Moves selected Categories (including content items), to another Section.

Copy: Copies selected Categories (including content items), to another Section.

Edit: Edits the selected Category.

Delete: Deletes selected Categories. This action cannot be performed if the Category contains content items.

Help: This Help Screen.

 

Columns

Category Name: This is the name of the Category. Click the name to edit the Category. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the Category.

Published: Shows if the Category is published or not. Click to change state. Acts as a toggle switch.

Order: Shows the current order of Categories. You can change it, by editing the order numbers and then clicking the "Save Order" icon next to column header.

Access: This is the user group that is allowed to view a Category. Click on it to change it. A popup window will appear.

Section: Shows the Section a Category belongs. Non editable. Click to edit Section.

Category ID: Shows Category's ID. Non editable.

# Active: Shows the number of published Content Items this Category has.

# Trash: Shows the number of Content Items belonging to this Category and are currently in the Trash Bin.

Languages: Shows for which site languages this Category is available. It will be shown only if users are viewing this web site in one of the listed languages. If it is set to 'All', then it will be visible for all languages.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of Categories is larger than the number of items displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed items per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
