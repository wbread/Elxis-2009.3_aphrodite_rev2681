Sections Manager: Copy

This page lets you copy selected Categories and Content Items into a new one.

This will copy the Categories listed and all the Content Items within these Categories to the new Section that it will be created.

 

Toolbar Icons

Save: Creates the new Section and returns to Sections Manager.

Cancel: Cancels the action.

Help: This Help Screen.

 

Copy Section

Copy to Section: Type here the name for the new Section.

Categories being copied: Lists the Categories that will be copied from source Section to the new Section.

Content Items being copied: Lists the Content Items that will be copied from the source Section to the new Section.
