Site Templates: Assign

This page lets you assign the chosen site template to specific Menu Items (pages). That way you can change the look of the site based on what the user selected to view.

The list 'Page(s)' displays all available Menu Items.

You can select multiple Menu Items, by holding the CTRL key while clicking.

If you choose 'None', the default site template will be used for all Menu Items (pages).

 

Toolbar Icons

Save: Saves your choice and return to Templates Manager.

Cancel: Cancel the action and return to Templates Manager.
