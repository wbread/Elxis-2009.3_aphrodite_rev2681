Languages Manager [Administrator]

Use Languages Manager (Administrator) to view, install and delete languages used at the Back-End.

This page shows the list of available administrator languages. Administrator languages are used in the Back-End of this site.

 

Toolbar

Default: Default is the language that is used by default for the administration of this site. To set an installed language as the administrator's default, select it and click the "Default" icon.

New: To install a new administrator language, click the "New" icon. You will be transferred to the install/uninstall screen.

Delete: To delete an administrator language, select it and click the "Delete" icon. You will be asked to confirm the deletion. Select the "Yes" icon to proceed, or the "No" icon to cancel. The administrator language files will be deleted from the server.

Help: Click the "Help" icon to view this Help Screen.

 

Columns

Language: This column shows the names of the installed administrator languages.

Flag: Shows the flag icon for this language.

Default: Shows the administrator language that is set as default. Only one administrator language can be the default.

Version: Shows the version of the language file.

Date: Shows the date the language file was created.

Author: Shows the author of the language file.

Author Email: Shows author's email address, if available.
