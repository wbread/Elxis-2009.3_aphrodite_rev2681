Banners Manager: New/Edit

This form is shown when you edit or add banners.

Toolbar Icons

Upload: Opens a new popup window which lets you upload a file or an image from your computer to /images/banners folder.

Save: Save banner and return to previous screen.

Close: Cancel changes and return to previous screen.

Help: This Help Screen.

 

Details

Banner Name: Type here the name for the Banner.

Banner Client: Select the client for this Banner.

Impressions Purchased: It can be either a specific number or unlimited. Type the number of purchased impressions or leave blank and click the "Unlimited".

Banner URL: Select the Banner image, from the list of available images. Banner images are stored in "images/banners" directory. The selected image is previewed at the bottom of the page.

Show Banner: Equivelant to publish/unpublish. Only published Banners are visible.

Click URL: Type the URL of the page or site (target) a user will be redirected if he clicks the banner.

Open Banner in: Select where the target page will be displayed.

Custom Banner Code: If you want to display a FLASH banner, then you can paste here the code. Tip: You have to select an image, although it will not be used.

Clicks: Shows the number of times this banner was clicked. Click 'Reset Clicks' button to reset it to zero.

Banner Image: Preview of the selected banner image.
