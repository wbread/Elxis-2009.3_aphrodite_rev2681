SoftDisk: Edit

Using this form you can add or edit softdisk variables.

For naming SoftDisk's sections and value names you are only allowed to use capital latin letters, digits and underscore (_).

 

Toolbar

Save: Save changes and return to previous screen.

Cancel: Cancel changes and return to previous screen.

Help: This Help Screen.

 

Details

This is the area where you can fill new SoftDisk's variable parameters or edit an existing SoftDisk variable.

When you are in an editing mode, Section, Name and Type parameters cannot be changed. You can only modify the value of the variable.

Also, when you are in an editing mode, you see a short description regarding the use of this variable.

Section: Select the Section the new SoftDisk variable will belong. By default, exist the Sections: Core, User profile and Users. By clicking the 'New' link, you can add a new Section. Please note the comment abouting naming Sections at the top of this help page.

Name: Type the name for the new SoftDisk variable. Please note the comment abouting naming SoftDisk variables at the top of this help page.

Type: Select the type of the SoftDisk variable. Value field is addapted to the type selected. Available types are:

  • Decimal (i.e. - 10.45)
  • Email (i.e. - user@domain.com)
  • Image (i.e. - http://www.domain.com/image01.png)
  • Integer (i.e. - 10)
  • String (i.e. - Hello World)
  • Text (i.e. - Full Text)
  • Time (i.e. 1190459123) Time is entered in Unix timestamp format (long integer containing the number of seconds between the Unix Epoch (January 1 1970 00:00:00 GMT) and the time specified.)
  • URL (i.e. - http://www.domain.com)
  • YESNO

Value: Type the SoftDisk variable's value.

Last modified: Shows the date and time, this variable was modified for the last time.
