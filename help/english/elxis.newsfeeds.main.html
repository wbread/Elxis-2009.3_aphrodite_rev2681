NewsFeeds Manager

Using NewsFeeds Manager you can define various sources of RSS newsfeeds. Users in the Front-End can check the newsfeeds and get informed about latest news broadcasted from these sites. They can then click a newsfeed and transfered to that site in order to read the full article.
Newsfeeds are organized in categories.

 

Toolbar

Publish and Unpublish: Publishes or unpublishes the selected newsfeed(s). Only published newsfeeds are visible in this web site.

New: Creates a new newsfeed.

Edit: Edits the selected newsfeed.

Delete: Deletes the selected newsfeed(s).

 

Columns

News Feed: Name of the newsfeed as it will appear on this site. Click to edit. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit. Then you have to Save or Close the newsfeed.

Published: Shows if the item is published or not. Click to change state. Acts as a toggle switch.

Reorder: You can change the order of newsfeeds, by using the arrows.

Category: This is the Category the newsfeed belongs. Category name acts as a link to category parameters.

#Articles: Shows the number of Articles that will be displayed from this newsfeeds source. You can change it by editing the newsfeed item.

Cache time (sec): This is the time between refreshes as defined in the parameters for the Newsfeed. Use of cache speeds up newsfeeds display.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of newsfeeds is larger than the number of newsfeeds displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed newsfeeds per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
