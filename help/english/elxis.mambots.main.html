Site Bots: Bot Manager

Bots are essentially code snippets which when called via various parameters either activate a program, script or perform a specific function. They can be as simple as a function which replaces typed text with code (such as typing to display an image in a content item) to full fledged third party libraries (such as WYSIWYG editors or other programs).

{mosimage}, {mospagebreak}, {mosflv}, are typical examples of bots.

Each bot has its own set of configuration parameters.

Bots are divided into 4 types, according to for what they are used:

  • content - Inserted into content and replaced by their output when viewing the Front-End.
  • editors - Editors accesible from the Front and Back End.
  • editors-xtd - Extensions to Editors.
  • search - Activated when users are searching the web site. Return results.

 

Filter

You can use the Filter field to limit listed botss or search for specific bot. You can type a phrase or part of a word and press <Enter>. The table will be updated showning only the bots that have these characters in their title. If you want to show again all bots, leave empty and press <Enter>.

 

Column Filters

You will notice that several columns have an arrow next to there headers. When you click them a filter window popups. You can use this filters to select the bots that you want to be displayed. This feature is addressed to web sites that have a large number of bots. It helps administrators to focus only to bots that each time interest them.

 

Toolbar Icons

Publish and Unpublish: Publishes or unpublishes selected bot(s). Only published bots can be used.

New: Creates a new bot.

Edit: Edit the selected bot.

Delete: Deletes/Uninstalls the selected bot(s).

Help: This Help Screen.

 

Columns

Bot Name: Bot's name.

Published: Shows if the bot is published or not. Click to change state. Acts as a toggle switch.

Reorder: You can change the order of bots, by using the arrows. That will be the order bots will be used.

Order: Shows the current order of bots. You can change it, by editing the order numbers and then clicking the "Save Order" icon next to column header.

Access: This is the user group that is allowed to use this bot. Click on it to change it. A popup window will appear.

Type: Shows bot's type.

File: Shows bot's file name.

 

Footer

At the bottom of the page there is the footer.

Navigation - If the number of bots is larger than the number of bots displayed per page, then navigation controls are activated. You can use them to switch pages.

Display # - Use this control to change the number of displayed bots per page. The default value is controlled from 'List Length' parameter defined in Global Configuration.
