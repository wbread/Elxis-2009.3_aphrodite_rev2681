Language Manager (Edit)

Use the Language Editor to edit online the contents of a Front-End language file. In order to be able to save your changes, the language file must be writable. Its status is displayed next to the file name, above the editor. Be sure that is set to Writable.

Checking the "Make unwriteable after saving" option, the file will be set to an unwritable state after saving changes.

 

Toolbar

Save: Click the "Save" icon to save changes.

Cancel: If you want to discard changes and return, click "Cancel" icon.

Help: Click the "Help" icon to view this Help Screen.
