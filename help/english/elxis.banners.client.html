Banner Clients Manager

Each banner is associated with a customer. Use this form to add or edit banner clients.

Toolbar Icons

New: Creates a new Client.

Edit: Edits the selected Client.

Delete: Deletes the selected Client.

Help: This Help Screen.

 

Columns

Client Name: Client's name. Click to edit Client.

Banner Client IDs: The ID of a Client. It can be used as parameter in Banners Module.

Contact: Contact person for this client.

No. of Active Banners: The number of published banners a client has.
