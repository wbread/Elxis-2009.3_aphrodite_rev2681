Mass Mail

This page allow you to send a mail message to one or more groups of users.

NOTE: Only users in the Super Administrator group are permitted to use this feature.

 

Toolbar

Send Mail: Click to send the mail.

Cancel: Do not send mail and return to previous screen.

Help: This Help Screen.

 

Details

Group: You can select the parent users group you want to send the mail. All members of that group will receive the mail. Select "All User Groups" if you want to send it to all registered users.

Mail to Child Groups: Check this box if you want mail to be send also to members of the groups that are children of the group you have selected.

Send in HTML mode: Message will be send in HTML mode, keeping the html tags that you have included in 'Message'.

Subject: Type a subject for the message.

Message: Type the body of the message. You can use html tags if you have checked the Send in HTML mode.
