Menu Item Type:
Blog - Content Category Archive

This screen, adds a new menu item of type "Blog - Content Category Archive". A menu item of this type presents the archived content items that belong to a single category in the form of blog articles. From each article is presented only the Intro Text. If the article has also its Main Text area filled then a "Read more" link is presented. Clicking this link you can view the full article.

We can define a number of parameters for this menu item type that control the number and the way the blogs are presented.

 

Toolbar Icons

Back: Return to previous screen and select a new menu item type to create.

Save: Save changes and return to previous screen.

Apply: Save changes and remain to this screen.

Cancel: Aborts the creation of the new menu item. Changes are discarted if you are editing a menu item.

Help: This Help Screen.

 

Details

Name: This is the name of the menu item, as it will be shown in this site.

Category: Select the content Category from which you want to show articles. You can select only one category.

URL: Shows the URL to this menu item. This readonly field is shown only after saving the menu item.

Parent Item: You can select an existing menu item as the parent of this new menu item. That way you create hierarchical structures.

Ordering: You select the order you want this menu item to appear. Available only after saving the new menu item. New menu items default to the last place.

Language: Select the language under which this menu item will be available. If you select "All", it will be available and viewable, under all languages.

Access Level: Select the user group a user must belong, in order to view this menu item.

Published: Select "Yes" to make the menu item avaialble to this site. Select "No" to hide this menu item.

 

Parameters

Menu Image: If you want you can show a small image next to the menu item. Select this image from the dropdown list. Note: For this image to be visible, the "Show Menu Icons" parameter of the menu module, must also be set to "Yes".

Use only the Menu Image: If you set this parameter to 'Yes', then instead of the menu name the menu item will be drawn only using the menu image.

Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page. You have to define them in your template's CSS stylesheet. For example, if you enter "-test" (without the quotes), you will have to define extra classes like: "contentpaneopen-test" and "contentheading-test".

Back Button: Controls whether the "Back" text link is shown or not at the bottom of the page. Clicking the "Back" link users are returned to the previous page. The dropdown list has 3 options:

  • "Use Global" - Shows or hides the Back link based on the setting in Global Configuration.
  • "Hide" - Hide the Back link, overriding the Global setting.
  • "Show" - Show the Back link, overriding the Global setting.

Page Title: If you wish to show a special Page Title, type it to the text box and select "Yes" in the next "Show Page Title" parameter.

#Leading: Set the number of arcived content items you want to display as leading (full width) items. Setting to 0, means that no content item will be displayed as leading.

# Intro: Set the number of arcived content items you want to show with the Intro text shown.

Columns: Set the number of columns you want the arcived content items to be arranged at.

# Links: Set the number of arcived content items you want to be displayed as links at the bottom of the page.

Order by: Select the order the arcived content items will be displayed.

Pagination: Select if you want the pagination controls to be visible.

Pagination Results: Hide or show the pagination results.

MOSImages: If content items contain mosimage tags, you may select to show them or not in blog view.

Section Name: Show or hide the section the content item belongs.

Section Name Linkable: If you have chosen to show the section name, you can make it linkable.

Item Tiles: Select if you want the archived content items title to be visible or not.

Linked Titles: If you have chosen to show archived content items title, you may select to act also as links to the whole article. You may select the global setting or bypass it for this menu item only.

Read More: Select if you want the "Read more" link to be visible. You may select the global setting or bypass it for this menu item only.

Item Rating: Select if you want the rating system to be visible. You may select the global setting or bypass it for this menu item only.

Author Names: Select if you want the Author names to be visible. You may select the global setting or bypass it for this menu item only.

Created Date and Time: Select if you want archived content item's created date and time to be visible. You may select the global setting or bypass it for this menu item only.

Modified Date and Time: Select if you want archived content item's modified date and time to be visible. You may select the global setting or bypass it for this menu item only.

RTF Icon: Select if you want to be visible the button for creating an RTF version of the archived content item. You may select the global setting or bypass it for this menu item only.

PDF Icon: Select if you want to be visible the button for creating a PDF version of the archived content item. You may select the global setting or bypass it for this menu item only.

Print Icon: Select if you want to be visible the button for printing the archived content item. You may select the global setting or bypass it for this menu item only.

Email Icon: Select if you want to be visible the button for emailing the archived content item to a friend. You may select the global setting or bypass it for this menu item only.
